## 가입 방법

### wavve 앱에서 가입
```
1. wavve 앱 설치 (App Store/Play Store)
2. 회원가입 또는 T ID 로그인
3. 요금제 선택
4. 결제 수단 등록
5. 이용 시작
```

### T월드 앱에서 가입
```
1. T월드 앱 > 부가서비스
2. wavve 선택
3. 요금제 선택
4. 가입 완료 (휴대폰 요금 합산)
```

### 결제 방법
| 방법 | 설명 |
|------|------|
| 휴대폰 요금 합산 | T월드에서 가입 시 자동 |
| 신용카드 | wavve 앱에서 직접 결제 |
| 간편 결제 | 카카오페이, 네이버페이 등 |

## B tv + wavve

### 연동 서비스
B tv 셋톱박스에서 wavve 바로 이용

```
B tv 리모컨 > wavve 버튼
또는
B tv 앱 목록 > wavve 선택
```

### 장점
- 리모컨 하나로 조작
- TV 대화면 시청
- 별도 기기 불필요

### 요금
- B tv 요금과 wavve 요금 별도
- 결합 프로모션 시 추가 할인

## 사용 방법

### 지원 기기
| 기기 | 앱 |
|------|-----|
| 스마트폰/태블릿 | wavve 앱 |
| 스마트 TV | 내장 앱 또는 설치 |
| PC/노트북 | wavve.com |
| 셋톱박스 | B tv, Apple TV 등 |
| 게임콘솔 | PlayStation, Xbox |

### 다운로드 오프라인 시청
```
1. 콘텐츠 선택
2. 다운로드 버튼 클릭
3. 화질 선택
4. 다운로드 완료 후 오프라인 시청
```

**다운로드 제한**:
- 저장 기간: 다운로드 후 30일
- 재생 시작 후: 48시간 내 시청

### 프로필 관리
- 최대 5개 프로필 생성
- 시청 기록 분리
- 키즈 프로필 (성인 콘텐츠 제한)

## 해지 방법

### wavve 앱에서 해지
```
1. wavve 앱 > 설정 > 이용권 관리
2. 구독 해지
3. 해지 사유 선택
4. 해지 완료
```

### T월드에서 해지
```
1. T월드 앱 > 부가서비스
2. wavve > 해지
3. 해지 확인
```

### 해지 시 주의
- 이용권 만료일까지 시청 가능
- 다운로드 콘텐츠 재생 불가
- 시청 기록/프로필은 유지

## 자주 묻는 질문 (FAQ)

**Q: wavve는 SK텔레콤 고객만 쓸 수 있나요?**
A: 아니요, 누구나 사용 가능합니다. SK텔레콤 고객은 T멤버십 할인 혜택이 있습니다.

**Q: 넷플릭스와 wavve 둘 다 가입해야 하나요?**
A: 콘텐츠 취향에 따라 다릅니다. 지상파 콘텐츠 선호 시 wavve, 해외 콘텐츠 선호 시 넷플릭스를 권장합니다.

**Q: 가족이 동시에 볼 수 있나요?**
A: 요금제에 따라 다릅니다. 프리미엄은 4명까지 동시 시청 가능합니다.

**Q: 해외에서도 볼 수 있나요?**
A: 저작권 문제로 해외에서는 대부분의 콘텐츠를 시청할 수 없습니다.

**Q: TV에서 어떻게 보나요?**
A: 스마트 TV wavve 앱, B tv 셋톱박스, 크롬캐스트, Apple TV 등을 이용하세요.
