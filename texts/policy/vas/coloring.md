# 컬러링 서비스 가입/해지/요금

> 최종 업데이트: 2025년 2월

## 컬러링이란?

상대방이 나에게 전화할 때 들리는 통화연결음을 음악이나 음성으로 바꾸는 서비스입니다.

---

## 서비스 요금

### 기본 요금
| 항목 | 요금 |
|------|------|
| 월 기본료 | 2,200원 |
| 곡당 요금 | 곡별 상이 (무료~2,000원) |
| 최대 보관 곡 | 10곡 |

### 요금 체계
```
총 요금 = 기본료(2,200원) + 곡 이용료

예시: 기본료 + 유료곡 1곡(1,500원)
= 2,200 + 1,500 = 3,700원/월
```

### 무료 콘텐츠
- 일부 클래식, 효과음
- 프로모션 무료곡
- 직접 녹음 음성

## 가입 방법

### T월드 앱
```
1. T월드 앱 > 부가서비스
2. 컬러링 선택
3. 가입하기
4. 원하는 곡 선택
5. 결제 및 가입 완료
```

### 114 전화
```
1. 114 전화
2. "컬러링" 선택
3. 상담사 연결
4. 가입 및 곡 설정
```

### 컬러링 앱
```
1. 앱스토어에서 "T컬러링" 검색
2. 앱 설치
3. 로그인 (T ID)
4. 원하는 곡 선택 및 가입
```

### 문자 가입
```
- 가입: "컬러링" 문자를 114로 발송
- ARS 안내에 따라 진행
```

## 곡 설정 방법

### 기본 설정
모든 발신자에게 동일한 컬러링 적용

```
T컬러링 앱 > 내 컬러링 > 기본 컬러링 설정
```

### 발신자별 설정
특정 번호에 다른 컬러링 적용

```
T컬러링 앱 > 내 컬러링 > 발신자별 설정
> 연락처 선택 > 곡 지정
```

### 시간대별 설정
시간에 따라 다른 컬러링 적용

```
예시:
- 출근 시간: 상쾌한 음악
- 퇴근 후: 팝송
- 심야: 잔잔한 음악
```

### 그룹별 설정
| 그룹 | 설정 예시 |
|------|-----------|
| 가족 | 따뜻한 음악 |
| 직장 | 클래식 |
| 친구 | 유행 팝송 |

## 콘텐츠 종류

### 최신 음악
- 신곡 발매 시 빠른 업데이트
- 인기 차트 반영
- 장르별 분류

### 클래식/재즈
- 베토벤, 모차르트 등
- 재즈 명곡
- 피아노 연주곡

### 효과음/음성
| 유형 | 예시 |
|------|------|
| 자연 | 새소리, 파도소리, 빗소리 |
| 재미 | 개구리, 고양이, 웃음소리 |
| 안내 | "전화 연결 중입니다" |

### 직접 녹음
- 본인 목소리로 녹음 가능
- 15초 이내
- 무료

## 해지 방법

### T월드 앱
```
1. T월드 앱 > 부가서비스
2. 컬러링 > 해지
3. 해지 확인
```

### 114 전화
```
1. 114 전화
2. "부가서비스 해지" 선택
3. 컬러링 해지 요청
```

### T컬러링 앱
```
1. T컬러링 앱 > 설정
2. 서비스 해지
3. 해지 사유 선택
4. 해지 완료
```

### 해지 시 주의사항
- 보유 곡 모두 삭제
- 재가입 시 곡 재구매 필요
- 월중 해지 시 일할 계산

## 자주 묻는 질문 (FAQ)

**Q: 컬러링 곡은 매달 요금이 나오나요?**
A: 네, 기본료 2,200원 + 유료곡 이용료가 매월 청구됩니다.

**Q: 무료 곡만 사용하면 요금이 안 나오나요?**
A: 무료 곡을 사용해도 기본료 2,200원은 청구됩니다.

**Q: 여러 곡을 설정하면 랜덤으로 나오나요?**
A: 기본 설정, 발신자별 설정, 시간대별 설정에 따라 지정한 곡이 재생됩니다. 랜덤 재생 설정도 가능합니다.

**Q: 컬러링 없이 기본음으로 돌리고 싶어요.**
A: 컬러링 서비스를 해지하면 기본 통화연결음으로 복원됩니다.

**Q: 상대방이 컬러링을 싫어하면요?**
A: 특정 번호에 대해 기본음을 설정하거나, 상대방이 컬러링 차단 설정을 할 수 있습니다.

**Q: 저작권 문제는 없나요?**
A: SK텔레콤이 저작권 계약을 체결한 합법적 서비스입니다.
