# T전화 서비스 (스팸차단, 발신자정보)

> 최종 업데이트: 2025년 2월

## T전화란?

SK텔레콤의 대표 전화 앱으로, 스팸 차단, 발신자 정보 표시, 안심 통화 등 다양한 기능을 제공합니다.

---

## 주요 기능

### 발신자 정보 표시
| 기능 | 설명 |
|------|------|
| 업체명 표시 | 등록된 업체의 경우 업체명 자동 표시 |
| 스팸 경고 | 스팸 의심 번호 경고 표시 |
| 안전 번호 | 안전한 발신자 표시 |
| 검색 | 모르는 번호 검색 |

### 스팸 차단
| 기능 | 설명 |
|------|------|
| 자동 차단 | AI 기반 스팸 자동 차단 |
| 수동 차단 | 특정 번호 직접 차단 |
| 차단 목록 | 차단한 번호 관리 |
| 스팸 신고 | 스팸 번호 신고 |

### 안심 통화
| 기능 | 설명 |
|------|------|
| 보이스피싱 경고 | 의심 통화 실시간 경고 |
| 녹음 | 통화 녹음 (법적 고지) |
| 통화 분석 | 의심 통화 내용 분석 |

## 설치 및 설정

### T전화 앱 설치
```
1. 앱스토어/Play스토어에서 "T전화" 검색
2. 앱 설치
3. 기본 전화 앱으로 설정
4. 권한 허용 (전화, 연락처, 알림)
```

### 기본 전화 앱 설정

**Android**:
```
설정 > 앱 > 기본 앱 > 전화 앱 > T전화 선택
```

**iPhone**:
```
설정 > 전화 > 차단 및 발신자 확인 > T전화 활성화
```

### 스팸 차단 설정
```
T전화 앱 > 설정 > 스팸 차단
- 스팸 자동 차단: ON
- 스팸 경고 표시: ON
- 차단 수준 설정: 낮음/중간/높음
```

---

## 스팸 차단 상세

### 차단 수준
| 수준 | 설명 |
|------|------|
| 낮음 | 확실한 스팸만 차단 |
| 중간 | 의심 스팸 포함 차단 (권장) |
| 높음 | 광범위한 차단 (오차단 가능성) |

### 스팸 유형
| 유형 | 설명 |
|------|------|
| 광고 | 대출, 보험, 부동산 등 광고 전화 |
| 보이스피싱 | 금융사기 의심 전화 |
| ARS 스팸 | 자동 응답 광고 |
| 원링 스팸 | 부재중 전화 유도형 |

### 차단 통계
- T전화 앱 > 스팸 차단 > 차단 통계
- 일별/월별 차단 건수 확인
- 차단된 번호 목록 확인

## 통화 녹음

### 녹음 기능
| 기능 | 무료 | 프리미엄 |
|------|------|----------|
| 녹음 가능 시간 | 5분 | 무제한 |
| 녹음 저장 | 기기 내 | 기기 내 + 클라우드 |
| 자동 녹음 | ❌ | ✅ |

### 녹음 설정
```
T전화 앱 > 설정 > 통화 녹음
- 통화 녹음: ON
- 자동 녹음: 특정 번호 선택 가능
- 녹음 안내: 상대방 고지 설정
```

### 녹음 법적 안내
- 통화 녹음 시 상대방에게 고지 권장
- "통화 내용이 녹음됩니다" 안내 음성 설정 가능
- 무단 녹음 후 유포 시 법적 책임

## 문제 해결

### 발신자 정보가 안 나올 때
1. T전화 앱 업데이트 확인
2. 네트워크 연결 확인
3. 앱 데이터 캐시 삭제
4. 앱 재설치

### 스팸 차단이 안 될 때
1. 스팸 차단 설정 ON 확인
2. 차단 수준 확인
3. 앱 권한 확인
4. 앱 업데이트

### 원하는 전화가 차단될 때
1. 차단 목록에서 해당 번호 확인
2. 차단 수준을 "낮음"으로 조정
3. 해당 번호를 연락처에 저장

## 자주 묻는 질문 (FAQ)

**Q: T전화는 SK텔레콤 고객만 쓸 수 있나요?**
A: 아니요, 타사 고객도 T전화 앱을 사용할 수 있습니다. 단, 일부 기능은 SK텔레콤 고객 전용입니다.

**Q: 스팸 차단하면 문자도 차단되나요?**
A: T전화의 스팸 차단은 전화만 해당됩니다. 문자 스팸 차단은 별도 설정이 필요합니다.

**Q: 통화 녹음은 불법 아닌가요?**
A: 본인이 참여한 통화의 녹음은 합법입니다. 단, 상대방 동의 없이 녹음물을 공개하면 문제가 될 수 있습니다.

**Q: 차단된 전화는 어떻게 되나요?**
A: 차단된 전화는 통화 연결이 거부되며, 차단 로그에 기록됩니다.

**Q: T전화 해지하면 기존 녹음 파일은요?**
A: 기기에 저장된 녹음 파일은 유지됩니다. 클라우드 저장은 해지 시 삭제됩니다.
