# T클라우드 서비스

> 최종 업데이트: 2025년 2월

## T클라우드란?

SK텔레콤이 제공하는 클라우드 저장 서비스로, 사진, 연락처, 파일 등을 안전하게 백업하고 여러 기기에서 접근할 수 있습니다.

## 가입 및 설정

### T클라우드 앱 설치
```
1. App Store/Play Store에서 "T클라우드" 검색
2. 앱 설치
3. T ID로 로그인
4. 백업 설정
```

### 자동 백업 설정
```
T클라우드 앱 > 설정 > 자동 백업
- 사진/동영상: ON
- 연락처: ON
- Wi-Fi에서만: 권장
- 충전 중에만: 선택
```

### PC 버전
```
1. cloud.tworld.co.kr 접속
2. T ID 로그인
3. 웹에서 파일 관리
```

## 기기 변경 시 데이터 이전

### T클라우드로 이전
```
기존 폰:
1. T클라우드 앱에서 백업
2. 백업 완료 확인

새 폰:
1. T클라우드 앱 설치
2. 동일 T ID로 로그인

4. 데이터 다운로드

### 용량 확인
```

- 사용 중인 용량
- 항목별 사용량
```

| 방법 | 설명 |
|------|------|

- 30일 후 자동 영구 삭제
- 휴지통도 용량 차지

## 해지 및 주의사항

### 해지 방법
```
T월드 앱 > 부가서비스 > T클라우드 > 해지

## 자주 묻는 질문 (FAQ)

**Q: T클라우드는 SK텔레콤 고객만 쓸 수 있나요?**
A: T ID가 있으면 누구나 사용 가능합니다. 단, T멤버십 추가 용량은 SK텔레콤 고객만 받습니다.

**Q: 폰을 잃어버려도 T클라우드 데이터는 안전한가요?**
A: 네, 클라우드에 백업된 데이터는 안전합니다. T ID로 다른 기기에서 접근 가능합니다.

**Q: 사진 원본 화질로 백업되나요?**
A: 설정에 따라 원본 또는 절약 모드(압축)로 백업할 수 있습니다.

**Q: 가족과 저장 공간을 나눠 쓸 수 있나요?**
A: 개인별로 별도 T ID가 필요하며, 용량 공유는 불가합니다. 공유 앨범으로 사진 공유는 가능합니다.

**Q: T클라우드와 삼성 클라우드 중 뭘 쓰면 좋나요?**
A: 삼성 폰 사용자는 삼성 클라우드도 편리합니다. 통신사 변경 대비 시 T클라우드 또는 구글드라이브 권장합니다.
