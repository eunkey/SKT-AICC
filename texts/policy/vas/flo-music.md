## 가입 방법

### FLO 앱에서 가입
```
1. FLO 앱 설치 (App Store/Play Store)
2. 회원가입 또는 T ID 로그인
3. 요금제 선택
4. 결제 수단 등록
5. 이용 시작
```

### T월드 앱에서 가입
```
1. T월드 앱 > 부가서비스
2. FLO 선택
3. 요금제 선택
4. 가입 완료 (휴대폰 요금 합산)
```

### 결제 방법
| 방법 | 설명 |
|------|------|
| 휴대폰 요금 합산 | T월드에서 가입 시 |
| 신용카드 | FLO 앱에서 직접 결제 |
| 카카오페이 등 | 간편 결제 지원 |

## T멤버십 연동

### 연동 방법
```
1. FLO 앱 > 설정 > 계정
2. T멤버십 연동
3. T월드 앱으로 인증
4. 할인 자동 적용
```

### 연동 혜택
- 월정액 T멤버십 등급별 할인
- T포인트 적립 (일부 프로모션)
- 통합 청구서

---

## 컬러링/RBT 연동

### FLO에서 컬러링 설정
```
1. FLO 앱에서 원하는 곡 선택
2. 더보기(···) > 컬러링 설정
3. 컬러링 서비스 가입 (미가입 시)
4. 곡 적용
```

### 컬러링 요금
- FLO 요금과 별도
- 컬러링 기본료 + 곡 이용료 적용

---

## 해지 방법

### FLO 앱에서 해지
```
1. FLO 앱 > 설정 > 결제 관리
2. 구독 해지
3. 해지 사유 선택
4. 해지 완료
```

### T월드에서 해지 (휴대폰 요금 합산 시)
```
1. T월드 앱 > 부가서비스
2. FLO > 해지
3. 해지 확인
```

### 해지 시 주의
- 다운로드한 곡 재생 불가
- 플레이리스트는 계정에 유지
- 재가입 시 이어서 이용 가능

## 자주 묻는 질문 (FAQ)

**Q: FLO는 SK텔레콤 고객만 쓸 수 있나요?**
A: 아니요, 누구나 사용 가능합니다. 단, T멤버십 할인은 SK텔레콤 고객만 받을 수 있습니다.

**Q: FLO 해지하면 다운로드한 곡은요?**
A: 해지 후에는 다운로드한 곡도 재생되지 않습니다.

**Q: 가족 요금제는 어떻게 쓰나요?**
A: 대표 가입 후 가족 구성원을 초대하면 각자 계정으로 이용 가능합니다.

**Q: 차량에서 FLO를 들을 수 있나요?**
A: 블루투스, CarPlay, Android Auto를 통해 차량에서 재생 가능합니다.

**Q: 오프라인 재생은 얼마나 저장되나요?**
A: 다운로드 요금제 기준 무제한이지만, 저장 공간에 따라 다릅니다.
