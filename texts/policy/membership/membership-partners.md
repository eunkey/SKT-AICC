## 제휴사 이용 방법

### T월드 앱 쿠폰 발급
```
1. T월드 앱 > 멤버십
2. 제휴사 선택
3. 쿠폰 발급
4. 결제 시 바코드 제시
```

### 제휴사 앱 연동
```
1. 제휴사 앱 설치
2. T멤버십 연동 메뉴 선택
3. T월드 앱으로 인증
4. 자동 할인 적용
```

### 현장 결제
```
1. 결제 전 T월드 앱 > 멤버십
2. 바코드 화면 표시
3. 직원에게 제시
4. 할인 적용 후 결제
```

## 주의사항

### 할인 제한
- 대부분 월별 이용 횟수 제한
- 일부 품목 할인 제외
- 다른 할인과 중복 적용 불가 (대부분)

### 변경 안내
- 제휴사 및 할인율은 수시 변경 가능
- 최신 정보는 T월드 앱에서 확인

## 자주 묻는 질문 (FAQ)

**Q: 제휴사 할인은 무제한으로 받을 수 있나요?**
A: 대부분 월별 이용 횟수 또는 금액 한도가 있습니다. T월드 앱에서 잔여 횟수를 확인해 주세요.

**Q: 여러 제휴사를 동시에 사용할 수 있나요?**
A: 네, 각 제휴사별로 별도 할인이 적용됩니다.

**Q: 가족 회선도 제휴 할인을 받을 수 있나요?**
A: 각 회선별로 멤버십이 적용되어 개별 할인을 받을 수 있습니다.
