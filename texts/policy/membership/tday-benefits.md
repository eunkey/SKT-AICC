# T데이 혜택 안내

> 최종 업데이트: 2024년 1월

## T데이란?

매달 특정 일에 T멤버십 고객에게 제공되는 특별 혜택 이벤트입니다.

---

## T데이 일정

### 정기 T데이
| 일자 | 이벤트명 | 주요 혜택 |
|------|---------|----------|
| 매월 9일 | T데이 | 커피/영화/편의점 특별 할인 |
| 매월 19일 | T데이 | 외식/주유 특별 할인 |
| 매월 29일 | T데이 | 쇼핑/레저 특별 할인 |

### 특별 T데이
- 명절 (설날, 추석): 명절 특별 이벤트
- 창립기념일 (3월): 연중 최대 혜택
- 연말 (12월): 감사 이벤트

## T데이 참여 방법

### 1. T월드 앱
1. T월드 앱 실행
2. 메인 화면 T데이 배너 클릭
3. 원하는 쿠폰 다운로드
4. 제휴사에서 사용

### 2. T멤버십 앱
1. T멤버십 앱 실행
2. T데이 탭 선택
3. 쿠폰 확인 및 다운로드
4. 바코드 제시 사용

### 3. 문자 알림
- T데이 전날/당일 SMS 발송
- 링크 클릭으로 쿠폰 다운로드

## T데이 쿠폰 사용

### 쿠폰 유효기간
| 쿠폰 유형 | 유효기간 |
|----------|----------|
| T데이 당일 쿠폰 | 당일 24시까지 |
| 주간 쿠폰 | 다운로드 후 7일 |
| 월간 쿠폰 | 당월 말일까지 |

### 사용 방법
1. T월드/T멤버십 앱에서 쿠폰 활성화
2. 제휴사 방문
3. 결제 전 쿠폰 바코드 제시
4. 할인 적용 후 결제

### 주의사항
- 쿠폰 1장당 1회 사용
- 타 할인/쿠폰과 중복 불가
- 일부 상품/메뉴 제외 가능
- 품절 시 조기 종료 가능

## T데이 경품 이벤트

### 응모 방법
1. T월드 앱 > T데이 이벤트
2. 응모하기 버튼 클릭
3. 등급별 응모권 자동 적용
4. 당첨자 발표 확인

### 경품 예시
| 등급 | 경품 예시 |
|------|----------|
| 1등 | 해외여행권, 최신 스마트폰 |
| 2등 | 백화점 상품권 50만원 |
| 3등 | 문화상품권 10만원 |
| 4등 | 편의점 상품권 5,000원 |
| 5등 | T포인트 1,000P |

### 당첨 확인
- T월드 앱 > 이벤트 > 당첨 확인
- 당첨 SMS 발송
- 경품 수령 방법 안내

## 자주 묻는 질문

### Q: T데이 쿠폰은 몇 장까지 받을 수 있나요?
A: 제휴사별로 각 1장씩 다운로드 가능합니다. 같은 제휴사 쿠폰을 여러 장 받을 수는 없습니다.

### Q: T데이 쿠폰을 못 받았어요
A: T데이 당일에만 다운로드 가능합니다. 날짜가 지나면 해당 쿠폰은 받을 수 없습니다.

### Q: 가족이 대신 사용할 수 있나요?
A: 아니요, T데이 쿠폰은 본인 회선에서만 사용 가능합니다. 바코드 인증이 필요합니다.

### Q: T데이와 일반 멤버십 할인 중 뭐가 더 좋나요?
A: 보통 T데이 할인이 더 큽니다. 하지만 일반 할인 횟수와 별도이므로 둘 다 활용하세요.

### Q: T데이 알림을 안 받고 싶어요
A: T월드 앱 > 설정 > 알림 설정에서 마케팅 수신 동의를 해제하면 됩니다.
