# T데이 혜택 및 이벤트

> 최종 업데이트: 2025년 2월

## T데이란?

매주 특정 요일에 T멤버십 고객에게 제공하는 특별 할인 혜택입니다. 일반 멤버십 할인보다 더 큰 폭의 할인을 받을 수 있습니다.

## T데이 이용 방법

### 쿠폰 발급
```
1. T월드 앱 > 멤버십 > T데이
2. 해당 요일 혜택 선택
3. 쿠폰 발급
4. 결제 시 바코드 제시
```

### 이용 조건
| 항목 | 내용 |
|------|------|
| 대상 | T멤버십 전 등급 |
| 이용 횟수 | 제휴사당 1회/일 |
| 쿠폰 유효시간 | 발급 후 24시간 |
| 발급 시간 | 해당 요일 00:00 ~ 23:59 |

### 주의사항
- T데이 혜택은 일반 멤버십 할인과 중복 적용 불가
- 해당 요일에만 쿠폰 발급/사용 가능
- 일부 품목 제외 (담배, 상품권 등)
- 매장 상황에 따라 조기 종료 가능

## 자주 묻는 질문 (FAQ)

**Q: T데이 쿠폰은 미리 발급받을 수 있나요?**
A: 일반 고객은 해당 요일에만, VIP/플래티넘은 전날부터 발급 가능합니다.

**Q: T데이 혜택과 일반 멤버십 할인 중 더 좋은 건 뭔가요?**
A: 보통 T데이 혜택이 더 큽니다. 비교 후 더 유리한 것을 선택하세요.

**Q: 쿠폰을 발급받고 사용 안 하면 어떻게 되나요?**
A: 24시간 후 자동 소멸됩니다. 다음 해당 요일에 다시 발급받을 수 있습니다.

**Q: T데이 혜택을 가족도 받을 수 있나요?**
A: 각 회선별로 T데이 쿠폰을 발급받을 수 있습니다. 동반자는 일부 혜택(영화 1+1 등)만 적용됩니다.

**Q: T데이 쿠폰이 조기 소진되었다고 하는데요?**
A: 일부 인기 혜택은 수량이 한정되어 조기 소진될 수 있습니다. 오픈 직후 발급을 권장합니다.
