## 등급 변경

### 상향 조건
- 상위 요금제 변경 시 즉시 상향
- 장기 고객 기준 충족 시 상향

### 하향 조건
- 하위 요금제 변경 시 다음 달부터 하향
- 해지 후 재가입 시 초기화

### 유지 조건
- 동일 요금제 유지 시 등급 유지
- 장기 고객 우대 등급은 이용 기간 유지 시 계속

## 멤버십 카드 발급

### T멤버십 앱 카드
- T월드 앱 내 멤버십 카드 자동 생성
- 바코드/QR코드로 제휴사 할인

### 실물 카드
- 현재 실물 카드 발급 중단
- 앱 카드로 대체

### 사용 방법
```
1. T월드 앱 > 멤버십
2. 바코드/QR코드 표시
3. 결제 시 제시
4. 할인 적용
```

## 자주 묻는 질문 (FAQ)

**Q: 등급은 언제 바뀌나요?**
A: 요금제 변경 시 상위 등급은 즉시, 하위 등급은 다음 달부터 적용됩니다.

**Q: 가족도 같은 등급 혜택을 받나요?**
A: 아니요, 등급은 개인 회선별로 적용됩니다. 각 회선의 요금제에 따라 등급이 결정됩니다.

**Q: 장기 고객인데 왜 VIP가 아닌가요?**
A: 20년 이상 이용 시 VIP, 10년 이상은 1등급 상향입니다. 이용 기간을 T월드 앱에서 확인해 주세요.

**Q: 해지 후 재가입하면 등급이 유지되나요?**
A: 아니요, 해지 후 재가입 시 이용 기간과 등급이 초기화됩니다.

**Q: T멤버십 포인트로 요금을 낼 수 있나요?**
A: 네, 적립된 포인트로 휴대폰 요금 납부가 가능합니다.
