## 이벤트 참여 방법

### T월드 앱
```
1. T월드 앱 실행
2. 이벤트 배너 확인
3. 원하는 이벤트 클릭
4. 참여 조건 확인
5. 참여하기 버튼 클릭
```

### 온라인
```
1. tworld.co.kr 접속
2. 이벤트 메뉴 선택
3. 이벤트 목록에서 선택
4. 참여 완료
```

### 매장
- T월드 매장 방문
- 매장 전용 이벤트 확인
- 직원 안내에 따라 참여

## 자주 묻는 질문 (FAQ)

**Q: 여러 이벤트에 동시에 참여할 수 있나요?**
A: 이벤트별로 다릅니다. 대부분 중복 참여 가능하나, 일부는 택1입니다.

**Q: 당첨되었는데 어디서 받나요?**
A: 데이터/쿠폰은 T월드 앱 쿠폰함, 실물은 배송으로 받습니다.

**Q: 가족도 이벤트 참여 가능한가요?**
A: 각 회선별로 참여 가능합니다. 가족이라도 별도 참여입니다.

**Q: 이벤트 기간이 지나면 혜택을 받을 수 없나요?**
A: 네, 이벤트 기간 내 참여가 원칙입니다.
