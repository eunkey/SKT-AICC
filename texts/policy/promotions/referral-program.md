# 친구추천 프로그램

> 최종 업데이트: 2025년 2월

## 친구추천 프로그램 개요

SK텔레콤 고객이 지인을 추천하여 가입하면 추천인과 피추천인 모두 혜택을 받는 프로그램입니다.

---

## 프로그램 종류

### 친구추천
| 항목 | 내용 |
|------|------|
| 대상 | SK텔레콤 가입 고객 → 타사 사용자 |
| 가입 유형 | 번호이동 / 신규가입 |
| 혜택 | 추천인 + 피추천인 모두 |

### 가족추천
| 항목 | 내용 |
|------|------|
| 대상 | SK텔레콤 가입 고객 → 가족 |
| 가입 유형 | 번호이동 / 신규가입 / 기기변경 |
| 혜택 | 추천인 + 피추천인 + 결합 할인 |

---

## 친구추천 혜택

### 추천인 혜택
| 혜택 | 내용 |
|------|------|
| 데이터 | 5GB 추가 제공 (1회) |
| 포인트 | 10,000P 적립 |
| 추가 | T멤버십 경험치 상승 |

### 피추천인 혜택
| 혜택 | 내용 |
|------|------|
| 데이터 | 10GB 추가 제공 (첫 달) |
| 요금 할인 | 첫 달 기본료 50% 할인 |
| 포인트 | 5,000P 지급 |

### 월별 추천 한도
| 추천 유형 | 월 한도 |
|-----------|---------|
| 일반 추천 | 5명/월 |
| 가족 추천 | 제한 없음 |

---

## 추천 방법

### T월드 앱 추천
```
1. T월드 앱 > 더보기 > 친구추천
2. 추천 코드 확인 또는 공유 링크 생성
3. 지인에게 코드/링크 공유
4. 지인이 가입 시 코드 입력
```

### 추천 코드 형식
```
추천 코드: 010XXXXXXXX (추천인 휴대폰 번호)
또는
고유 추천 코드: ABCD1234
```

### 문자/카카오톡 공유
```
T월드 앱 > 친구추천 > 공유하기
> 카카오톡/문자 선택 > 지인에게 전송
```

---

## 가입 시 추천 코드 입력

### 온라인 가입 시
```
T다이렉트샵 > 단말기/요금제 선택
> 추천인 정보 입력란 > 추천 코드 입력
```

### 매장 가입 시
```
1. 매장 방문 시 추천 코드 사전 확보
2. 계약서 작성 시 추천인 정보 기입
3. 직원에게 추천 적용 확인
```

### 114 전화 가입 시
```
1. 가입 상담 진행
2. 추천인 번호/코드 안내
3. 상담사가 추천 정보 등록
```

## 혜택 지급 시점

### 추천인 혜택
| 혜택 | 지급 시점 |
|------|-----------|
| 데이터 | 피추천인 개통 후 익일 |
| 포인트 | 피추천인 개통 후 7일 이내 |

### 피추천인 혜택
| 혜택 | 지급 시점 |
|------|-----------|
| 데이터 | 개통 즉시 |
| 요금 할인 | 첫 청구서에 반영 |
| 포인트 | 개통 후 7일 이내 |

### 혜택 확인 방법
```
T월드 앱 > 친구추천 > 추천 현황
- 추천 성공 건수
- 지급된 혜택 내역
```

## 추천 조건

### 추천인 조건
| 조건 | 내용 |
|------|------|
| 가입 상태 | SK텔레콤 정상 사용 중 |
| 요금 납부 | 미납 없음 |
| 추천 제한 | 월 5명까지 |

### 피추천인 조건
| 조건 | 내용 |
|------|------|
| 가입 유형 | 번호이동 또는 신규 |
| 요금제 | 월 33,000원 이상 요금제 |
| 기존 고객 | 최근 6개월 내 SK텔레콤 이용 이력 없음 |

### 제외 대상
- 동일인 추천/피추천 (명의 다르더라도)
- 추천인이 피추천인의 법정대리인인 경우 (일부)
- 부정 가입 의심 시

## 자주 묻는 질문 (FAQ)

**Q: 추천 코드를 입력하지 않고 가입했어요. 소급 적용 되나요?**
A: 가입 후 추천 코드 등록은 불가합니다. 반드시 가입 시점에 입력해야 합니다.

**Q: 가족도 친구추천으로 가입할 수 있나요?**
A: 네, 가능합니다. 가족추천은 결합 할인과 추천 혜택을 동시에 받을 수 있어 더 유리합니다.

**Q: 추천한 친구가 중도 해지하면 혜택이 환수되나요?**
A: 이미 지급된 혜택은 환수되지 않습니다. 단, 부정 가입으로 판명 시 환수될 수 있습니다.

**Q: 한 달에 5명 이상 추천할 수 있나요?**
A: 월 5명 한도가 있으며, 다음 달에 추가 추천이 가능합니다.

**Q: 추천 혜택이 지급되지 않았어요.**
A: T월드 앱에서 추천 현황을 확인하고, 미지급 시 114로 문의해 주세요.

**Q: 기기변경도 추천이 되나요?**
A: 기기변경은 일반 친구추천 대상이 아닙니다. 가족 추천 시 일부 혜택이 적용될 수 있습니다.

## 주의사항

### 추천 시
- 피추천인에게 정확한 코드 전달
- 가입 전 추천 코드 입력 안내
- 월 추천 한도 확인

### 혜택 관련
- 혜택 지급까지 최대 7일 소요
- 조건 미충족 시 혜택 미지급
- 부정 추천 시 혜택 환수 및 제재

### 문의
- T월드 앱 > 친구추천 > 문의하기
- 114 고객센터
