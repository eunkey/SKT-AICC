## 데이터 요금 계산 예시

### 종량제 데이터 사용 시
```
예시: 일본에서 100MB 데이터 사용

100MB = 100,000KB
요금 = 100,000KB × 4원 = 400,000원

⚠️ 로밍패스(9,350원) 대비 약 40배 비용 발생
```

### 이것이 로밍패스를 권장하는 이유입니다!

## 요금 절약 팁

### 팁 1: 로밍패스 필수
- 데이터 사용 예정이라면 반드시 로밍패스 가입
- 1일 1GB 사용 기준 수십만 원 절약

### 팁 2: 데이터 로밍 끄기
- 로밍패스 없이 출국 시 데이터 로밍 OFF
- 설정 > 모바일 네트워크 > 데이터 로밍 끄기

### 팁 3: 현지 Wi-Fi 활용
- 호텔, 공항, 카페 Wi-Fi 이용
- 대용량 작업은 Wi-Fi에서

### 팁 4: 장기 체류 시 현지 SIM
- 2주 이상 체류 시 현지 SIM 카드 고려
- 공항에서 SIM 카드 구매 가능

## 자주 묻는 질문 (FAQ)

**Q: 해외에서 전화 받으면 요금이 나오나요?**
A: 네, 수신도 요금이 부과됩니다. 국가별 수신 요금을 확인해 주세요.

**Q: SMS 수신도 요금이 있나요?**
A: SMS 수신은 무료입니다. 발신만 요금이 부과됩니다.

**Q: 데이터 요금이 너무 비싼데 실수로 쓰면 어떡하죠?**
A: 로밍패스 가입 또는 데이터 로밍 OFF 설정을 권장합니다. 요금 한도 설정도 가능합니다.

**Q: 요금표에 없는 국가는요?**
A: 114로 문의하시면 해당 국가 요금을 안내받을 수 있습니다.
