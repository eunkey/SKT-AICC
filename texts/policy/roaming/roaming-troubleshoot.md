# 로밍 문제 해결 가이드

> 최종 업데이트: 2025년 2월

## 일반적인 로밍 문제

### 증상별 진단

| 증상 | 가능한 원인 | 확인 사항 |
|------|-------------|-----------|
| 로밍 안 됨 | 설정 문제, 네트워크 미연결 | 데이터 로밍 설정 |
| 통화만 됨 | 데이터 로밍 꺼짐 | 데이터 로밍 ON |
| 데이터만 됨 | 음성 로밍 미설정 | 네트워크 모드 확인 |
| 요금 폭탄 | 로밍패스 미가입 | 로밍패스 확인 |
| 속도 느림 | 로밍패스 용량 초과 | 사용량 확인 |

---

## 로밍 연결 안 될 때

### 1단계: 기본 설정 확인

#### 데이터 로밍 설정 확인

**Android**:
```
설정 > 연결 > 모바일 네트워크 > 데이터 로밍 > ON
```

**iPhone**:
```
설정 > 셀룰러 > 셀룰러 데이터 옵션 > 데이터 로밍 > ON
```

#### 비행기 모드 토글
```
1. 비행기 모드 ON
2. 10초 대기
3. 비행기 모드 OFF
4. 네트워크 재연결 대기 (1~2분)
```

### 2단계: 네트워크 수동 선택

자동 연결이 안 될 경우 수동으로 현지 통신사 선택

**Android**:
```
설정 > 연결 > 모바일 네트워크 > 네트워크 운영자 > 수동 선택
> 목록에서 현지 통신사 선택
```

**iPhone**:
```
설정 > 셀룰러 > 네트워크 선택 > 자동 OFF
> 목록에서 현지 통신사 선택
```

### 3단계: 단말기 재부팅
```
1. 단말기 전원 완전히 끄기
2. 30초 대기
3. 전원 켜기
4. 네트워크 연결 대기
```

### 4단계: USIM 재장착
```
1. 전원 끄기
2. USIM 트레이 열기
3. USIM 빼서 접촉면 확인
4. 다시 삽입
5. 전원 켜기
```

## 로밍패스가 적용 안 될 때

### 확인 사항

| 항목 | 확인 방법 |
|------|-----------|
| 가입 상태 | T월드 앱 > 로밍 > 가입 현황 |
| 적용 국가 | 현재 국가가 로밍패스 적용 지역인지 |
| 시작 시점 | 로밍패스 시작 시간 확인 |
| 종료 시점 | 이미 종료되었는지 확인 |

### 해결 방법

#### 로밍패스 미시작
- 첫 해외 접속 시 자동 시작
- 데이터 로밍 ON 상태에서 접속 필요

#### 적용 국가 아님
- 로밍패스 상품별 적용 국가 확인
- 해당 국가용 별도 로밍패스 신청 필요

#### 이미 종료됨
- 새로운 로밍패스 추가 구매
- T월드 앱 또는 +82-2-6343-9000으로 신청

## 해외에서 통화 문제

### 발신 안 됨

**확인 사항**:
```
1. 잔액/요금 납부 상태 확인
2. 국제전화 차단 설정 확인
3. 번호 형식 확인:
   - 한국으로: +82-10-XXXX-XXXX
   - 현지: 현지 번호 그대로
```

### 수신 안 됨

**확인 사항**:
```
1. 착신전환 설정 확인
2. 비행기 모드 확인
3. 네트워크 연결 상태 확인
4. 발신자에게 +82-10-XXXX-XXXX 형식으로 안내
```

### 통화 품질 불량

**원인**:
- 현지 네트워크 품질 문제
- 혼잡 지역

**해결**:
- Wi-Fi 통화(VoWiFi) 사용
- 카카오톡 등 인터넷 전화 사용
- 위치 이동 후 재시도

## 요금 관련 문제

### 예상치 못한 고액 요금

**원인 파악**:
1. 로밍패스 미가입 상태에서 데이터 사용
2. 로밍패스 적용 국가 외 지역 사용
3. 로밍패스 종료 후 계속 사용
4. 경유지에서 자동 접속

**대응 방법**:
```
1. T월드 앱에서 사용 내역 확인
2. 114 또는 +82-2-6343-9000 문의
3. 사유 설명 후 조정 요청
```

### 요금 폭탄 예방

**출국 전**:
- 로밍패스 미리 신청
- 데이터 로밍 요금 한도 설정

**설정 방법**:
```
T월드 앱 > 로밍 > 요금 한도 설정
기본 한도: 20만원
초과 시 데이터 자동 차단
```

## 특수 상황 해결

### 경유지에서 로밍

**문제**: 경유지에서 자동 로밍 → 예상치 못한 요금

**예방**:
- 경유 중 비행기 모드 유지
- 필요 시만 로밍 ON

### 크루즈/항공기 내 로밍

**주의**: 일반 로밍패스 미적용 지역

- 크루즈: 해상 로밍 별도 요금 (매우 비쌈)
- 항공기: 기내 Wi-Fi/로밍 별도 서비스

**권장**: 비행기 모드 유지

### 중국에서 앱 차단

**차단 앱**: 구글, 유튜브, 페이스북, 인스타그램 등

**해결**:
- SK텔레콤 로밍 시 카카오톡은 사용 가능
- VPN 앱 사전 설치 (일부 VPN도 차단됨)
- 네이버, 카카오 서비스 활용

## 귀국 후 조치

### 로밍 요금 확인
```
1. T월드 앱 > 사용량 > 로밍 사용 내역
2. 청구서 발행 후 상세 확인
3. 이의 있을 시 114 문의
```

### 자동 로밍 해제
- 귀국 후 국내 네트워크 자동 연결
- 별도 해제 작업 불필요

### 데이터 로밍 끄기 (권장)
국내에서는 데이터 로밍 OFF 권장
- 실수로 해외 네트워크 접속 방지
- 국경 지역 자동 로밍 방지

## 자주 묻는 질문 (FAQ)

**Q: 로밍이 갑자기 안 되는데 어떻게 하나요?**
A: 비행기 모드 토글, 단말기 재부팅, 네트워크 수동 선택 순서로 시도해 보세요.

**Q: 로밍패스 샀는데 요금이 나왔어요.**
A: 로밍패스 적용 국가/기간을 확인해 주세요. 적용 외 사용 시 종량제 요금이 부과됩니다.

**Q: 해외에서 데이터만 안 됩니다.**
A: 데이터 로밍 설정이 꺼져 있을 수 있습니다. 설정에서 데이터 로밍을 켜주세요.

**Q: 로밍 요금이 너무 많이 나왔는데 조정 가능한가요?**
A: 114로 연락하여 상황을 설명하면 일부 조정이 가능할 수 있습니다. 단, 정당한 사용분은 조정 어려울 수 있습니다.

**Q: 현지 유심으로 바꿨다가 다시 SK텔레콤 유심 쓰면 로밍 되나요?**
A: 네, SK텔레콤 USIM을 다시 장착하면 로밍 서비스가 정상 작동합니다.
