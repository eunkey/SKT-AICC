# 로밍 문제 해결 가이드

> 최종 업데이트: 2024년 1월

## 자주 발생하는 문제

### 1. 현지 도착 후 네트워크 연결 안 됨

#### 즉시 확인사항
1. **비행기 모드 해제**
   - 설정에서 비행기 모드 OFF 확인

2. **데이터 로밍 설정 확인**
   - iPhone: 설정 > 셀룰러 > 데이터 로밍 ON
   - Galaxy: 설정 > 연결 > 모바일 네트워크 > 데이터 로밍 ON

3. **네트워크 수동 선택**
   - 설정 > 네트워크 운영자 > 자동 OFF
   - 표시된 통신사 목록에서 선택
   - 연결 후 다시 자동 ON

4. **단말기 재부팅**
   - 전원 완전 OFF → 30초 대기 → ON

#### 추가 조치
- USIM 분리 후 재장착
- APN 설정 확인 (보통 자동 설정됨)
- 다른 위치에서 재시도

---

### 2. 데이터는 되는데 전화가 안 됨

#### 확인사항
1. **VoLTE 로밍 설정**
   - 일부 국가는 VoLTE 미지원
   - VoLTE OFF 후 재시도

2. **발신번호 형식 확인**
   - 한국으로 발신: +82-10-XXXX-XXXX
   - 현지 번호 발신: 국가코드 + 번호

3. **요금 미납 확인**
   - 미납 시 로밍 발신 제한
   - T월드 앱에서 요금 확인

---

### 3. 인터넷 속도가 너무 느림

#### 원인 및 해결
| 원인 | 해결 방법 |
|------|----------|
| 일 1GB 소진 | 400Kbps 제한 (정상), 다음 날 리셋 |
| 현지 네트워크 혼잡 | 다른 통신사 수동 선택 |
| 3G 연결됨 | 네트워크 모드 LTE 우선 설정 |
| 로밍패스 만료 | baro로 전환됨, 추가 구매 |

---

### 4. 문자가 안 보내짐/안 받아짐

#### SMS 발신 안 됨
1. 메시지 센터 번호 확인
2. 수신자 번호 국제 형식 확인 (+82-10-XXXX-XXXX)
3. 단말기 재부팅

#### SMS 수신 안 됨
1. 발신자가 국제 형식으로 보냈는지 확인
2. 스팸 차단 앱 확인
3. 저장 공간 확인

---

### 5. 로밍 요금이 예상보다 많이 나옴

#### 확인사항
1. **사용 내역 확인**
   - T월드 앱 > 로밍 > 사용내역

2. **예상치 못한 데이터 사용**
   - 앱 백그라운드 데이터
   - 자동 업데이트
   - 클라우드 동기화

3. **크루즈/항공기 로밍**
   - 매우 고가 (22,000원/MB)
   - 의도치 않게 연결됐을 수 있음

## 국가별 특이사항

### 중국
| 문제 | 원인 | 해결 |
|------|------|------|
| 구글/SNS 접속 불가 | 중국 인터넷 규제 | VPN 앱 사용 |
| 카카오톡 불안정 | 간헐적 차단 | VPN 또는 Wi-Fi 시도 |

### 일본
- 대부분 문제없이 사용 가능
- 지하철 등 일부 지역 음영지역

### 동남아
| 국가 | 특이사항 |
|------|----------|
| 태국 | 실내 커버리지 약함 |
| 베트남 | 일부 지역 3G만 가능 |
| 인도네시아 | 섬 지역 커버리지 제한 |

### 유럽
- EU 국가 간 이동 시 자동 전환
- 스위스/노르웨이는 EU 별도 요금

### 미국
- 뉴욕/LA 등 대도시는 양호
- 국립공원 등 오지 커버리지 제한

## 로밍 설정 초기화

### iPhone
1. 설정 > 일반 > 전송 또는 iPhone 재설정
2. 네트워크 설정 재설정
3. 재부팅 후 로밍 재설정

### Galaxy
1. 설정 > 일반 > 초기화
2. 네트워크 설정 초기화
3. 재부팅 후 로밍 재설정

**주의**: Wi-Fi 비밀번호 등 네트워크 설정이 삭제됩니다.

- [ ] 테스트 발신 (문자/통화)
- [ ] 로밍 시작 SMS 수신

## 자주 묻는 질문

### Q: 로밍 안 되면 114로 전화할 수 있나요?
A: 로밍이 완전히 안 되면 114 연결이 어렵습니다. +82-2-6343-9000으로 국제전화 또는 현지 Wi-Fi를 통해 T월드 앱 채팅 상담을 이용하세요.

### Q: 현지 통신사가 여러 개 표시되는데 뭘 선택해야 하나요?
A: SK텔레콤과 제휴된 통신사면 어떤 것이든 괜찮습니다. 신호가 가장 강한 통신사를 선택하세요. 요금은 동일합니다.

### Q: 로밍 중에 요금제 변경 가능한가요?
A: 가능하지만, 변경 사항이 즉시 반영되지 않을 수 있습니다. 귀국 후 변경하시는 것을 권장합니다.

### Q: 데이터 로밍만 끄고 통화/문자만 사용할 수 있나요?
A: 네, 설정에서 데이터 로밍만 OFF하면 됩니다. 통화/문자는 계속 사용 가능합니다.

### Q: eSIM인데 로밍이 되나요?
A: 네, eSIM도 일반 USIM과 동일하게 로밍 가능합니다.
