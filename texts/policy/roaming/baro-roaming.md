# baro 로밍 서비스 안내

> 최종 업데이트: 2024년 1월

## baro 로밍이란?

SK텔레콤 고객이 해외에서 별도 신청 없이 자동으로 이용할 수 있는 로밍 서비스입니다. 국내 요금제 기반으로 해외에서도 합리적인 요금으로 데이터/통화/문자를 사용할 수 있습니다.

---

## 자동 적용 대상

### 5G/LTE 요금제 가입 고객
- 별도 신청 없이 자동 활성화
- 출국 시 자동 로밍 시작
- 입국 시 자동 로밍 종료

### 제외 대상
- 선불폰
- 일부 알뜰폰(MVNO)
- 법인 요금제 일부

## baro 로밍 설정

### 출국 전 확인사항
1. **데이터 로밍 ON**
   - iPhone: 설정 > 셀룰러 > 데이터 로밍 ON
   - Galaxy: 설정 > 연결 > 모바일 네트워크 > 데이터 로밍 ON

2. **로밍 자동 연결 확인**
   - T월드 앱 > 로밍 > 자동 로밍 설정 확인

### 현지 도착 후
1. 비행기 모드 해제
2. 자동으로 현지 통신사 연결
3. SK텔레콤 로밍 시작 SMS 수신

---

## 데이터 절약 팁

### 꼭 필요한 설정
1. **앱 백그라운드 데이터 제한**
   - 설정에서 각 앱의 백그라운드 데이터 차단

2. **자동 업데이트 끄기**
   - App Store/Play Store 자동 업데이트 OFF
   - 앱 자동 동기화 OFF

3. **저화질 모드**
   - YouTube: 설정 > 화질 > 자동(저화질)
   - 넷플릭스: 설정 > 셀룰러 데이터 사용량 > 절약

### 데이터 모니터링
- T월드 앱에서 실시간 로밍 데이터 확인
- 설정한 용량 도달 시 알림 수신

## 요금 알림 서비스

### 자동 알림
| 시점 | 알림 내용 |
|------|----------|
| 로밍 시작 | 현지 도착 안내, 요금 안내 |
| 데이터 500MB | 사용량 알림 |
| 데이터 1GB | 일 정액 완료 알림 |
| 일 요금 3만원 | 과금 주의 알림 |

### 요금 한도 설정
- T월드 앱에서 일/월 요금 한도 설정 가능
- 한도 도달 시 데이터 자동 차단 (통화/문자 가능)

---

## 로밍 중 주의사항

### 데이터 사용
- Wi-Fi 이용 권장 (호텔, 카페 등)
- 대용량 다운로드는 Wi-Fi에서
- 영상통화는 데이터 많이 소모

### 통화
- 국내 발신 시: +82-10-XXXX-XXXX
- 현지 발신 시: 국가코드 + 번호
- 수신은 평소대로 (요금 발생 주의)

### 문자
- 국제문자 발신 시 건당 100원
- 수신은 무료

---

## 로밍 OFF 방법

로밍을 사용하지 않으려면:

### 데이터만 차단
- 설정 > 셀룰러 > 데이터 로밍 OFF
- 통화/문자는 가능, 데이터만 차단

### 완전 차단 (비행기 모드)
- 비행기 모드 ON
- Wi-Fi만 별도 활성화 가능

## 자주 묻는 질문

### Q: baro 로밍은 별도 신청이 필요한가요?
A: 아니요, 5G/LTE 요금제 가입 시 자동 적용됩니다. 별도 신청 없이 해외에서 바로 사용 가능합니다.

### Q: 일 1GB 초과하면 어떻게 되나요?
A: 400Kbps 속도로 무제한 사용 가능합니다. 추가 요금 없이 기본적인 메신저, 지도 검색은 가능합니다.

### Q: baro 로밍보다 저렴한 방법이 있나요?
A: 장기 체류 시 로밍패스 상품이 더 저렴할 수 있습니다. 국가별로 비교해 보세요.

### Q: 로밍 중 데이터 사용량은 어디서 확인하나요?
A: T월드 앱 > 로밍 > 사용량 조회에서 실시간 확인 가능합니다.

### Q: 현지 통신사가 자동 연결되지 않아요
A: 설정 > 네트워크 운영자 > 수동 선택에서 다른 통신사를 선택해 보세요.
