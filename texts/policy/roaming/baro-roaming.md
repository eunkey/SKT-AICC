# baro 로밍 서비스 상세

> 최종 업데이트: 2025년 2월

## baro 로밍이란?

SK텔레콤의 해외 로밍 브랜드로, 별도 설정 없이 해외에서 자동으로 통화·데이터 서비스를 이용할 수 있습니다.

---

## 자동 로밍

### 자동 로밍 개요
| 항목 | 내용 |
|------|------|
| 설정 | 별도 설정 불필요 (기본 활성화) |
| 작동 방식 | 해외 도착 시 현지 네트워크 자동 접속 |
| 요금 | 종량제 (사용한 만큼) |

### 자동 로밍 요금 (종량제)
| 구분 | 요금 |
|------|------|
| 음성 발신 | 국가별 1,300원~3,500원/분 |
| 음성 수신 | 국가별 700원~2,500원/분 |
| SMS 발신 | 200원~350원/건 |
| SMS 수신 | 무료 |
| 데이터 | 0.5원~22원/KB (국가별 상이) |

### 자동 로밍 장점
- 별도 신청/설정 불필요
- 즉시 사용 가능
- 단기간 적은 사용량에 적합

### 자동 로밍 단점
- 요금이 비쌈 (장기 체류 시)
- 데이터 요금 부담 큼

## 로밍 신청 방법

### T월드 앱
```
1. T월드 앱 > 로밍
2. 로밍 상품 선택
3. 여행 기간 설정
4. 결제 및 신청 완료
```

### 114 전화
```
1. 114 전화 > 로밍 서비스
2. 상담사 연결
3. 상품 상담 후 신청
```

### 공항 로밍센터
| 위치 | 영업시간 |
|------|----------|
| 인천공항 제1터미널 3층 | 06:00~22:00 |
| 인천공항 제2터미널 3층 | 06:00~22:00 |
| 김포공항 국제선 | 06:00~21:00 |

## 로밍 설정

### 데이터 로밍 설정

**Android**:
```
설정 > 연결 > 모바일 네트워크 > 데이터 로밍 > 켜기
```

**iPhone**:
```
설정 > 셀룰러 > 셀룰러 데이터 옵션 > 데이터 로밍 > 켜기
```

| 클라우드 자동 동기화 | 끄기 |
| 백그라운드 데이터 | 제한 |

## 로밍 요금 절약 팁

### 팁 1: 로밍패스 사전 신청
- 출국 전 T월드 앱에서 미리 신청
- 현지 도착 시 자동 적용

### 팁 2: Wi-Fi 적극 활용
- 호텔, 카페 등 Wi-Fi 이용
- 대용량 다운로드는 Wi-Fi에서

### 팁 3: 오프라인 지도 다운로드
- 구글맵 오프라인 지도 미리 저장
- 데이터 사용량 절약

### 팁 4: 데이터 절약 모드
- 앱 백그라운드 데이터 제한
- 자동 재생/업데이트 끄기

## 자주 묻는 질문 (FAQ)

**Q: 로밍 신청 안 했는데 해외에서 요금이 나왔어요.**
A: 자동 로밍이 기본 활성화되어 있습니다. 데이터 로밍 설정을 끄지 않으면 자동으로 과금됩니다.

**Q: 로밍패스 기간이 남았는데 귀국했어요.**
A: 귀국 후에는 로밍패스가 자동 종료됩니다. 미사용 기간 환불은 불가합니다.

**Q: 비행기 안에서 로밍이 되나요?**
A: 일부 항공사는 기내 Wi-Fi/로밍을 제공합니다. 별도 요금이 발생하며, SK텔레콤 로밍과는 다릅니다.

**Q: 해외에서 114 전화되나요?**
A: +82-2-6343-9000으로 연락하시면 됩니다. 국제전화 요금이 부과됩니다.

**Q: 현지 유심 vs 로밍패스 뭐가 좋나요?**
A: 장기 체류(2주 이상)는 현지 유심이, 단기 여행은 로밍패스가 편리합니다.
