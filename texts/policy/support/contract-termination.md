# 해지/번호이동 절차 및 위약금

> 최종 업데이트: 2025년 2월

## 해지 유형

### 일반 해지
| 항목 | 내용 |
|------|------|
| 정의 | SK텔레콤 서비스 완전 종료 |
| 번호 | 번호 소멸 |
| 방법 | 114 또는 매장 |

### 번호이동 해지
| 항목 | 내용 |
|------|------|
| 정의 | 타 통신사로 번호 유지 이동 |
| 번호 | 번호 유지 |
| 방법 | 타 통신사에서 신청 |

---

## 일반 해지 절차

### 해지 신청 방법

#### 전화 해지 (114)
```
1. 114 전화
2. "해지" 선택
3. 본인 확인 (ARS 또는 상담사)
4. 해지 사유 선택
5. 위약금 안내
6. 해지 확정
```

#### 매장 해지
```
1. T월드 직영점/대리점 방문
2. 신분증 지참
3. 해지 신청서 작성
4. 위약금 확인
5. 해지 완료
```

### 해지 불가 채널
- T월드 앱: 해지 불가 (정보 조회만)
- 온라인: 해지 불가

### 해지 시 필요 사항
| 항목 | 내용 |
|------|------|
| 본인 해지 | 신분증 |
| 대리인 해지 | 위임장 + 인감증명서 + 대리인 신분증 |
| 법정대리인 | 가족관계증명서 + 대리인 신분증 |

## 번호이동 절차

### 절차
```
1. 타 통신사 매장/온라인 방문
      ↓
2. 번호이동 신청
   - 현재 통신사: SK텔레콤
   - 이동할 번호 입력
      ↓
3. 본인 인증 (PASS 등)
      ↓
4. 요금제/단말기 선택
      ↓
5. 번호이동 완료 (1~2시간)
   - SK텔레콤 자동 해지
```

### 번호이동 소요 시간
| 시간대 | 소요 시간 |
|--------|-----------|
| 평일 09:00~18:00 | 약 1~2시간 |
| 야간/주말 | 다음 영업일 처리 |

### 번호이동 주의사항
- SK텔레콤에 별도 해지 신청 불필요
- 위약금은 SK텔레콤에서 청구
- USIM은 새 통신사에서 발급

## 위약금 안내

### 위약금 구성
```
총 위약금 = 약정 할인 반환금 + 지원금 반환금 + 설치비 잔여분
```

### 선택약정 할인 반환금

**계산 방식**:
```
반환금 = 월 할인액 × 잔여 개월 × 반환율

반환율:
- 계약 후 1개월~12개월: 잔여 할인의 40~100%
- 계약 후 13개월~24개월: 잔여 할인의 20~40%
```

**예시**: 5GX 스탠다드(89,000원), 24개월 약정, 12개월 사용 후 해지
```
월 할인액: 89,000 × 25% = 22,250원
잔여 개월: 12개월
잔여 할인: 22,250 × 12 = 267,000원
반환금: 약 150,000~200,000원 (반환율 적용)
```

### 공시지원금 반환금

**계산 방식**:
```
반환금 = 지원금 × (잔여일수 / 총약정일수)

약정 기간: 보통 24개월 (730일)
```

**예시**: 지원금 300,000원, 12개월 사용 후 해지
```
잔여일수: 365일
반환금 = 300,000 × (365/730) = 150,000원
```

### 약정 유형별 위약금 비교
| 약정 유형 | 해지 시 반환금 |
|-----------|----------------|
| 선택약정 | 잔여 할인금 일부 반환 |
| 공시지원금 | 잔여 기간 비례 반환 |
| 무약정 | 위약금 없음 |

## 위약금 면제/감면

### 위약금 면제 사유
| 사유 | 면제 범위 | 필요 서류 |
|------|-----------|-----------|
| 사망 | 전액 면제 | 사망진단서, 가족관계증명서 |
| 해외 이주 (1년+) | 전액 면제 | 출국 증빙 (항공권, 비자 등) |
| 천재지변 | 전액 면제 | 이재민 증명서 |
| 기초생활수급자 | 감면 | 수급자 증명서 |
| 장애 등급 (1~3급) | 감면 | 장애인 증명서 |

### 감면 신청
```
1. 114 전화 또는 매장 방문
2. 면제/감면 사유 설명
3. 증빙 서류 제출
4. 심사 후 결과 안내
```

## 해지 시 요금 정산

### 해지 월 요금
| 구분 | 청구 |
|------|------|
| 기본료 | 해지일까지 일할 계산 |
| 부가서비스 | 해지일까지 일할 계산 |
| 단말 할부금 | 잔여 할부금 일시 청구 또는 유지 |

### 해지 후 청구서
- 해지 익월에 최종 청구서 발송
- 위약금 포함 청구
- 잔여 할부금 포함 (일시 청구 선택 시)

### 미납 요금
- 해지 전 미납 요금 완납 필요
- 미납 상태에서 해지 시 신용 영향 가능

## 자주 묻는 질문 (FAQ)

**Q: 앱이나 온라인으로 해지할 수 없나요?**
A: 해지는 114 전화 또는 매장 방문만 가능합니다. 통신사 정책상 본인 확인 절차가 필요합니다.

**Q: 위약금을 정확히 알고 싶어요.**
A: 114로 전화하시면 본인 회선의 정확한 위약금을 안내받을 수 있습니다.

**Q: 번호이동하면 위약금을 새 통신사가 내주나요?**
A: 일부 프로모션에서 위약금 지원이 있을 수 있으나, 기본적으로 본인 부담입니다.

**Q: 단말 할부금이 남았는데 해지하면 어떻게 되나요?**
A: 잔여 할부금을 일시 납부하거나, 매월 계속 납부할 수 있습니다. 선택 가능합니다.

**Q: 가족이 대신 해지할 수 있나요?**
A: 위임장, 인감증명서, 대리인 신분증이 있으면 가능합니다.

**Q: 해지 후 같은 번호로 재가입 가능한가요?**
A: 번호이동 없이 일반 해지한 경우 해당 번호는 소멸되어 재사용이 어렵습니다.
