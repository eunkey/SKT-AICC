# 해지 및 번호이동 정책

> 최종 업데이트: 2024년 1월

## 해지 유형

### 일반 해지
| 항목 | 내용 |
|------|------|
| 정의 | SK텔레콤 서비스 완전 종료 |
| 번호 | 전화번호 반납 |
| 약정 | 위약금 발생 가능 |

### 번호이동 (MNP)
| 항목 | 내용 |
|------|------|
| 정의 | 타 통신사로 전환 |
| 번호 | 기존 번호 유지 |
| 약정 | 위약금 발생 가능 |

---

## 해지 절차

### 일반 해지
1. **114 고객센터 연락** (본인 확인)
2. **해지 상담** (위약금 안내, 혜택 안내)
3. **해지 확정** (본인 동의)
4. **처리 완료** (즉시 또는 월말)

### 번호이동 절차
1. **이동할 통신사 매장 방문**
2. **본인 확인** (신분증)
3. **번호이동 신청** (신청서 작성)
4. **개통 완료** (30분~2시간)
5. **SK텔레콤 자동 해지**

---

## 해지 시 필요 서류

### 본인 해지
| 방법 | 필요 서류 |
|------|----------|
| 114 전화 | 본인 확인만 |
| 매장 방문 | 신분증 |

### 대리인 해지
| 관계 | 필요 서류 |
|------|----------|
| 가족 | 위임장 + 가족관계증명서 + 양측 신분증 |
| 제3자 | 위임장 (인감) + 인감증명서 + 양측 신분증 |

### 사망자 해지
- 사망자 신분증
- 사망진단서 또는 제적등본
- 신청인 신분증
- 가족관계증명서

---

## 위약금

### 선택약정 위약금
| 잔여 기간 | 위약금 비율 |
|----------|------------|
| 24~19개월 | 할인금액 × 50% |
| 18~13개월 | 할인금액 × 40% |
| 12~7개월 | 할인금액 × 30% |
| 6~1개월 | 할인금액 × 20% |

### 공시지원금 위약금
| 잔여 기간 | 반환 비율 |
|----------|----------|
| 전체 | 지원금 × (잔여일수 / 730) |

### 위약금 계산 예시
**상황**: 5G 스탠다드 선택약정, 12개월 사용 후 해지

1. 월 할인액: 21,250원
2. 받은 총 할인: 21,250원 × 12개월 = 255,000원
3. 위약금 비율 (잔여 12개월): 30%
4. **위약금: 255,000원 × 30% = 76,500원**

## 위약금 면제

### 면제 사유
| 사유 | 필요 서류 |
|------|----------|
| 해외 이주 | 출국 예정 증명서, 비자 |
| 군 입대 | 입영 통지서 |
| 사망 | 사망진단서 |
| 서비스 불가 지역 이전 | 주소 증빙 |
| 질병 (6개월 이상) | 진단서 |

### 부분 감면
- 장기 고객 감면 (3년 이상)
- 이동전화 + 인터넷 결합 고객
- 가족 결합 유지 고객

## 해지 시 유의사항

### 해지 전 확인
- [ ] 약정 잔여 기간 및 위약금
- [ ] 잔여 포인트 (해지 시 소멸)
- [ ] 할부금 잔액 (해지와 무관하게 납부)
- [ ] 결합 상품 할인 중단 여부

### 해지 후 처리
| 항목 | 처리 |
|------|------|
| T포인트 | 즉시 소멸 |
| 멤버십 | 즉시 종료 |
| 부가서비스 | 자동 해지 |
| 미납 요금 | 별도 청구 |

---

## 번호이동 안내

### 번호이동 준비물
| 항목 | 내용 |
|------|------|
| 신분증 | 주민등록증, 운전면허증, 여권 |
| 인증수단 | 기존 폰 (인증번호 수신) |
| 결제수단 | 신용카드 또는 계좌 |

### 번호이동 소요 시간
| 시간대 | 처리 시간 |
|--------|----------|
| 평일 주간 | 30분~1시간 |
| 야간/주말 | 1~2시간 |

### 번호이동 중 주의
- 이동 처리 중 통화/데이터 불가 (수십 분)
- 중요한 통화 예정 있으면 피하기
- USIM 교체 필요

---

## 번호이동 후

### SK텔레콤 처리
- 자동 해지 처리
- 위약금 청구 (해당 시)
- 마지막 청구서 발송

### 확인 사항
- 기존 USIM 보관 (일정 기간)
- 최종 청구서 확인
- 결합 할인 해제 확인

---

## 해지 번복 (취소)

### 해지 취소 가능 기간
| 상황 | 취소 가능 |
|------|----------|
| 해지 신청 당일 | 가능 |
| 해지 처리 완료 후 | 불가 |
| 번호이동 완료 후 | 불가 |

### 재가입
- 해지 후 동일 번호 재사용: 불가
- 새 번호로 재가입: 가능
- 번호이동으로 복귀: 가능 (타사 경유)

## 자주 묻는 질문

### Q: 해지 시 할부금은 어떻게 되나요?
A: 해지와 관계없이 할부금은 계속 납부해야 합니다. 일시불 전환도 가능합니다.

### Q: 해지 후에도 청구서가 오는 이유는?
A: 해지 전까지 사용한 요금과 위약금이 청구됩니다. 보통 해지 익월에 최종 청구됩니다.

### Q: 번호이동과 해지 중 뭐가 유리한가요?
A: 번호를 유지하고 싶다면 번호이동, 번호가 필요 없다면 해지입니다. 위약금은 동일합니다.

### Q: 해지 예약이 가능한가요?
A: 네, 특정 날짜에 해지되도록 예약 가능합니다. 월말 해지를 많이 선택합니다.

### Q: 가족 결합 중인데 해지하면?
A: 본인 회선만 결합에서 빠지고, 나머지 가족 할인은 회선 수에 맞게 재계산됩니다.
