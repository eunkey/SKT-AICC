# 주소 변경 및 이전 안내

> 최종 업데이트: 2024년 1월

## 개요

SK텔레콤 서비스 이용 중 주소가 변경된 경우, 요금 통지서 수령 및 각종 서비스 이용을 위해 주소 변경이 필요합니다.

---

## 주소 변경 대상

### 변경 가능한 주소
| 구분 | 설명 |
|------|------|
| 청구지 주소 | 요금 통지서 수령 주소 |
| 설치 주소 | 인터넷/IPTV 설치 주소 (해당 시) |
| 배송지 주소 | 단말기/유심 배송 주소 |
| 연락처 주소 | 기본 등록 주소 |

### 주의사항
- 휴대폰 서비스는 **청구지 주소**만 변경 가능
- 인터넷/IPTV는 이전 설치 절차 필요 (별도 신청)
- 주소 변경과 명의 변경은 별개 절차

---

## 주소 변경 방법

### 방법 1: T월드 앱 (권장)

**절차:**
1. T월드 앱 로그인
2. **전체메뉴** > **내 정보 관리**
3. **주소 변경** 선택
4. 현재 주소 확인
5. **변경하기** 터치
6. 새 주소 검색 (도로명/지번)
7. 상세 주소 입력 (동, 호수 등)
8. **변경 완료** 터치

**소요 시간:** 즉시 반영

**적용 시점:** 다음 청구서부터 적용

---

### 방법 2: 114 고객센터

**절차:**
1. 114 전화
2. ARS 1번 (요금 관련) 선택
3. 상담원 연결 요청
4. 본인 확인 (가입자명, 생년월일, 연락처)
5. 새 주소 안내
6. 변경 완료 확인

**운영 시간:**
- 평일: 09:00 ~ 18:00
- 토요일: 09:00 ~ 12:00
- 일요일/공휴일: 휴무

**소요 시간:** 통화 중 즉시 처리

---

### 방법 3: 매장 방문

**준비물:**
- 신분증 (주민등록증, 운전면허증, 여권)
- 본인 방문 필수 (대리인 불가)

**절차:**
1. 가까운 SK텔레콤 매장 방문
2. 번호표 발급
3. 신분증 제시 및 본인 확인
4. 새 주소 안내
5. 변경 완료 확인서 수령 (선택)

**매장 찾기:** T월드 앱 > 매장 찾기

---

## 주소 입력 방식

### 도로명 주소 (권장)
```
서울특별시 강남구 테헤란로 427
경기도 성남시 분당구 판교역로 166
부산광역시 해운대구 센텀중앙로 55
```

### 지번 주소
```
서울특별시 강남구 삼성동 157-27
경기도 성남시 분당구 삼평동 670
부산광역시 해운대구 우동 1495
```

### 상세 주소 예시
| 건물 유형 | 상세 주소 예시 |
|----------|---------------|
| 아파트 | 삼성아파트 101동 1201호 |
| 오피스텔 | SK허브 1805호 |
| 빌라/연립 | 302호 |
| 단독주택 | (상세주소 없음) |
| 상가/사무실 | 3층 301호 |

## 인터넷/IPTV 이전 설치

### 이전 설치란?
기존 인터넷/IPTV 서비스를 새 주소로 옮기는 것입니다. 단순 주소 변경이 아닌 **이전 설치 신청**이 필요합니다.

### 이전 설치 절차
1. **신청:** 114 또는 T월드 앱에서 이전 설치 신청
2. **설치 가능 여부 확인:** 새 주소에서 서비스 가능 여부 확인
3. **설치 일정 예약:** 희망 날짜/시간 선택
4. **기존 장비 철거:** 이전 주소 장비 수거 (설치 기사 방문)
5. **신규 설치:** 새 주소에 장비 설치

### 이전 설치 비용
| 구분 | 비용 |
|------|------|
| 기본 이전 설치비 | 무료 (약정 유지 시) |
| 추가 배선 공사 | 실비 청구 가능 |
| 장비 교체 | 무료 (노후 장비 시) |

### 소요 기간
- 신청 후 3~7일 내 설치
- 설치 집중 기간(이사철)에는 지연 가능

## 주소 변경 시 주의사항

### 자동 변경되지 않는 항목
| 항목 | 별도 변경 필요 |
|------|---------------|
| 우체국 전입신고 | 주민센터에서 별도 신고 |
| 카드사 청구지 | 각 카드사 앱에서 변경 |
| 은행 주소 | 각 은행 앱에서 변경 |
| 타 통신사 주소 | 해당 통신사에서 변경 |

### 변경 전 확인사항
- 새 주소가 정확한지 재확인
- 상세주소(동/호수) 누락 없는지 확인
- 우편번호 자동 입력 확인

### 변경 후 확인사항
- 변경 완료 문자 수신 확인
- T월드 앱에서 변경된 주소 확인
- 다음 달 요금 통지서 수령 확인

## 자주 묻는 질문

### Q: 주소 변경 후 언제부터 적용되나요?
A: 변경 즉시 시스템에 반영됩니다. 단, 요금 통지서는 다음 청구 주기부터 새 주소로 발송됩니다. 이미 발송된 통지서는 기존 주소로 배달됩니다.

### Q: 이사했는데 주소를 안 바꾸면 어떻게 되나요?
A: 요금 통지서가 기존 주소로 발송되어 수령하지 못할 수 있습니다. 요금 납부에는 영향이 없지만, 중요한 안내문을 받지 못할 수 있으니 변경을 권장합니다.

### Q: 가족 명의 회선 주소도 변경할 수 있나요?
A: 아니요, 주소 변경은 가입자 본인만 가능합니다. 가족 회선의 주소 변경은 해당 명의자가 직접 변경해야 합니다.

### Q: 해외 주소로 변경할 수 있나요?
A: 요금 통지서 수령 주소는 국내 주소만 등록 가능합니다. 해외 거주 시 이메일 또는 T월드 앱으로 청구서를 받으시기 바랍니다.

### Q: 인터넷 이전 시 기존 전화번호/IP는 유지되나요?
A: 인터넷 전화번호는 지역번호가 같으면 유지 가능합니다. IP 주소는 변경될 수 있습니다.

### Q: 주소 변경 시 위약금이 있나요?
A: 단순 주소 변경은 위약금이 없습니다. 단, 인터넷/IPTV 이전 설치 시 새 주소에서 서비스가 불가능한 경우에만 해지 위약금이 발생할 수 있습니다.
