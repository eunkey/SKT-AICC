# T휴대폰 보험 가입/보상 정책

> 최종 업데이트: 2025년 2월

## T휴대폰 보험 개요

SK텔레콤에서 제공하는 휴대폰 보험 서비스로, 분실/파손/도난 시 보상받을 수 있습니다.

---

## 보험 상품 종류

### T휴대폰 보험 플러스
| 항목 | 내용 |
|------|------|
| 월 보험료 | 출고가에 따라 6,600원~9,900원 |
| 보상 한도 | 출고가 100% |
| 보상 횟수 | 연 3회 |
| 보상 범위 | 분실, 도난, 파손, 침수, 화재 |
| 자기부담금 | 20~30% |

### T휴대폰 보험 라이트
| 항목 | 내용 |
|------|------|
| 월 보험료 | 3,300원~5,500원 |
| 보상 한도 | 출고가 70% |
| 보상 횟수 | 연 2회 |
| 보상 범위 | 파손, 침수 (분실 미포함) |
| 자기부담금 | 30% |

### 상품 비교
| 항목 | 플러스 | 라이트 |
|------|--------|--------|
| 월 보험료 | 높음 | 낮음 |
| 분실/도난 | ✅ | ❌ |
| 파손/침수 | ✅ | ✅ |
| 보상 한도 | 100% | 70% |
| 연간 횟수 | 3회 | 2회 |

---

## 가입 조건

### 가입 시점
| 시점 | 가입 가능 |
|------|-----------|
| 단말기 구매 시 | ✅ (권장) |
| 구매 후 30일 이내 | ✅ |
| 구매 후 30일 초과 | ❌ 불가 |

### 가입 대상 단말기
- SK텔레콤에서 정상 개통된 단말기
- 공시지원금/선택약정 모두 가능
- 중고폰, 자급제폰 가입 불가

### 가입 방법
| 방법 | 절차 |
|------|------|
| 매장 | 개통 시 동시 신청 |
| 114 | 구매 30일 이내 전화 신청 |
| T월드 앱 | 부가서비스 > 보험 가입 |

---

## 보상 범위

### 보상 대상 사고
| 사고 유형 | 플러스 | 라이트 |
|-----------|--------|--------|
| 분실 | ✅ | ❌ |
| 도난 | ✅ | ❌ |
| 파손 (화면 깨짐) | ✅ | ✅ |
| 침수 | ✅ | ✅ |
| 화재 | ✅ | ✅ |
| 고장 (자연 불량) | ❌ | ❌ |

### 보상 제외 사항
- 고의로 인한 파손
- 제조사 보증 기간 내 자연 고장
- 음주/약물 상태에서의 사고
- 전쟁/테러로 인한 손해
- 소프트웨어 문제

---

## 보상 절차

### 파손/침수 보상
```
1. 사고 발생
      ↓
2. 보험 청구 신청
   - 114 전화
   - T월드 앱 > 보험 청구
      ↓
3. 단말기 점검
   - 서비스센터 방문 또는 택배
      ↓
4. 심사 (1~3영업일)
      ↓
5. 보상금 지급 또는 동일/유사 기종 제공
```

### 분실/도난 보상 (플러스만 해당)
```
1. 분실/도난 발생
      ↓
2. 분실 신고 (114)
      ↓
3. 경찰 신고 (도난 시)
      ↓
4. 보험 청구 신청
   - 필요 서류: 분실 확인서, 도난 신고서 (해당 시)
      ↓
5. 심사 (3~5영업일)
      ↓
6. 동일/유사 기종 제공 + 자기부담금 납부
```

## 보상금 계산

### 자기부담금
| 보험 상품 | 자기부담금 |
|-----------|------------|
| 플러스 | 보상금의 20~30% |
| 라이트 | 보상금의 30% |

### 보상금 예시

**Galaxy S24 (출고가 1,155,000원) 파손 시**

플러스 가입:
```
보상 한도: 1,155,000원 (100%)
자기부담금: 231,000원 (20%)
실 보상액: 924,000원
```

라이트 가입:
```
보상 한도: 808,500원 (70%)
자기부담금: 242,550원 (30%)
실 보상액: 565,950원
```

### 감가상각
- 가입 후 시간 경과에 따라 보상금 감소
- 2년 후 보상금 약 50% 수준

---

## 보상 방법

### 수리 보상
- 수리 가능한 파손의 경우
- 수리비를 보상금으로 지급
- 서비스센터 연계

### 대체폰 제공
- 수리 불가 또는 분실/도난의 경우
- 동일 기종 또는 동급 기종 제공
- 자기부담금 선납 후 수령

### 대체폰 제공 기준
| 상황 | 제공 기종 |
|------|-----------|
| 동일 기종 재고 있음 | 동일 기종 (리퍼 또는 신품) |
| 동일 기종 재고 없음 | 유사 스펙 기종 |
| 단종 모델 | 후속 모델 또는 유사 가격대 |

## 보험료 안내

### 월 보험료 (플러스 기준)
| 출고가 | 월 보험료 |
|--------|-----------|
| 50만원 미만 | 6,600원 |
| 50~100만원 | 7,700원 |
| 100~150만원 | 8,800원 |
| 150만원 이상 | 9,900원 |

### 보험료 납부
- 휴대폰 요금에 합산 청구
- 자동 납부

---

## 가입/해지

### 해지 방법
| 방법 | 절차 |
|------|------|
| 114 | 상담사에게 해지 요청 |
| T월드 앱 | 부가서비스 > 보험 해지 |
| 매장 | 방문하여 해지 신청 |

### 해지 시 주의사항
- 해지 후 재가입 불가
- 보상 진행 중 해지 불가
- 해지 월 보험료는 일할 계산

### 자동 해지 조건
| 조건 | 해지 시점 |
|------|-----------|
| 회선 해지 | 해지 시 자동 종료 |
| 기기변경 | 기존 보험 종료, 신규 가입 필요 |
| 번호이동 | 해지 시 자동 종료 |

## 자주 묻는 질문 (FAQ)

**Q: 보험 없이 단말기를 샀는데 지금 가입할 수 있나요?**
A: 구매 후 30일 이내라면 가입 가능합니다. 30일이 지나면 가입이 불가합니다.

**Q: 화면만 깨졌는데 보험 청구해도 되나요?**
A: 네, 파손에 해당하므로 보험 청구가 가능합니다.

**Q: 자기부담금은 언제 내나요?**
A: 대체폰 수령 시 또는 보상금 지급 시 차감됩니다.

**Q: 보험 청구하면 보험료가 올라가나요?**
A: 아니요, 청구 횟수와 관계없이 보험료는 동일합니다.

**Q: 기기변경하면 보험은 어떻게 되나요?**
A: 기존 보험은 자동 해지되며, 새 단말기에 대해 신규 가입이 필요합니다.

**Q: Apple Care+와 T휴대폰 보험 중 뭐가 좋나요?**
A: T휴대폰 보험 플러스는 분실도 보상합니다. Apple Care+는 분실 보상이 없지만 수리 품질이 우수합니다. 분실 우려가 크다면 T휴대폰 보험을 권장합니다.
