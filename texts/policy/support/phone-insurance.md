# T휴대폰 보험 정책

> 최종 업데이트: 2024년 1월

## T휴대폰 보험이란?

휴대폰 분실, 파손, 고장 시 보상받을 수 있는 SK텔레콤 공식 보험 서비스입니다.

## 보험 가입

### 가입 조건
| 항목 | 조건 |
|------|------|
| 가입 시점 | 단말기 구매 후 30일 이내 |
| 단말기 상태 | 정상 작동 (파손/고장 없음) |
| 중고폰 | 가입 불가 |
| 법인폰 | 가입 가능 |

### 가입 방법
| 방법 | 연락처 |
|------|--------|
| 매장 | 구매 시 동시 가입 |
| 114 | 고객센터 전화 |
| T월드 앱 | 부가서비스 > 보험 |

### 가입 불가 사유
- 구매 후 30일 경과
- 중고폰, 리퍼폰
- 이미 파손/고장 상태
- 분실 상태

## 보상 범위

### 보상 대상
| 사고 유형 | 프리미엄/스탠다드 | 라이트 |
|----------|-----------------|--------|
| 분실 | O | X |
| 도난 | O | X |
| 파손 (화면 깨짐) | O | O |
| 침수 | O | O |
| 고장 (비정상 작동) | O | O |

### 보상 제외
- 고의적 파손
- 자연 마모, 변색
- 소프트웨어 오류
- 배터리 수명 저하
- 분해/개조에 의한 손상
- 보험 가입 전 발생한 파손

## 보상 금액 계산

### 감가상각 적용
| 사용 기간 | 보상 비율 |
|----------|----------|
| 1개월 미만 | 100% |
| 1~6개월 | 95% |
| 6~12개월 | 85% |
| 12~18개월 | 75% |
| 18~24개월 | 65% |
| 24개월 이상 | 55% |

### 보상금 계산 예시
**상황**: 프리미엄 보험, 1년 사용, 단말기 출고가 150만원, 화면 파손

1. 출고가: 1,500,000원
2. 감가 적용 (85%): 1,275,000원
3. 보상 한도 적용 (100만원): 1,000,000원
4. 자기부담금 (20%): -200,000원
5. **최종 보상금: 800,000원**

## 보상 청구 절차

### 1단계: 사고 신고
1. 114 고객센터 또는 보험사 연락
2. 사고 일시, 경위 설명
3. 접수 번호 발급

### 2단계: 서류 제출
| 사고 유형 | 필요 서류 |
|----------|----------|
| 분실 | 분실신고확인서 (경찰서 발급) |
| 도난 | 도난신고확인서 (경찰서 발급) |
| 파손/침수 | 손상된 단말기 (반납) |
| 고장 | 고장 단말기 (반납) |

### 3단계: 심사 및 승인
- 서류 심사: 2~3일
- 승인 통보: SMS 또는 전화
- 거절 시 사유 안내

### 4단계: 보상금 지급
| 방법 | 내용 |
|------|------|
| 계좌 입금 | 승인 후 3~5일 |
| 현장 보상 | 매장에서 새 단말기 할인 |

## 보험 해지

### 해지 방법
- 114 고객센터
- T월드 앱 > 부가서비스 > 해지

### 해지 시 유의사항
- 해지 후 재가입 불가 (동일 단말기)
- 해지 월 보험료 일할 계산 환급
- 보상 청구 후 1개월 내 해지 불가

## 자주 묻는 질문

### Q: 화면만 깨져도 보상받을 수 있나요?
A: 네, 화면 파손도 보상 대상입니다. 수리비 또는 보상금을 받을 수 있습니다.

### Q: 보험 가입 후 바로 보상 청구 가능한가요?
A: 가입 후 15일 이후 발생한 사고부터 보상 가능합니다 (대기 기간).

### Q: 수리 받고 보상 청구해도 되나요?
A: 수리 후 영수증으로 실비 보상 청구 가능합니다. 단, 보상 한도와 자기부담금이 적용됩니다.

### Q: 분실 보상 후 휴대폰을 찾으면 어떻게 되나요?
A: 보상금을 반환하거나, 찾은 휴대폰을 반납해야 합니다.

### Q: AppleCare+와 중복 가입 가능한가요?
A: 가능합니다. 단, 보상은 하나만 선택해서 받을 수 있습니다.
