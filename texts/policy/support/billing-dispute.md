# 요금 이의제기 처리 절차

> 최종 업데이트: 2024년 1월

## 이의제기 대상

### 이의제기 가능 항목
| 항목 | 예시 |
|------|------|
| 요금 오류 | 청구 금액 계산 오류 |
| 중복 청구 | 동일 항목 이중 청구 |
| 미사용 서비스 | 신청하지 않은 부가서비스 |
| 로밍 요금 | 예상치 못한 고액 로밍비 |
| 소액결제 | 인지 못한 결제 내역 |
| 데이터 초과 | 비정상적 데이터 사용량 |

### 이의제기 불가 항목
- 정상 사용한 서비스 요금
- 약정 위약금 (정당한 해지 시)
- 명시적으로 동의한 부가서비스

---

## 이의제기 절차

### 1단계: 내역 확인
1. T월드 앱 > 요금 > 상세 내역
2. 문제 항목 확인
3. 스크린샷 저장

### 2단계: 이의제기 신청
| 방법 | 연락처 |
|------|--------|
| 114 고객센터 | 114 > 1번 > 상담원 연결 |
| T월드 앱 | 고객지원 > 요금 문의 |
| 매장 방문 | 신분증 지참 |

### 3단계: 접수 및 조사
- 접수 번호 발급
- 담당 부서 조사 (3~7일)
- 필요 시 추가 자료 요청

### 4단계: 결과 통보
- SMS 또는 전화로 결과 안내
- 환불 또는 기각 결정
- 이의 시 재심 가능

---

## 유형별 처리

### 부가서비스 오청구
**상황**: 신청하지 않은 부가서비스 요금 청구

**처리**:
1. 가입 경위 조사
2. 본인 동의 없으면 전액 환불
3. 동의 있으면 해지만 가능

**필요 자료**: 없음 (SK텔레콤 조사)

---

### 소액결제 분쟁
**상황**: 모르는 소액결제 내역

**처리**:
1. 결제 내역 확인
2. CP사(콘텐츠 제공사) 확인
3. 본인 결제 아닌 경우 환불 협의

**필요 자료**:
- 결제 당시 단말기 소지 증명
- 경찰 신고 (사기 의심 시)

---

### 로밍 고액 요금
**상황**: 예상보다 높은 로밍 요금

**처리**:
1. 사용 내역 확인
2. 로밍 안내 수신 여부 확인
3. 의도치 않은 사용 입증 시 감면 협의

**필요 자료**:
- 여행 일정표
- 데이터 로밍 설정 상태 증명

---

### 데이터 이상 사용
**상황**: 비정상적으로 많은 데이터 사용량

**처리**:
1. 사용 패턴 분석
2. 앱별 사용량 확인
3. 시스템 오류 확인 시 감면

**필요 자료**:
- 단말기 데이터 사용량 스크린샷
- 해당 기간 단말기 사용 상태

---

## 환불 처리

### 환불 방법
| 방법 | 내용 |
|------|------|
| 요금 차감 | 다음 달 청구 요금에서 차감 |
| 계좌 환불 | 등록 계좌로 입금 |
| 포인트 전환 | T포인트로 전환 |

### 환불 기간
| 결정 후 | 처리 기간 |
|---------|----------|
| 요금 차감 | 다음 청구 시 |
| 계좌 환불 | 7~14일 |
| 포인트 | 즉시 |

---

## 불복 시 대응

### 내부 재심
1. 1차 결정에 불복 시
2. 재심 요청 (14일 이내)
3. 상위 부서 검토

### 외부 기관 신고
| 기관 | 연락처 | 역할 |
|------|--------|------|
| 방송통신위원회 | 1335 | 통신 분쟁 조정 |
| 한국소비자원 | 1372 | 소비자 피해 구제 |
| 통신분쟁조정위원회 | 02-500-9000 | 분쟁 조정 |

### 법적 절차
- 소액사건심판 (2,000만원 이하)
- 민사소송

---

## 예방 방법

### 요금 모니터링
- T월드 앱 알림 설정
- 월 요금 알리미 설정
- 데이터 한도 알림 설정

### 소액결제 차단
1. T월드 앱 > 소액결제
2. 결제 한도 조정 또는 차단
3. 비밀번호 설정

### 부가서비스 확인
- 월 1회 가입 내역 확인
- 무료 체험 자동 전환 주의
- 불필요한 서비스 해지

## 자주 묻는 질문

### Q: 이의제기하면 요금을 안 내도 되나요?
A: 분쟁 중에도 요금 납부 의무가 있습니다. 환불 결정 시 돌려받습니다. 단, 분쟁 금액만큼 유예 요청 가능합니다.

### Q: 이의제기 기한이 있나요?
A: 청구일로부터 6개월 이내에 제기해야 합니다. 이후는 처리가 어려울 수 있습니다.

### Q: 증거가 없어도 이의제기 가능한가요?
A: 가능합니다. SK텔레콤 시스템에서 조사를 진행합니다. 단, 증거가 있으면 처리가 빠릅니다.

### Q: 환불받으면 세금계산서도 수정되나요?
A: 환불 금액에 대해 수정 세금계산서가 발행됩니다.
