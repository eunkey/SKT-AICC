# 요금 이의제기 처리 절차

> 최종 업데이트: 2025년 2월

## 요금 이의제기란?

청구된 요금에 대해 이상이 있다고 생각될 때 SK텔레콤에 이의를 제기하고 검토를 요청하는 절차입니다.

---

## 이의제기 가능한 경우

### 주요 이의제기 사유
| 사유 | 예시 |
|------|------|
| 과다 청구 | 사용하지 않은 서비스 요금 청구 |
| 부가서비스 | 신청하지 않은 부가서비스 과금 |
| 데이터 요금 | 비정상적인 데이터 사용량 |
| 해외 로밍 | 예상치 못한 로밍 요금 |
| 소액결제 | 본인이 하지 않은 결제 |
| 요금제 적용 | 잘못된 요금제 적용 |
| 할인 미적용 | 결합/약정 할인 누락 |

### 이의제기 불가능한 경우
| 사유 | 설명 |
|------|------|
| 정상 사용 | 본인이 정상적으로 사용한 요금 |
| 약관 동의 | 동의한 약관에 따른 과금 |
| 자녀/가족 사용 | 회선 명의자 책임 |
| 기간 경과 | 청구일로부터 6개월 초과 |

---

## 이의제기 절차

### 1단계: 이의제기 신청
```
신청 방법:
- 114 전화 > "요금 이의제기" 선택
- T월드 앱 > 고객센터 > 1:1 문의 > 요금 이의
- 매장 방문 > 이의제기 서면 신청
```

### 2단계: 접수 및 확인
```
- 접수 즉시 접수 번호 발급
- 담당자 배정 (1영업일 이내)
- 고객에게 접수 안내 SMS 발송
```

### 3단계: 사실 확인 및 검토
```
- 통화/데이터 사용 이력 분석
- 시스템 오류 여부 확인
- 부가서비스 가입 이력 확인
- 필요 시 고객 추가 확인
```

### 4단계: 결과 통보
```
- 검토 완료 후 결과 안내 (전화/SMS)
- 인정 시: 요금 조정 및 환불
- 불인정 시: 사유 설명 및 재심 안내
```

---

## 처리 기간

| 이의 유형 | 처리 기간 |
|-----------|-----------|
| 단순 문의 | 즉시~1영업일 |
| 일반 이의제기 | 3~7영업일 |
| 복잡한 사안 | 7~14영업일 |
| 외부 기관 연계 | 30영업일 이상 |

**처리 지연 시**:
- 중간 진행 상황 안내
- 예상 완료일 통보

---

## 이의제기 유형별 안내

### 부가서비스 요금 이의

**확인 사항**:
- 가입 일시 및 경로 (앱/웹/매장/ARS)
- 가입 시 약관 동의 여부
- 사용 이력

**처리 방향**:
| 상황 | 처리 |
|------|------|
| 본인 가입 확인 | 이의 불인정 (정상 과금) |
| 가입 이력 불명확 | 조정 가능 |
| 시스템 오류 확인 | 전액 환불 |

### 데이터 과다 사용 이의

**확인 사항**:
- 사용 시간대, 위치
- 앱별 사용량
- 단말기 설정 (백그라운드 데이터)

**처리 방향**:
| 상황 | 처리 |
|------|------|
| 정상 사용 확인 | 이의 불인정 |
| 시스템 오류 | 요금 조정 |
| 앱 오작동 의심 | 부분 조정 가능 |

### 해외 로밍 요금 이의

**확인 사항**:
- 로밍 설정 여부
- 로밍패스 가입 여부
- 경유지 자동 접속 여부

**처리 방향**:
| 상황 | 처리 |
|------|------|
| 로밍 설정 ON 상태 | 이의 불인정 |
| 로밍패스 적용 오류 | 전액 조정 |
| 경유지 자동 접속 | 부분 조정 가능 |

### 소액결제 이의

**확인 사항**:
- 결제 내역 (일시, 금액, 업체)
- 본인 결제 여부
- 타인 사용 가능성

**처리 방향**:
| 상황 | 처리 |
|------|------|
| 본인 결제 | 이의 불인정, 업체 환불 안내 |
| 부정 결제 의심 | 경찰 신고 후 조정 검토 |
| 자녀 결제 | 명의자 책임 (원칙) |

---

## 환불 방법

### 환불 유형
| 유형 | 내용 |
|------|------|
| 익월 차감 | 다음 달 요금에서 차감 |
| 계좌 환불 | 본인 명의 계좌로 입금 |
| 즉시 차감 | 미납 요금에서 차감 |

### 환불 신청
```
1. 이의제기 인정 시 환불 방법 선택
2. 계좌 환불 시 본인 명의 계좌 정보 제공
3. 환불 처리 (3~5영업일)
```

## 자주 묻는 질문 (FAQ)

**Q: 이의제기하면 요금을 안 내도 되나요?**
A: 이의제기와 관계없이 청구 요금은 납부해야 합니다. 이의가 인정되면 환불됩니다.

**Q: 몇 개월 전 요금도 이의제기 가능한가요?**
A: 원칙적으로 청구일로부터 6개월 이내 건에 대해 이의제기가 가능합니다.

**Q: 이의제기 결과에 불복하면 어떻게 하나요?**
A: 재검토 요청을 하거나, 외부 기관(통신분쟁조정위원회, 소비자원)에 조정을 신청할 수 있습니다.

**Q: 가족이 쓴 요금도 이의제기 되나요?**
A: 명의자가 책임지는 것이 원칙입니다. 다만 사전 동의 없는 부가서비스 등은 이의제기 가능합니다.

**Q: 이의제기하면 불이익이 있나요?**
A: 이의제기로 인한 불이익은 없습니다. 정당한 권리입니다.
