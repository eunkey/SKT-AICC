# 분실/도난 신고 및 정지 정책

> 최종 업데이트: 2024년 1월

## 분실 신고 방법

### 114 고객센터 (24시간)
1. 114 전화 (다른 전화에서)
2. ARS 3번 (분실/정지)
3. 즉시 정지 처리
4. 담당자 연결 시 추가 안내

### T월드 앱
1. T월드 앱 실행 (다른 기기에서)
2. 고객지원 > 분실/정지
3. 분실 신고 선택
4. 본인 인증 후 즉시 정지

### 경찰 신고 (도난 시)
- 경찰청: 112
- 분실신고 확인서 발급
- 보험 청구 시 필요

## 이용 정지 종류

### 분실 정지
| 항목 | 내용 |
|------|------|
| 적용 대상 | 분실/도난 휴대폰 |
| 정지 내용 | 발신/수신 전면 차단 |
| 요금 | 정지 기간 요금 미발생 |
| 기간 | 무제한 (해제 시까지) |

### 임시 정지
| 항목 | 내용 |
|------|------|
| 적용 대상 | 분실 의심 (찾을 가능성 있음) |
| 정지 내용 | 발신만 차단, 수신 가능 |
| 요금 | 기본료 50% 청구 |
| 기간 | 최대 1개월 |

---

## 정지 해제

### 해제 방법
| 방법 | 연락처 |
|------|--------|
| 114 고객센터 | 114 (다른 전화에서) |
| T월드 앱 | 고객지원 > 정지 해제 |
| 매장 방문 | 신분증 지참 |

### 해제 조건
- 본인 확인 필수 (신분증 또는 본인 인증)
- 분실 정지: 휴대폰 회수 시
- 도난 정지: 새 단말기로 재개통 시

## 분실폰 찾기 서비스

### 내 폰 찾기 (T안심)
| 항목 | 내용 |
|------|------|
| 서비스명 | T안심 분실폰 찾기 |
| 가입 | 무료 (사전 가입 필요) |
| 기능 | 위치 확인, 원격 잠금, 데이터 삭제 |

### 삼성 Find My Mobile
- 갤럭시폰 사용자
- 삼성 계정 로그인 필요
- findmymobile.samsung.com

### Apple 나의 찾기
- 아이폰 사용자
- Apple ID 로그인 필요
- icloud.com/find

## 분실 후 조치 순서

### 1단계: 즉시 정지
1. 114 전화 또는 T월드 앱
2. 분실 정지 요청
3. 부정 사용 방지

### 2단계: 위치 추적
1. 내 폰 찾기 서비스 이용
2. 마지막 위치 확인
3. 원격 잠금 설정

### 3단계: 경찰 신고 (도난 시)
1. 경찰서 방문 또는 112
2. 분실신고 확인서 발급
3. IMEI 번호 제출

### 4단계: 보험 확인
1. T휴대폰 보험 가입 여부 확인
2. 보상 청구 절차 진행
3. 새 단말기 구매

---

## 부정 사용 방지

### 정지 전 발생한 요금
| 상황 | 책임 |
|------|------|
| 정지 전 사용 | 사용자 부담 |
| 정지 후 사용 | SK텔레콤 차단 (요금 미발생) |

### IMEI 등록 (분실폰 차단)
- 분실 신고 시 IMEI 자동 등록
- 타 통신사에서도 개통 불가
- 전국 대리점에서 사용 불가

### IMEI 확인 방법
- 휴대폰: \*#06# 입력
- 박스: 바코드 스티커
- T월드 앱: 내 정보 > 단말기 정보

---

## 분실 후 단말기 구매

### 새 단말기 구매 시
1. 분실 정지 상태에서 매장 방문
2. 새 단말기 선택
3. 기존 번호로 재개통
4. USIM 신규 발급 (또는 eSIM)

### 보험 보상 이용 시
1. T휴대폰 보험 청구
2. 승인 후 보상금 수령
3. 새 단말기 구매

→ 상세 내용은 [T휴대폰 보험](./phone-insurance.md) 참조

## 자주 묻는 질문

### Q: 분실폰을 찾으면 어떻게 하나요?
A: 114 또는 T월드 앱에서 정지 해제하시면 됩니다. 본인 확인 후 즉시 정상 사용 가능합니다.

### Q: 분실 정지 중에도 요금이 나오나요?
A: 분실 정지 중에는 기본료가 청구되지 않습니다. 단, 정지 전 사용 요금은 청구됩니다.

### Q: 분실폰을 누가 습득해서 사용하면 요금은 누가 내나요?
A: 정지 전까지 사용한 요금은 명의자 부담입니다. 빠른 분실 신고가 중요합니다.

### Q: 보험 없이 분실하면 어떻게 하나요?
A: 새 단말기를 자비로 구매하셔야 합니다. 기기변경 또는 신규 구매 모두 가능합니다.

### Q: 분실폰이 해외에서 사용되면 로밍 요금이 나오나요?
A: 정지 전까지 사용된 요금은 청구될 수 있습니다. 분실 즉시 정지하세요.
