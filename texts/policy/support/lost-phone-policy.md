# 분실/도난 신고 및 정지 절차

> 최종 업데이트: 2025년 2월

## 분실 신고 개요

휴대폰을 분실하거나 도난당한 경우, 즉시 신고하여 서비스 정지 및 추가 피해를 방지해야 합니다.

---

## 분실 신고 방법

### 1. 전화 신고 (가장 빠름)
| 번호 | 용도 |
|------|------|
| 114 | SK텔레콤 휴대폰에서 |
| 1599-0011 | 유선/타사 휴대폰에서 |
| 080-011-6200 | 무료 (VIP 전용) |

**운영시간**: 24시간 (분실 신고)

### 2. T월드 앱 신고
```
1. 가족/지인 휴대폰에 T월드 앱 설치
2. 본인 인증 로그인
3. 더보기 > 분실/습득
4. 분실 신고 선택
5. 정지 유형 선택
6. 신고 완료
```

### 3. T월드 웹사이트
```
1. tworld.co.kr 접속
2. 로그인 (본인인증)
3. 고객센터 > 분실 신고
4. 분실 회선 선택
5. 정지 신청
```

### 4. 매장 방문
- T월드 직영점/대리점 방문
- 신분증 지참 필수
- 영업시간 내 방문

## 정지 유형

### 일시정지 (분실 정지)
| 항목 | 내용 |
|------|------|
| 용도 | 분실 후 찾을 가능성이 있을 때 |
| 요금 | 기본료 50% 부과 |
| 기간 | 최대 3개월 |
| 해제 | 언제든 가능 |
| 착신전환 | 설정 가능 |

### 완전정지 (도난 정지)
| 항목 | 내용 |
|------|------|
| 용도 | 도난 또는 찾을 가능성이 없을 때 |
| 요금 | 기본료 면제 |
| 기간 | 제한 없음 |
| 해제 | 본인 확인 후 가능 |
| 착신전환 | 불가 |

### 정지 유형 비교
| 구분 | 일시정지 | 완전정지 |
|------|----------|----------|
| 요금 | 50% 부과 | 면제 |
| 전화 수신 | 불가 | 불가 |
| 착신전환 | 가능 | 불가 |
| 해제 난이도 | 쉬움 | 본인확인 필요 |
| 추천 상황 | 잠시 분실 | 도난/완전 분실 |

## 분실 신고 후 절차

### 즉시 해야 할 일
```
1. 분실 신고 (114 또는 앱)
      ↓
2. 정지 유형 선택
      ↓
3. 착신전환 설정 (선택)
      ↓
4. 경찰 신고 (도난 시)
      ↓
5. 보험 청구 준비 (가입 시)
```

### 휴대폰 찾기 서비스
| 서비스 | 방법 |
|--------|------|
| 삼성 Find My Mobile | findmymobile.samsung.com |
| iPhone 나의 찾기 | icloud.com/find |
| 구글 내 기기 찾기 | google.com/android/find |

**기능**:
- 위치 확인
- 벨소리 울리기
- 원격 잠금
- 원격 데이터 삭제

## 정지 해제

### 해제 조건
| 정지 유형 | 해제 방법 |
|-----------|-----------|
| 일시정지 | 114, 앱, 웹에서 바로 해제 |
| 완전정지 | 본인 확인 후 114 또는 매장 |

### 해제 절차
```
1. 114 전화 또는 T월드 앱
2. 분실 정지 해제 요청
3. 본인 확인
4. 해제 완료 (즉시)
```

### 해제 후 주의사항
- 정지 기간 중 발생한 요금 정산
- USIM 정상 작동 확인
- 보험 청구 진행 중이면 해제 전 확인

## 도난 시 추가 조치

### 경찰 신고
| 방법 | 내용 |
|------|------|
| 112 신고 | 도난 즉시 |
| 경찰서 방문 | 도난 신고서 작성 |
| 증빙 서류 | 도난 신고 접수증 발급 |

**도난 신고서 용도**:
- 보험 청구 시 필요
- 부정 사용 분쟁 시 증거
- 위약금 감면 신청 시

### 결제 수단 보호
| 조치 | 방법 |
|------|------|
| 소액결제 차단 | 114에서 설정 |
| 모바일 카드 정지 | 각 카드사 연락 |
| 페이 서비스 정지 | 삼성페이/애플페이 등 원격 삭제 |

## 자주 묻는 질문 (FAQ)

**Q: 분실 신고는 몇 시까지 가능한가요?**
A: 114는 24시간 운영되며, 분실 신고는 언제든 가능합니다.

**Q: 정지 중에도 전화가 왔는지 알 수 있나요?**
A: 일시정지 시 착신전환을 설정하면 다른 번호로 전화를 받을 수 있습니다.

**Q: 분실 신고 전 누군가 요금을 썼어요.**
A: 분실 신고 전 사용 요금은 원칙적으로 고객 부담입니다. 도난 신고서가 있으면 일부 감면 협의가 가능합니다.

**Q: 휴대폰을 찾았는데 정지 해제는 어떻게 하나요?**
A: 114 전화 또는 T월드 앱에서 바로 해제할 수 있습니다.

**Q: 해외에서 분실했는데 어떻게 하나요?**
A: +82-2-6343-9000으로 연락하거나, T월드 앱(Wi-Fi 연결)으로 신고해 주세요.

**Q: 분실 정지 중 요금제 변경이 가능한가요?**
A: 정지 중에도 요금제 변경은 가능합니다.
