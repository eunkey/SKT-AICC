## 속도 안내

### 이론 최대 속도 vs 실제 속도
| 상품 | 이론 최대 | 실제 평균* |
|------|-----------|------------|
| 2.5Gbps | 2,500Mbps | 1,500~2,000Mbps |
| 1Gbps | 1,000Mbps | 800~950Mbps |
| 500Mbps | 500Mbps | 400~480Mbps |
| 100Mbps | 100Mbps | 85~95Mbps |

*실제 속도는 PC 성능, 공유기, 연결 방식(유선/무선)에 따라 달라집니다.

### 속도 측정 방법
```
1. 유선 연결 상태에서 측정 (가장 정확)
2. 속도 측정 사이트: fast.com 또는 speedtest.net
3. 여러 번 측정 후 평균값 확인
```

## 가입 방법

### 온라인 신청
```
1. skbroadband.com 접속
2. 인터넷 상품 선택
3. 주소 입력 및 개통 가능 여부 확인

```

### 전화 신청
- **전화번호**: 106
- **운영시간**: 평일 09:00~18:00

## 설치 안내

### 설치 절차
```
1. 설치 일정 확정 (1~3일 내)
      ↓
2. 설치 기사 방문 (약 1시간 소요)
      ↓
3. 광케이블/모뎀/공유기 설치
      ↓
4. 인터넷 연결 테스트
```

### 설치 필요 조건

## 자주 묻는 질문 (FAQ)

**Q: 인터넷 속도가 광고보다 느린데요?**
A: Wi-Fi 사용 시 속도 저하가 발생합니다. 정확한 측정은 유선 연결 상태에서 해주세요.

**Q: 공유기를 직접 사용해도 되나요?**
A: 가능합니다. 단, 기본 제공 공유기 반납 시 AS는 고객 부담입니다.

**Q: 아파트에서 기가 속도가 안 나옵니다.**
A: 아파트 내부 배선(랜선)이 구형이면 속도가 제한됩니다. 106으로 문의해 주세요.

**Q: 이사 시 인터넷은 어떻게 하나요?**
A: 106으로 이전 설치 신청하시면 됩니다. 이전비 22,000원이 발생합니다.

**Q: 결합 할인 중인데 인터넷만 해지하면요?**
A: 결합이 해제되어 휴대폰 할인도 사라집니다.
