# SK브로드밴드 결합상품 할인 정책

> 최종 업데이트: 2025년 2월

## 결합할인 개요

SK텔레콤 휴대폰과 SK브로드밴드 인터넷/B tv를 함께 사용하면 월 요금을 할인받을 수 있습니다.

---

## 온가족 프리

### 기본 구조
```
SK브로드밴드 인터넷 + SK텔레콤 휴대폰 = 결합 할인
```

### 휴대폰 결합 수에 따른 할인

| 결합 회선 | 인터넷 할인 | 휴대폰 할인 (합계) | 회선당 할인 |
|-----------|-------------|-------------------|-------------|
| 1회선 | 5,500원 | 5,500원 | 5,500원 |
| 2회선 | 5,500원 | 11,000원 | 5,500원 |
| 3회선 | 5,500원 | 16,500원 | 5,500원 |
| 4회선 | 5,500원 | 22,000원 | 5,500원 |
| 5회선 | 5,500원 | 27,500원 | 5,500원 |

### 할인 예시

**예시 1: 4인 가족 결합**
```
인터넷 (기가라이트): 33,000원 → 27,500원 (5,500원 할인)
휴대폰 4회선: 각 5,500원씩 총 22,000원 할인

총 할인: 27,500원/월
```

**예시 2: 2인 가구 결합**
```
인터넷: 5,500원 할인
휴대폰 2회선: 11,000원 할인

총 할인: 16,500원/월
```

## 신청 방법

### 신규 결합 신청
```
1. 106 전화 또는 T월드 매장 방문
2. 인터넷 가입자 정보 확인
3. 결합 대상 휴대폰 회선 지정
4. 가족 증빙 서류 제출 (타 명의 시)
5. 결합 완료 (익월 청구분부터 적용)
```

### 결합 회선 추가
```
1. 106 또는 T월드 앱에서 신청
2. 추가할 휴대폰 번호 입력
3. 가족 관계 확인 (필요 시)
4. 익월부터 추가 할인 적용
```

### 결합 해지
```
1. 인터넷 해지 시: 결합 자동 해지
2. 휴대폰 해지 시: 해당 회선만 결합 해제
3. 자진 탈퇴: 언제든 가능
```

## 주의사항

### 할인 적용
- 결합 신청 익월 1일부터 할인 적용
- 청구서에 결합 할인 내역 표시

### 결합 제한
| 제한 사항 | 내용 |
|-----------|------|
| MVNO | 알뜰폰은 결합 불가 |
| 법인 | 법인 회선은 일반 결합 불가 |
| 타사 인터넷 | KT/LG 인터넷은 결합 불가 |

### 해지 시 영향
| 상황 | 결과 |
|------|------|
| 인터넷 해지 | 전체 결합 해지, 휴대폰 할인 소멸 |
| 휴대폰 1회선 해지 | 해당 회선만 결합 해제 |
| 휴대폰 전체 해지 | 인터넷 결합 할인 유지 (인터넷만) |

## 결합 할인 vs 단독 가입 비교

### 3인 가족 예시 (월 기준)

**단독 가입 시**
```
인터넷 기가라이트: 33,000원
휴대폰 A: 89,000원
휴대폰 B: 69,000원
휴대폰 C: 55,000원
---------------
총: 246,000원
```

**결합 시**
```
인터넷: 33,000원 - 5,500원 = 27,500원
휴대폰 A: 89,000원 - 5,500원 = 83,500원
휴대폰 B: 69,000원 - 5,500원 = 63,500원
휴대폰 C: 55,000원 - 5,500원 = 49,500원
---------------
총: 224,000원 (22,000원 절약)
```

## 자주 묻는 질문 (FAQ)

**Q: 인터넷은 아버지 명의, 휴대폰은 제 명의인데 결합 되나요?**
A: 가족관계증명서로 가족임을 증명하면 결합 가능합니다.

**Q: 알뜰폰(MVNO)도 결합 되나요?**
A: SK텔레콤 직접 가입 회선만 결합 가능합니다. MVNO는 결합 대상이 아닙니다.

**Q: 결합 할인과 선택약정 할인 중복 되나요?**
A: 네, 휴대폰 선택약정 25% 할인과 결합 할인은 중복 적용됩니다.

**Q: 이사하면 결합이 해제되나요?**
A: 이전 설치하면 결합이 유지됩니다. 해지 후 재가입 시에는 재결합이 필요합니다.

**Q: 결합 중인데 인터넷 상품을 변경하면요?**
A: 결합은 유지되며 변경된 상품에 결합 할인이 적용됩니다.
