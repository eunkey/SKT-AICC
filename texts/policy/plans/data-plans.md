## 데이터 쉐어링 서비스

### 서비스 개요
메인 회선의 데이터를 서브 회선(태블릿, 워치 등)과 공유하는 서비스

### 요금
| 항목 | 금액 |
|------|------|
| 쉐어링 기본료 | 3,300원/회선 |
| 메인 회선 | 5G/LTE 무제한 요금제 권장 |

### 쉐어링 가능 조합
```
메인 회선 (휴대폰)
    ├── 서브 1: 태블릿
    ├── 서브 2: 스마트워치
    └── 서브 3: 에그/라우터
```

### 쉐어링 제한
- 최대 서브 회선: 4회선
- 메인 회선 요금제에 따라 쉐어링 가능 용량 상이
- 테더링 용량과 별도

## 자주 묻는 질문 (FAQ)

**Q: 태블릿 요금제로 통화할 수 있나요?**
A: 일반 음성통화는 불가능합니다. 카카오톡, FaceTime 등 인터넷 기반 통화는 가능합니다.

**Q: 워치 쉐어는 메인 회선 데이터를 얼마나 쓰나요?**
A: 워치 쉐어는 메인 회선 데이터를 공유하므로, 메인 회선 사용량에 포함됩니다. 워치 자체 사용량은 적은 편입니다.

**Q: 태블릿에 5G 요금제를 사용할 수 있나요?**
A: 5G 지원 태블릿이면 5G 데이터 요금제 가입이 가능합니다. 일반 태블릿 요금제와 가격대가 다릅니다.

**Q: 에그와 테더링의 차이점은?**
A: 테더링은 휴대폰 데이터를 공유하는 것이고, 에그는 별도 회선의 전용 장치입니다. 에그는 배터리가 오래가고 여러 기기 연결에 적합합니다.

**Q: 데이터 쉐어링과 데이터 선물의 차이는?**
A: 쉐어링은 동일 명의 회선 간 실시간 데이터 공유이고, 선물은 1회성 데이터 전송입니다.
