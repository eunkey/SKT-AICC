# 요금제 변경 정책

> 최종 업데이트: 2024년 1월

## 요금제 변경 기본 원칙

### 변경 가능 시점
| 조건 | 변경 가능 여부 |
|------|---------------|
| 신규 가입 후 | 즉시 가능 |
| 요금제 변경 후 | 다음 달부터 가능 |
| 약정 기간 중 | 가능 (조건에 따라 위약금 발생) |
| 일시정지 중 | 정지 해제 후 가능 |

### 변경 적용일
| 선택 | 적용 시점 |
|------|----------|
| 당일 적용 | 신청 즉시 (일할 계산) |
| 익월 적용 | 다음 달 1일부터 |

---

## 상향/하향 변경

### 상향 변경 (더 비싼 요금제로)
- **제한 없음**: 언제든 가능
- **데이터**: 즉시 상향된 데이터 적용
- **요금**: 변경일부터 일할 계산

### 하향 변경 (더 저렴한 요금제로)
- **월 1회 제한**: 같은 달 재변경 불가
- **데이터**: 변경 시점부터 새 요금제 데이터 적용
- **잔여 데이터**: 이월되지 않음

---

## 약정 관련 규정

### 선택약정 할인 중 변경

#### 동일/상위 요금제로 변경
- 위약금 없음
- 약정 기간 유지
- 할인율 동일하게 적용

#### 하위 요금제로 변경
| 상황 | 위약금 |
|------|--------|
| 동일 계열 내 하향 | 없음 (예: 5G 프라임 → 5G 슬림) |
| 계열 변경 하향 | 발생 가능 (예: 5G → LTE 최저가) |

### 공시지원금 약정 중 변경

#### 유지 요금제 조건
- 최소 유지 요금제 이상 유지 필수
- 최소 기준 미달 시 차액 반환

#### 하위 요금제로 변경 시
```
반환금 = 공시지원금 × 잔여 개월 ÷ 24 × 차액비율
```

## 위약금 계산

### 선택약정 위약금
| 잔여 기간 | 위약금 |
|----------|--------|
| 24~19개월 | 할인받은 금액 × 50% |
| 18~13개월 | 할인받은 금액 × 40% |
| 12~7개월 | 할인받은 금액 × 30% |
| 6~1개월 | 할인받은 금액 × 20% |

### 공시지원금 위약금
| 잔여 기간 | 반환 비율 |
|----------|----------|
| 24~19개월 | 지원금 × 잔여일수 ÷ 730 |
| 18~13개월 | 지원금 × 잔여일수 ÷ 730 |
| 12~1개월 | 지원금 × 잔여일수 ÷ 730 |

※ 잔여일수 비례로 계산

## 요금제 변경 절차

### T월드 앱
1. T월드 앱 로그인
2. 요금제 > 요금제 변경
3. 원하는 요금제 선택
4. 적용일 선택 (당일/익월)
5. 약관 동의 및 변경 완료

### 114 고객센터
1. 114 전화
2. ARS 2번 (요금제/부가서비스)
3. 상담원 연결
4. 본인 확인 후 변경 요청

### 매장 방문
- 신분증 지참
- 요금제 변경 신청
- 현장 즉시 처리

## 특수 변경 케이스

### 5G ↔ LTE 전환
| 변경 방향 | 조건 |
|----------|------|
| 5G → LTE | 5G 단말기도 LTE 요금제 사용 가능 (단, LTE 속도만) |
| LTE → 5G | 5G 단말기 + 5G USIM 필수 |

### 일반 ↔ 청년/시니어 요금제
| 변경 | 조건 |
|------|------|
| 일반 → 청년 | 나이 확인 (만 19~34세) |
| 일반 → 시니어 | 나이 확인 (만 65세 이상) |
| 청년/시니어 → 일반 | 제한 없음 |

### 음성 ↔ 데이터 전용
- 서로 다른 회선 유형으로 변경 불가
- 해지 후 재가입 필요

## 자주 묻는 질문

### Q: 요금제 변경하면 기존 데이터는 어떻게 되나요?
A: 변경 시점에 잔여 데이터는 소멸됩니다. 새 요금제 데이터가 새로 부여됩니다.

### Q: 당일 변경과 익월 변경 중 뭐가 유리한가요?
A:
- **당일 변경**: 데이터가 급하게 필요할 때 (일할 계산으로 요금 정산)
- **익월 변경**: 이번 달 데이터 다 쓰고 변경할 때 (요금 중복 없음)

### Q: 한 달에 여러 번 요금제 변경 가능한가요?
A: 상향 변경은 제한 없습니다. 하향 변경은 월 1회로 제한됩니다.

### Q: 약정 중인데 위약금 없이 변경 가능한 요금제가 있나요?
A: 동일 계열 내에서 상향 변경은 위약금이 없습니다. 예를 들어 5G 슬림 → 5G 프라임은 위약금 없이 가능합니다.

### Q: 요금제 변경하면 멤버십 등급도 바뀌나요?
A: 네, 요금제에 따라 멤버십 등급이 결정됩니다. 등급 변경은 익월 1일부터 적용됩니다.

### Q: 해외에서 요금제 변경 가능한가요?
A: 네, T월드 앱이나 114(+82-2-6343-9000)로 변경 가능합니다. 단, 로밍 중에는 변경이 즉시 반영되지 않을 수 있습니다.
