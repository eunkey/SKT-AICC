# 요금제 변경 정책

> 최종 업데이트: 2025년 2월

## 요금제 변경 기본 원칙

### 변경 가능 시점
| 변경 유형 | 적용 시점 |
|-----------|-----------|
| 상위 요금제 변경 | 즉시 적용 가능 |
| 하위 요금제 변경 | 다음 청구일부터 적용 |
| 동일 가격대 변경 | 즉시 또는 다음 청구일 선택 |

### 변경 제한
| 제한 사항 | 내용 |
|-----------|------|
| 변경 횟수 | 월 1회 (일부 요금제) |
| 약정 중 변경 | 약정 조건에 따라 제한 |
| 공시지원금 수령 | 일정 기간 동일 요금제 유지 필요 |

## 변경 신청 방법

### T월드 앱 (권장)
```
1. T월드 앱 > 요금제
2. 현재 요금제 확인
3. "변경하기" 선택
4. 희망 요금제 선택
5. 적용일 선택 (즉시/다음 청구일)
6. 약관 동의 후 신청 완료
```

### 고객센터 (114)
```
1. 114 전화
2. "요금제 변경" 선택
3. 상담사 연결 또는 ARS 처리
4. 본인 확인 후 변경 신청
```

### 매장 방문
```
1. T월드 직영점/대리점 방문
2. 신분증 지참
3. 요금제 상담 후 변경 신청
```

## 변경 시 요금 계산

### 월 중 변경 시 일할 계산

**상위 요금제 변경 (즉시 적용)**
```
예시: 15일에 55,000원 → 89,000원 변경

전반기 (1~14일): 55,000원 × (14/30) = 25,667원
후반기 (15~30일): 89,000원 × (16/30) = 47,467원
총 요금: 73,134원
```

**하위 요금제 변경 (다음 청구일 적용)**
```
이번 달: 기존 요금제 전액
다음 달부터: 새 요금제 적용
```

### 데이터 계산
- 상위 변경 시: 새 요금제 데이터로 즉시 변경
- 하위 변경 시: 이번 달은 기존 데이터 유지

## 특수 상황 변경

### 휴대폰 번호이동 후 요금제 변경
- 번호이동 완료 후 즉시 변경 가능
- 이동 전 통신사 약정은 자동 해지 (위약금 별도)

### 명의변경 후 요금제 변경
- 명의변경 완료 후 새 명의자가 변경 가능
- 기존 약정은 승계 또는 해지 선택

### 일시정지 중 요금제 변경
- 일시정지 중에도 요금제 변경 가능
- 정지 해제 후 새 요금제 적용

## 위약금 안내

### 선택약정 해지 시 반환금

**계산 공식**:
```
반환금 = (월 할인액 × 잔여 개월) × 일정 비율
```

**예시**: 89,000원 요금제, 12개월 남음
```
월 할인액: 89,000 × 25% = 22,250원
잔여 할인: 22,250 × 12 = 267,000원
반환금: 약 267,000원 (잔여 기간에 따라 조정)
```

### 공시지원금 해지 시 반환금

**계산 공식**:
```
반환금 = 지원금 × (잔여일수 / 총약정일수) × 일정 비율
```

**예시**: 지원금 300,000원, 12개월 남음 (24개월 약정)
```
반환금 = 300,000 × (12/24) = 150,000원 (대략적 계산)
실제 금액은 해지 시점에 따라 달라짐
```

## 자주 묻는 질문 (FAQ)

**Q: 요금제를 바꾸면 데이터는 어떻게 되나요?**
A: 상위 변경 시 새 요금제 데이터로 즉시 변경됩니다. 하위 변경은 다음 달부터 적용됩니다.

**Q: 한 달에 여러 번 바꿀 수 있나요?**
A: 일반적으로 월 1회로 제한됩니다. 긴급한 경우 고객센터 문의가 필요합니다.

**Q: 요금제 바꾸면 T멤버십 등급도 바뀌나요?**
A: 네, 요금제에 따라 기본 T멤버십 등급이 변경됩니다. 하위 요금제로 변경 시 등급이 낮아질 수 있습니다.

**Q: 약정 중에 더 비싼 요금제로 바꾸면 약정은 어떻게 되나요?**
A: 선택약정은 유지되며 새 요금제에 25% 할인이 적용됩니다.

**Q: 가족결합 중인데 요금제를 바꿔도 되나요?**
A: 가능합니다. 단, 결합 할인 조건에 영향을 줄 수 있으니 확인 후 변경하세요.
