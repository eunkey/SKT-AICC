# 가족결합 할인 정책

> 최종 업데이트: 2025년 2월

## 가족결합 할인 개요

SK텔레콤 가족결합 서비스는 가족 구성원이 함께 SK텔레콤을 이용하면 추가 할인을 받을 수 있는 서비스입니다.

| 항목 | 내용 |
|------|------|
| 결합 가능 회선 | 최대 10회선 |
| 대상 | 가족 관계 증명 가능한 구성원 |
| 할인 방식 | 회선당 정액 할인 |

| 조건 | 내용 |
|------|------|

| 결합 회선 수 | 회선당 할인액 |

**예시**: 4인 가족 결합 시
```

가족 3: 10,000원 할인

```

## 가족결합 신청 방법

### 신청 채널
| 채널 | 방법 |
|------|------|
| T월드 앱 | 결합/가족 > 가족결합 신청 |
| 114 | 상담사 연결 > 가족결합 신청 |
| 매장 방문 | 신분증 + 관계증명서 지참 |

### 필요 서류
| 구분 | 필요 서류 |
|------|-----------|
| 배우자 | 가족관계증명서 또는 혼인관계증명서 |
| 자녀 | 가족관계증명서 |
| 부모 | 가족관계증명서 |
| 형제자매 | 가족관계증명서 (부모 경유 확인) |
| 조부모/손자녀 | 가족관계증명서 (3대 확인) |

### 신청 절차
```
1. 대표 회선 지정 (가장 높은 요금제 권장)
      ↓
2. 결합 회선 추가 (가족 동의 필요)
      ↓
3. 관계 증명 서류 제출
      ↓
4. 결합 완료 (익월 청구분부터 할인 적용)
```

## 대표 회선 관리

### 대표 회선 역할
| 역할 | 내용 |
|------|------|

| 데이터 쉐어링 | 대표 회선 데이터 공유 (해당 시) |

### 대표 회선 변경
- T월드 앱 또는 114에서 변경 가능
- 기존 대표자 동의 필요
- 새 대표자 본인 인증 필요

### 회선 추가/탈퇴
| 구분 | 방법 |
|------|------|
| 회선 추가 | 대표 회선에서 신청 + 추가 회선 동의 |
| 자진 탈퇴 | 본인 회선에서 탈퇴 신청 |
| 강제 탈퇴 | 대표 회선에서 특정 회선 제외 |

## 결합 해지

### 해지 사유
| 사유 | 처리 |
|------|------|
| 번호이동 | 자동 결합 해지 |
| SK텔레콤 해지 | 자동 결합 해지 |
| 자진 탈퇴 | 해당 회선만 결합 해지 |
| 대표 회선 해지 | 새 대표 지정 또는 결합 해지 |

### 해지 시 주의사항
- 결합 해지 익월부터 할인 미적용
- 최소 회선(2회선) 미달 시 결합 자동 해지
- 인터넷 결합은 인터넷 해지 시 자동 해지

## 자주 묻는 질문 (FAQ)

**Q: 사실혼 관계도 가족결합 가능한가요?**
A: 주민등록등본에 동거인으로 등재되어 있으면 결합 가능합니다. 서류 확인이 필요합니다.

**Q: 형제자매끼리만 결합할 수 있나요?**
A: 가능합니다. 부모를 경유한 가족관계증명서로 증명하면 됩니다.

**Q: 결합 중인데 다른 결합으로 바꿀 수 있나요?**
A: 기존 결합 해지 후 새 결합 신청이 필요합니다. 해지 익월부터 새 결합이 적용됩니다.

**Q: 알뜰폰(MVNO)도 가족결합 되나요?**
A: SK텔레콤 직접 가입 회선만 가능합니다. MVNO는 결합 대상이 아닙니다.

**Q: 결합 회선 중 한 명이 요금을 안 내면 어떻게 되나요?**
A: 각 회선의 요금은 개별 청구됩니다. 미납이 있어도 다른 회선에 영향은 없습니다.

**Q: 법인 회선도 가족결합 가능한가요?**
A: 법인 회선은 개인 가족결합 대상이 아닙니다. 법인 전용 결합 상품을 이용해 주세요.
