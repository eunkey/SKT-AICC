# 가족결합 할인 정책

> 최종 업데이트: 2024년 1월

## T가족모아 할인

### 할인 개요
SK텔레콤 회선 2개 이상을 결합하여 월정액 할인을 받는 서비스

### 결합 조건
| 항목 | 내용 |
|------|------|
| 최소 회선 | 2회선 이상 |
| 최대 회선 | 5회선 |
| 대상 요금제 | 5G/LTE 음성 요금제 |
| 가족 범위 | 본인, 배우자, 직계존비속, 형제자매 |

### 회선 수별 할인

| 회선 수 | 회선당 할인 | 총 할인 |
|--------|-----------|--------|
| 2회선 | 10,000원 | 20,000원 |
| 3회선 | 13,000원 | 39,000원 |
| 4회선 | 15,000원 | 60,000원 |
| 5회선 | 17,000원 | 85,000원 |

### 할인 적용 요금제

| 요금제 등급 | 적용 가능 |
|------------|----------|
| 5G 프라임 이상 | O |
| 5G 스탠다드 | O |
| 5G 슬림 | O |
| 5G 라이트 | O |
| LTE 프리미어 이상 | O |
| LTE 에센셜 | O |
| LTE 베이직 | O |
| LTE 라이트 | X (33,000원 미만) |
| 시니어/청소년 요금제 | 일부 가능 |

※ 월정액 33,000원 이상 요금제만 결합 가능

## T가족모아 데이터

### 데이터 공유 서비스
가족 간 데이터를 공유하여 함께 사용하는 서비스

### 공유 가능 조건
| 항목 | 내용 |
|------|------|
| 대표 회선 | 데이터 100GB 이상 요금제 |
| 공유 회선 | 제한 없음 |
| 공유 방식 | 대표 회선 데이터를 분배 |

### 이용 요금
- 기본료: 무료 (T가족모아 할인 결합 시)
- 별도 가입: 회선당 2,200원/월

## 온가족 프리 할인

### 할인 개요
SK텔레콤 + SK브로드밴드 결합 시 추가 할인

### 결합 상품
| 결합 유형 | 월 할인 |
|----------|--------|
| 이동전화 1회선 + 인터넷 | 5,500원 |
| 이동전화 2회선 + 인터넷 | 11,000원 |
| 이동전화 3회선 + 인터넷 | 16,500원 |
| 이동전화 4회선 이상 + 인터넷 | 22,000원 |
| + IPTV(B tv) 추가 시 | +5,500원 |

### 결합 조건
- SK텔레콤 이동전화 1회선 이상
- SK브로드밴드 인터넷 가입
- 동일 명의 또는 가족 명의

→ 상세 내용은 [결합상품 할인](../broadband/bundle-discount.md) 참조

## 가족 등록 방법

### 온라인 (T월드 앱)
1. T월드 앱 > 부가서비스
2. T가족모아 > 가족 등록
3. 가족 회선 번호 입력
4. 가족 관계 선택 (부모, 자녀, 배우자 등)
5. 상대방 동의 (앱 알림 또는 ARS)
6. 등록 완료

### 매장 방문
- 가족 전원 방문 (또는 위임장)
- 신분증 + 가족관계증명서
- 즉시 등록 및 할인 적용

### 114 고객센터
- 대표자 전화 신청
- 가족 회선 번호 및 관계 확인
- 상대방 ARS 동의 후 등록

## 자주 묻는 질문

### Q: 결혼하지 않은 커플도 가족결합 가능한가요?
A: 아니요, 법적 가족 관계만 가능합니다. 결혼, 부모-자녀, 형제자매 관계가 확인되어야 합니다.

### Q: 가족 중 한 명이 해지하면 어떻게 되나요?
A: 남은 회선 수에 맞게 할인이 재산정됩니다. 예: 4회선 → 3회선 시 회선당 15,000원 → 13,000원으로 변경

### Q: 가족이 다른 통신사로 이동하면요?
A: 해당 회선은 자동 탈퇴되고, 남은 회선 기준으로 할인이 적용됩니다.

### Q: 가족 모두 33,000원 미만 요금제면 결합 불가인가요?
A: 네, 모든 회선이 33,000원 미만이면 T가족모아 결합이 불가합니다. 1회선이라도 33,000원 이상이어야 합니다.

### Q: T가족모아와 온가족 프리를 동시에 받을 수 있나요?
A: 네, 두 할인은 중복 적용됩니다. 이동전화 가족결합 + 인터넷 결합 시 최대 할인을 받을 수 있습니다.
