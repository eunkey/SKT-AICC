# 5G 요금제 정책

> 최종 업데이트: 2025년 2월

## 5G 요금제 라인업

SK텔레콤 5G 요금제는 데이터 사용량과 부가 혜택에 따라 프라임, 스탠다드, 슬림, 라이트로 구분됩니다.

---

## 5GX 프라임 (최상위)

### 요금제 상세
| 항목 | 내용 |
|------|------|
| 월정액 | 110,000원 |
| 데이터 | 완전 무제한 (속도 제한 없음) |
| 음성 | 무제한 |
| 문자 | 무제한 |
| 테더링/핫스팟 | 무제한 |

### 부가 혜택
| 혜택 | 내용 |
|------|------|
| T멤버십 | VIP 등급 부여 |
| 구독 서비스 | 웨이브, FLO, 아마존 프라임 1개 선택 무료 |
| 로밍 | 로밍 데이터 월 1GB 무료 |
| 가족결합 | 최대 6회선 결합, 회선당 추가 할인 |

### 추천 대상
- 영상 스트리밍 많이 사용
- 테더링으로 노트북/태블릿 연결
- 데이터 제한 없이 사용 원하는 분

## 주의사항

### 데이터 속도 제한
- "무제한" 표기 요금제도 기본 용량 초과 시 속도 제한 적용
- 프라임 제외 모든 요금제에 해당
- 다음 월 1일 자동 복구

### 해외 사용
- 5G 요금제 데이터는 국내 전용
- 해외에서는 별도 로밍 상품 필요

### 요금제 변경
- 상위 요금제: 즉시 변경 가능
- 하위 요금제: 다음 청구일부터 적용
- 선택약정 중: 약정 유지 또는 해지 선택

## 자주 묻는 질문 (FAQ)

**Q: 5G 요금제인데 LTE만 나옵니다.**
A: 5G 음영지역에서는 LTE로 자동 전환됩니다. 5G 커버리지 지역에서 5G 아이콘이 표시됩니다.

**Q: 무제한 요금제인데 속도가 느려졌어요.**
A: 기본 제공량 초과 시 속도 제한이 적용됩니다. T월드 앱에서 사용량을 확인해 주세요.

**Q: 테더링/핫스팟 용량이 별도인가요?**
A: 프라임을 제외한 요금제는 테더링 별도 용량입니다. 스탠다드 50GB, 슬림 20GB 등 제한이 있습니다.

**Q: 5G 요금제에서 LTE 요금제로 변경 가능한가요?**
A: 가능합니다. 단, 단말기 약정이 있는 경우 약정 조건을 확인해 주세요.

**Q: 가족결합 할인은 얼마나 되나요?**
A: 결합 회선 수에 따라 회선당 최대 25% 추가 할인이 적용됩니다.
