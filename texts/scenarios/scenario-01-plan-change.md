# 시나리오 01: 요금제 변경 및 가족결합 문의

> 상담사가 고객의 요금제 변경과 가족결합 관련 문의를 처리하는 시나리오

---

## 고객 정보

| 항목 | 내용 |
|------|------|
| 고객명 | 김민수 |
| 연령 | 35세 |
| 현재 요금제 | 5GX 프라임 (110,000원) |
| 약정 상태 | 선택약정 24개월 중 12개월 사용 |
| T멤버십 | VIP |
| 가족 회선 | 없음 (단독) |

---

## 상담 흐름

### 1단계: 고객 문의 접수

**고객:**
> "안녕하세요. 요금제를 좀 낮추고 싶은데요. 지금 5GX 프라임 쓰는데 너무 비싼 것 같아요. 그리고 와이프랑 아이들 회선도 SK텔레콤인데 가족결합하면 할인 받을 수 있나요?"

**상담사 내부 메모:**
- 요청 1: 요금제 하향 변경 (5GX 프라임 → 하위 요금제)
- 요청 2: 가족결합 할인 문의
- 확인 필요: 약정 상태, 위약금, 가족 회선 정보

---

### 2단계: 검색 및 정보 수집

#### 검색 1: 요금제 변경 정책
**검색 키워드:** "요금제 변경", "하위 요금제"
**참조 문서:** `policy/plans/plan-change-policy.md`

```
[검색 결과 요약]
- 하위 요금제 변경: 다음 청구일부터 적용
- 선택약정 중 하위 변경: 약정 유지 가능 (최소 요금 조건 확인 필요)
- 변경 시 T멤버십 등급 하락 가능
```

#### 검색 2: 5G 요금제 라인업
**검색 키워드:** "5G 요금제", "스탠다드", "슬림"
**참조 문서:** `policy/plans/5g-plans.md`

```
[검색 결과 요약]
- 5GX 프라임: 110,000원 (완전 무제한, VIP)
- 5GX 스탠다드: 89,000원 (150GB 후 5Mbps, 플래티넘)
- 5GX 슬림: 69,000원 (100GB 후 5Mbps, 골드)
- 5GX 라이트: 55,000원 (8GB, 골드)
```

#### 검색 3: 가족결합 할인
**검색 키워드:** "가족결합", "T가족결합"
**참조 문서:** `policy/plans/family-plans.md`

```
[검색 결과 요약]
- T가족결합: 최대 10회선, 회선당 5,000원~12,500원 할인
- 4회선 결합 시: 회선당 10,000원 할인
- 필요 서류: 가족관계증명서
- 신청: T월드 앱, 114, 매장 방문
```

---

### 3단계: 고객 응대

**상담사:**
> "김민수 고객님, 문의 주신 내용 확인해 드리겠습니다.
>
> 먼저 **요금제 변경**에 대해 안내드릴게요.
>
> 현재 5GX 프라임(110,000원)을 사용 중이시고, 선택약정 12개월이 남아있으시네요. 하위 요금제로 변경하셔도 **선택약정 25% 할인은 유지**됩니다.
>
> 추천드리는 요금제는:
> - **5GX 스탠다드 (89,000원)**: 데이터 150GB 기본 제공, 초과 시 5Mbps
> - **5GX 슬림 (69,000원)**: 데이터 100GB 기본 제공, 초과 시 5Mbps
>
> 참고로 요금제를 낮추시면 T멤버십 등급이 **VIP에서 플래티넘(스탠다드) 또는 골드(슬림)로 변경**됩니다.
>
> 고객님의 월 데이터 사용량이 어느 정도 되시나요?"

**고객:**
> "보통 50~60GB 정도 쓰는 것 같아요."

**상담사:**
> "그러시면 **5GX 슬림(69,000원)**이 적합하실 것 같습니다. 100GB까지 고속으로 사용 가능하고, 선택약정 25% 할인 적용하면 **월 51,750원**에 이용 가능하세요.
>
> 다음으로 **가족결합** 안내드릴게요.
>
> 배우자분과 자녀분 회선까지 총 **4회선**을 결합하시면, 회선당 **10,000원씩 할인**받으실 수 있어요.
> 4회선 기준 **월 40,000원 추가 절약**이 가능합니다.
>
> 가족결합 신청하시려면 **가족관계증명서**가 필요하고, T월드 앱이나 매장에서 신청 가능하세요."

---

### 4단계: 요금 비교 및 확인

**상담사:**
> "현재와 변경 후를 비교해 드릴게요.
>
> | 구분 | 현재 | 변경 후 |
> |------|------|---------|
> | 요금제 | 5GX 프라임 | 5GX 슬림 |
> | 월정액 | 110,000원 | 69,000원 |
> | 선택약정 할인 | -27,500원 | -17,250원 |
> | 가족결합 할인 | 없음 | -10,000원 |
> | **실납부액** | **82,500원** | **41,750원** |
>
> 고객님 회선만 기준으로 **월 40,750원 절약**되시고, 가족 전체로는 더 많이 절약되세요."

**고객:**
> "와, 많이 줄어드네요! 그런데 T멤버십 VIP 혜택이 없어지면 아쉬운데..."

**상담사:**
> "네, 골드 등급으로 변경되면 CGV 무료 영화가 2편에서 50% 할인 1편으로 줄어들고, 스타벅스 할인도 50%에서 30%로 줄어듭니다.
>
> 다만 가족결합으로 절약하시는 금액이 월 4만원 이상이니, 혜택 차이보다 절약 효과가 훨씬 크실 거예요.
>
> 요금제 변경과 가족결합 진행해 드릴까요?"

**고객:**
> "네, 진행해 주세요."

---

### 5단계: 처리 및 마무리

**상담사:**
> "감사합니다.
>
> 1. **요금제 변경**: 5GX 슬림으로 다음 청구일(매월 1일)부터 적용되도록 신청해 드렸습니다.
>
> 2. **가족결합**: T월드 앱에서 신청하시거나, 가족관계증명서 지참하시고 가까운 T월드 매장 방문하시면 바로 처리 가능합니다. T월드 앱에서 하시면 서류 없이 간편하게 신청 가능하세요.
>
> 결합 신청 방법 문자로 안내해 드릴까요?"

**고객:**
> "네, 문자로 보내주세요. 감사합니다!"

**상담사:**
> "네, 문자 발송해 드렸습니다. 추가 문의 사항 있으시면 언제든 114로 연락 주세요. 좋은 하루 되세요!"

## 상담 포인트

### 잘한 점
1. 고객의 사용 패턴(50~60GB)을 확인하고 적합한 요금제 추천
2. 현재 vs 변경 후 요금을 표로 명확하게 비교 제시
3. T멤버십 등급 변경에 따른 혜택 차이도 안내
4. 가족결합 신청 방법을 구체적으로 안내

### 주의 사항
1. 선택약정 중 하위 변경 시 최소 요금 조건 확인 필요
2. 가족결합은 SK텔레콤 직접 가입 회선만 가능 (MVNO 제외)
3. 요금제 변경은 다음 청구일부터 적용되므로 고객에게 명확히 안내
