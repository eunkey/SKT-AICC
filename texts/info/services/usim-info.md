# USIM 및 eSIM 안내

> 최종 업데이트: 2024년 1월

## USIM 개요

### USIM이란?
- **U**niversal **S**ubscriber **I**dentity **M**odule
- 가입자 정보를 저장하는 칩
- 단말기에 장착하여 통신 서비스 이용

### USIM 종류
| 규격 | 크기 | 주요 사용 기기 |
|------|------|---------------|
| Standard SIM | 25×15mm | 구형 피처폰 |
| Micro SIM | 15×12mm | 구형 스마트폰 |
| Nano SIM | 12.3×8.8mm | 최신 스마트폰 |
| eSIM | 내장형 | iPhone XS 이후, 최신 Galaxy 등 |

## USIM 발급/교체

### 발급 사유
| 사유 | 비용 |
|------|------|
| 신규 가입 | 무료 |
| 분실/파손 재발급 | 7,700원 |
| 단말기 규격 변경 | 무료~7,700원 |
| 5G USIM 업그레이드 | 무료 |
| 기기변경 시 | 무료 (동일 규격) |

### 발급 방법

#### 온라인 (배송)
1. T월드 앱 > 고객지원 > USIM 재발급
2. 발급 사유 선택
3. 배송지 입력
4. 결제 (유료 시)
5. 택배 수령 (1~2일)
6. 셀프 개통 또는 114 개통

#### 매장 방문
1. 신분증 지참
2. T월드 직영점 또는 대리점 방문
3. USIM 발급 신청
4. 현장 즉시 개통

## eSIM 안내

### eSIM이란?
- **e**mbedded **SIM**: 단말기에 내장된 SIM
- 물리적 USIM 없이 통신사 프로필 다운로드
- 하나의 기기에서 듀얼 심 가능

### eSIM 지원 기기
| 제조사 | 지원 모델 |
|--------|----------|
| Apple | iPhone XS/XR 이후 전 모델 |
| Samsung | Galaxy S20 이후, Z Flip/Fold 시리즈 |
| Google | Pixel 3 이후 |

### eSIM 장점
- 물리적 교체 불필요
- 분실 위험 없음
- 듀얼 심 활용 (물리 USIM + eSIM)
- 해외 로밍 시 편리

### eSIM 발급 방법

#### T월드 앱 (셀프 발급)
1. T월드 앱 > 고객지원 > eSIM 신청
2. 지원 기기 확인
3. 본인 인증
4. eSIM 프로필 다운로드
5. 기기에서 활성화

#### 순서 (iPhone)
1. 설정 > 셀룰러 > 셀룰러 요금제 추가
2. QR 코드 스캔 또는 수동 입력
3. SK텔레콤 프로필 다운로드
4. 기본 회선 설정

#### 순서 (Galaxy)
1. 설정 > 연결 > SIM 관리자
2. eSIM 추가
3. 통신사에서 QR 코드 스캔
4. SK텔레콤 선택 및 활성화

### eSIM 비용
| 항목 | 비용 |
|------|------|
| 최초 발급 | 무료 |
| 재발급 | 2,750원 |
| 기기 변경 | 2,750원 |

## 5G USIM 업그레이드

### 왜 5G USIM이 필요한가?
- 구형 USIM은 5G 네트워크 미지원
- 5G 요금제 사용 시 교체 권장
- 보안 강화 (최신 암호화)

### 무료 교체 대상
- 5G 요금제 가입 고객
- 5G 단말기 사용 고객
- 2019년 이전 발급 USIM 보유

### 교체 확인 방법
1. USIM 뒷면 확인
2. "5G" 또는 "LTE" 표기 확인
3. T월드 앱 > 내 정보 > USIM 정보 확인

## USIM 분실/파손 시

### 분실 시 조치
1. **즉시 114로 분실 신고**
   - 이용 정지 처리
   - 부정 사용 방지

2. **USIM 재발급**
   - 온라인 또는 매장 방문
   - 신분증 필수

3. **정지 해제**
   - USIM 재발급 후 자동 해제
   - 또는 114에서 수동 해제

### 파손 시 조치
- USIM 금속 부분 손상: 재발급 필요
- 플라스틱 부분만 손상: 규격 맞으면 사용 가능

## 해외 USIM 사용

### 해외에서 SK USIM 사용
- 로밍 자동 활성화 (설정에 따라)
- baro 로밍 자동 적용
- 현지 통신사 자동 연결

### 현지 USIM 구매 시
- 기존 SK USIM 보관 필수
- 귀국 후 SK USIM 재장착
- eSIM 기기: 듀얼 심 활용 권장

## 듀얼 SIM 활용

### 지원 기기
- iPhone XS 이후: 나노 SIM + eSIM
- Galaxy S21 이후: 나노 SIM + eSIM
- iPhone 14 이후 (미국판): eSIM + eSIM

### 활용 사례
| 조합 | 용도 |
|------|------|
| 개인 + 업무 | 번호 분리 |
| 국내 + 해외 | 로밍 비용 절감 |
| 데이터 + 통화 | 요금제 조합 |

### 설정 방법 (iPhone)
1. 설정 > 셀룰러
2. 기본 음성 회선 선택
3. 셀룰러 데이터 회선 선택
4. 연락처별 회선 지정 가능

## 자주 묻는 질문

### Q: USIM을 다른 폰에 옮기면 바로 사용 가능한가요?
A: 네, 동일 규격 USIM이면 다른 폰에 장착하여 바로 사용 가능합니다. 단, 5G USIM은 5G 폰에서만 5G 사용 가능합니다.

### Q: eSIM과 일반 USIM을 동시에 사용할 수 있나요?
A: 네, 듀얼 SIM 지원 기기에서 가능합니다. 두 번호로 동시에 전화/문자를 받을 수 있습니다.

### Q: USIM 비밀번호(PIN)는 어떻게 변경하나요?
A: 휴대폰 설정 > 보안 > SIM 카드 잠금 메뉴에서 변경 가능합니다. 초기 PIN은 0000입니다.

### Q: eSIM을 발급받으면 기존 USIM은 어떻게 되나요?
A: eSIM 전환 시 기존 물리 USIM은 자동 해지됩니다. 듀얼 심 사용 시에는 별도 eSIM 회선 신규 개통이 필요합니다.

### Q: 해외에서 eSIM 발급 가능한가요?
A: 해외에서는 eSIM 신규 발급이 어렵습니다. 출국 전 미리 발급받으시거나, 현지 USIM을 사용하세요.
