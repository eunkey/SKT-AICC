# USIM/eSIM 정보 및 발급 안내

> 최종 업데이트: 2025년 2월

## USIM이란?

USIM(Universal Subscriber Identity Module)은 가입자 정보를 저장하는 작은 칩으로, 휴대폰에 삽입하여 통신 서비스를 이용할 수 있게 합니다.

### USIM 종류
| 종류 | 크기 | 적용 단말기 |
|------|------|-------------|
| Mini SIM | 25mm x 15mm | 구형 폰 (거의 없음) |
| Micro SIM | 15mm x 12mm | 일부 구형 스마트폰 |
| Nano SIM | 12.3mm x 8.8mm | 대부분의 최신 스마트폰 |
| eSIM | 내장형 | iPhone XS 이후, Galaxy S20 이후 |

---

## eSIM이란?

eSIM(embedded SIM)은 물리적인 카드 없이 단말기에 내장된 칩에 가입자 정보를 다운로드하여 사용하는 방식입니다.

### eSIM 장점
| 장점 | 설명 |
|------|------|
| 즉시 개통 | 매장 방문 없이 원격 개통 |
| 듀얼 SIM | 물리 SIM + eSIM으로 2개 번호 사용 |
| 분실 위험 감소 | 물리적 카드 없음 |
| 공간 활용 | 트레이 공간 활용 가능 |

### eSIM 지원 단말기

#### Apple
- iPhone XS/XS Max/XR 이후 전 모델
- iPad Pro (3세대 이후)
- Apple Watch (셀룰러 모델)

#### Samsung
- Galaxy S20 이후 S시리즈
- Galaxy Z Fold/Flip 시리즈
- Galaxy Watch (LTE 모델)

## USIM 발급/재발급

### 발급이 필요한 경우
| 상황 | 비고 |
|------|------|
| 신규 가입 | 새 번호 개통 시 |
| 기기 변경 | SIM 규격이 다른 경우 |
| 분실/파손 | 기존 USIM 사용 불가 시 |
| 불량 | 인식 오류, 접촉 불량 |

### 발급 방법

#### 매장 방문
```
1. 신분증 지참하여 T월드 매장 방문
2. USIM 발급 요청
3. 본인 확인 후 발급 (약 5~10분)
4. 비용: 7,700원 (VAT 포함)
```

#### T다이렉트샵 (온라인)
```
1. shop.tworld.co.kr 접속
2. USIM 구매 (배송 2~3일)
3. 수령 후 T월드 앱에서 셀프 개통
4. 비용: 7,700원 + 배송비
```

### USIM 비용
| 유형 | 비용 | 비고 |
|------|------|------|
| 일반 USIM | 7,700원 | 신규/재발급 동일 |
| 무료 교체 | 0원 | 불량 교체 (개통 14일 이내) |
| eSIM 발급 | 2,750원 | 온라인 발급 시 |

## eSIM 발급

### 발급 조건
- eSIM 지원 단말기 보유
- 본인 명의 SK텔레콤 회선
- 본인인증 가능

### 발급 방법

#### T월드 앱 (권장)
```
1. T월드 앱 > 더보기 > eSIM
2. "eSIM 신청" 선택
3. 본인인증 (PASS 또는 공동인증서)
4. QR코드 발급 (유효기간 24시간)
5. 단말기 설정에서 QR코드 스캔
6. 개통 완료
```

#### 매장 방문
```
1. 신분증 지참하여 T월드 매장 방문
2. eSIM 발급 요청
3. QR코드 또는 직접 설정
```

### eSIM 설치 방법

#### iPhone
```
설정 > 셀룰러 > 셀룰러 요금제 추가 > QR 코드 사용
> QR코드 스캔 > 요금제 추가 완료
```

#### Android (Samsung)
```
설정 > 연결 > SIM 관리자 > eSIM 추가 > QR 코드 스캔
> 요금제 추가 완료
```

### eSIM 비용
| 발급 방법 | 비용 |
|-----------|------|
| T월드 앱 | 2,750원 |
| 매장 발급 | 2,750원 |
| 재발급 | 2,750원 |

## 듀얼 SIM 사용

### 듀얼 SIM 조합
| 조합 | 지원 여부 |
|------|-----------|
| Nano SIM + eSIM | 대부분 지원 |
| eSIM + eSIM | 일부 최신 기기 지원 |
| Nano SIM + Nano SIM | 일부 Android 기기 |

### 듀얼 SIM 활용 예
- 개인 번호 + 업무 번호
- 국내 번호 + 해외 로밍 번호
- 음성 전용 + 데이터 전용

### 설정 방법 (iPhone)
```
설정 > 셀룰러 > 기본 회선 설정
- 음성 통화용 회선 선택
- 데이터용 회선 선택
- iMessage/FaceTime 회선 선택
```

## USIM 문제 해결

### USIM 인식 안 됨

**점검 순서**:
```
1. USIM 트레이 열어서 재장착
2. 접촉면(금색 부분) 부드러운 천으로 닦기
3. 다른 단말기에 USIM 테스트
4. 지속 시 매장 방문하여 USIM 점검
```

### USIM 등록 실패

**원인**:
- 개통 절차 미완료
- 요금 미납으로 서비스 정지
- USIM 불량

**해결**:
```
1. 114 또는 T월드 앱에서 회선 상태 확인
2. 개통 상태 확인
3. 요금 납부 상태 확인
4. 매장 방문하여 USIM 교체
```

### eSIM 설치 실패

**점검 사항**:
- Wi-Fi 연결 상태 확인
- QR코드 유효기간 확인 (24시간)
- 단말기 eSIM 지원 여부 확인
- 기존 eSIM 프로필 삭제 후 재시도

## USIM 관련 서비스

### USIM 잠금 (PIN)
USIM에 비밀번호를 설정하여 보안 강화

**설정 방법 (Android)**:
```
설정 > 생체 인식 및 보안 > 기타 보안 설정 > SIM 카드 잠금 설정
```

**기본 PIN**: 0000 (변경 권장)
**PUK 코드**: 114 또는 T월드 앱에서 확인 가능

### USIM 정보 확인
```
T월드 앱 > 더보기 > 나의 정보 > USIM 정보
- USIM 일련번호
- IMEI 번호
- 개통일
```

## 자주 묻는 질문 (FAQ)

**Q: USIM만 바꾸면 기기변경 되나요?**
A: 아닙니다. 새 단말기에 기존 USIM을 넣으면 서비스는 이용 가능하지만, 공식 기기변경 절차를 거쳐야 단말기 보험 등이 적용됩니다.

**Q: eSIM과 물리 SIM을 동시에 사용할 수 있나요?**
A: 듀얼 SIM 지원 단말기에서 가능합니다. 두 개의 번호를 하나의 폰에서 사용할 수 있습니다.

**Q: USIM 비용은 왜 내야 하나요?**
A: USIM은 가입자 인증 및 보안을 위한 정밀 장치로, 제조 및 개통 비용이 포함됩니다.

**Q: USIM을 다른 사람에게 주면 안 되나요?**
A: USIM은 본인 명의 회선 정보가 담겨 있어, 양도할 경우 명의변경 절차가 필요합니다.

**Q: PUK 코드를 잊어버렸어요.**
A: 114 또는 T월드 앱에서 본인인증 후 PUK 코드를 확인할 수 있습니다.

**Q: 해외에서 eSIM 발급받을 수 있나요?**
A: 해외에서는 eSIM 신규 발급이 어렵습니다. 출국 전 발급받으시기 바랍니다.

**Q: eSIM을 다른 기기로 옮길 수 있나요?**
A: eSIM은 기기 간 이동이 불가합니다. 새 기기에서 eSIM을 재발급받아야 합니다.
