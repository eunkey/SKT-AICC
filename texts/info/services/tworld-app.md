# T월드 앱 기능 안내

> 최종 업데이트: 2024년 1월

## 앱 다운로드

| 플랫폼 | 다운로드 |
|--------|----------|
| iOS | App Store에서 'T월드' 검색 |
| Android | Google Play에서 'T월드' 검색 |
| 최소 버전 | iOS 13.0 / Android 8.0 이상 |

---

## 주요 기능

### 1. 요금/데이터 조회

#### 실시간 조회
- **데이터 잔여량**: 홈 화면에서 즉시 확인
- **통화/문자**: 이번 달 사용량
- **청구 요금**: 예상 요금 및 지난 달 청구 내역
- **납부 현황**: 납부일, 자동이체 정보

#### 상세 내역
- 일별/시간별 데이터 사용 그래프
- 데이터 사용 앱별 분석
- 통화 상세 내역 (수신/발신)
- 문자 상세 내역

---

### 2. 요금제/부가서비스 관리

#### 요금제 변경
1. 하단 메뉴 '요금제' 선택
2. 현재 요금제 확인
3. '요금제 변경' 터치
4. 원하는 요금제 선택
5. 변경 적용일 선택 (당일/익월)
6. 변경 완료

#### 부가서비스
- 가입 중인 부가서비스 목록 확인
- 원클릭 해지 가능
- 신규 부가서비스 가입
- 월정액/무료 서비스 구분

---

### 3. 요금 납부

#### 납부 방법
| 방법 | 설명 |
|------|------|
| 신용/체크카드 | 즉시 납부 |
| 계좌이체 | 실시간 이체 |
| 간편결제 | 카카오페이, 네이버페이, 삼성페이 등 |
| 휴대폰 소액결제 | 타 통신사 번호로 결제 |

#### 자동이체 설정
1. 요금 > 자동납부 관리
2. 결제 수단 선택 (카드/계좌)
3. 카드/계좌 정보 입력
4. 출금일 선택 (1일/15일/25일)
5. 설정 완료

---

### 4. T멤버십

#### 멤버십 카드
- 바코드/QR 코드 제시 (결제 시)
- 카드 위젯으로 빠른 접근
- 현재 등급 및 다음 등급 안내

#### 포인트 조회
- 보유 포인트 확인
- 적립/사용 내역
- 소멸 예정 포인트

#### 제휴사 검색
- 카테고리별 제휴사 목록
- 내 주변 제휴사 검색
- 할인율/적립율 안내

---

### 5. 매장/서비스

#### 매장 찾기
1. 하단 '매장' 메뉴 선택
2. 현재 위치 기반 주변 매장 표시
3. 직영점/대리점 필터
4. 매장 상세 정보 (영업시간, 연락처)
5. 길찾기 연동

#### 방문 예약
- 원하는 매장 선택
- 날짜/시간 예약
- 예약 확인 SMS 발송
- 예약 변경/취소

---

### 6. 고객센터

#### 채팅 상담 (에이닷)
- AI 챗봇 24시간 운영
- 간단한 문의 즉시 해결
- 상담원 연결 가능 (운영시간 내)

#### 상담 예약
- 원하는 시간에 상담원 콜백 예약
- 대기 없이 상담 가능

#### FAQ
- 자주 묻는 질문 검색
- 카테고리별 브라우징

---

### 7. 부가 기능

#### 안심기능
| 기능 | 설명 |

- 데이터 선물하기

- 청구서 조회/출력
- 가입 증명서 발급
- 개인정보 관리
- 앱 설정 (알림, 보안 등)

## 간편 인증 설정

### 생체인증 (지문/Face ID)
1. 설정 > 인증/보안
2. 생체인증 사용 > 켜기
3. 지문 또는 Face ID 등록
4. 이후 로그인 시 생체인증 사용

### PIN 번호
1. 설정 > 인증/보안
2. PIN 번호 설정
3. 6자리 숫자 입력
4. 재입력으로 확인

### 자동 로그인
- 한 번 로그인 후 자동 유지
- 30일 미접속 시 재로그인 필요
- 기기 변경 시 재로그인 필요

## 위젯 설정

### iOS 위젯
1. 홈 화면 길게 누르기
2. 좌측 상단 '+' 터치
3. 'T월드' 검색
4. 원하는 위젯 크기 선택
5. 위젯 추가

### Android 위젯
1. 홈 화면 길게 누르기
2. '위젯' 선택
3. T월드 위젯 찾기
4. 원하는 위젯 드래그

### 위젯 종류
| 위젯 | 표시 정보 |
|------|----------|
| 데이터 | 잔여 데이터량 |
| 멤버십 | 바코드/등급 |
| 요금 | 현재 요금 |
| 퀵메뉴 | 자주 쓰는 기능 바로가기 |

## 자주 묻는 질문

### Q: 로그인이 안 됩니다
A: 다음 사항을 확인해 주세요:
1. SK텔레콤 가입 회선인지 확인
2. 본인 명의 회선인지 확인
3. 앱 업데이트 여부 확인
4. 앱 삭제 후 재설치

### Q: 다른 회선도 조회할 수 있나요?
A: 가족 회선 등록 시 가능합니다. 가족 > 가족 등록 메뉴에서 회선을 추가하세요. 단, 본인 확인 절차가 필요합니다.

### Q: 알림이 안 옵니다
A: 휴대폰 설정에서 T월드 앱 알림을 허용해 주세요. 또한 앱 내 설정 > 알림 설정도 확인해 주세요.

### Q: 앱에서 요금제 변경 시 바로 적용되나요?
A: 당일 적용 또는 익월 적용을 선택할 수 있습니다. 당일 적용 시에도 일부 요금제는 익일 적용될 수 있습니다.
