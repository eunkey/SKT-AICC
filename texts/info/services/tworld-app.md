# T월드 앱 기능 안내

> 최종 업데이트: 2025년 2월

## T월드 앱 소개

SK텔레콤 공식 고객 서비스 앱으로, 요금 조회부터 요금제 변경, 부가서비스 관리까지 대부분의 업무를 앱에서 처리할 수 있습니다.

### 다운로드
| 플랫폼 | 다운로드 |
|--------|----------|
| Android | Google Play 스토어 > "T월드" 검색 |
| iOS | App Store > "T월드" 검색 |

### 로그인 방법
| 방법 | 설명 |
|------|------|
| 자동 로그인 | SK텔레콤 회선으로 접속 시 자동 인증 |
| 본인인증 | 공동인증서, 간편인증(PASS) |
| T ID | T ID 계정으로 로그인 |

---

## 주요 기능

### 1. 나의 요금

#### 실시간 요금 조회
- 이번 달 예상 요금
- 항목별 상세 내역 (음성, 데이터, 부가서비스 등)
- 전월 대비 요금 비교

#### 요금 청구서
- 최근 6개월 청구서 조회
- PDF 다운로드
- 상세 이용 내역

#### 요금 납부
| 방법 | 설명 |
|------|------|
| 즉시 납부 | 카드/계좌이체로 바로 납부 |
| 자동이체 | 은행/카드 자동이체 설정 |
| 포인트 납부 | T멤버십 포인트로 납부 |
| 휴대폰 소액결제 | 다음 달 요금에 합산 |

### 2. 나의 사용량

#### 데이터 사용량
- 실시간 데이터 잔여량
- 일별/시간별 사용 추이
- 데이터 무제한 요금제: 기준용량 사용량

#### 음성/문자 사용량
- 무료 통화 잔여량
- 문자 잔여 건수
- 부가통화 사용량

#### 데이터 관리
| 기능 | 설명 |
|------|------|
| 데이터 리필 | 추가 데이터 구매 |
| 데이터 선물 | 타 회선에 데이터 선물 |
| 데이터 쉐어링 | 가족 간 데이터 공유 |

### 3. 요금제/부가서비스

#### 요금제 변경
```
1. T월드 앱 > 요금제
2. 현재 요금제 확인
3. "변경하기" 선택
4. 희망 요금제 선택
5. 변경 일자 선택 (즉시/다음달)
6. 약관 동의 후 신청
```

#### 부가서비스 관리
- 가입 중인 부가서비스 조회
- 신규 가입/해지
- 월정액/종량제 서비스 관리

### 4. T멤버십

#### 멤버십 조회
- 현재 등급 및 다음 등급 조건
- 적립 포인트/사용 가능 포인트
- 포인트 적립/사용 내역

#### 혜택 이용
- 제휴사 할인 쿠폰
- T데이 혜택
- 이벤트 응모

### 5. 고객센터

#### 상담 채널
| 채널 | 운영시간 |
|------|----------|
| AI 상담 | 24시간 |
| 채팅 상담 | 평일 09:00~18:00 |
| 전화 연결 | 114 바로 연결 |

#### 자주 찾는 메뉴
- 분실 신고/해제
- 요금 이의제기
- 통화품질 신고
- 명의변경 안내

### 6. 쇼핑

#### T다이렉트샵
- 최신 단말기 구매
- 요금제 가입과 동시 구매
- 온라인 전용 혜택

#### 액세서리샵
- 케이스, 충전기 등
- T멤버십 할인 적용

- 모바일 신분증 연계

## 자주 사용하는 기능 바로가기

### 데이터 선물하기
```
1. T월드 앱 > 더보기 > 데이터 선물
2. 선물할 데이터량 선택 (1GB/2GB/3GB 등)
3. 받을 번호 입력
4. 선물 완료
```

**조건**:
- 동일 명의 회선 간 또는 가족 회선
- 잔여 데이터 내에서 선물 가능

### 요금 납부하기
```
1. T월드 앱 > 요금 > 요금 납부
2. 납부 금액 확인 (전액/일부)
3. 결제 수단 선택 (카드/계좌)
4. 결제 완료
```

### 분실 신고하기
```
1. T월드 앱 > 더보기 > 분실/습득
2. "분실 신고" 선택
3. 분실 회선 선택
4. 정지 유형 선택 (일시정지/분실정지)
5. 신고 완료
```

### 데이터 리필하기
```
1. T월드 앱 > 사용량 > 데이터 리필
2. 리필 상품 선택
3. 결제 수단 선택
4. 구매 완료 (즉시 충전)
```

## 앱 문제 해결

### 로그인 안 될 때
1. 앱 강제 종료 후 재실행
2. Wi-Fi 끄고 모바일 데이터로 접속
3. 앱 캐시 삭제 후 재시도
4. 앱 삭제 후 재설치

### 알림이 안 올 때
```
Android: 설정 > 앱 > T월드 > 알림 > 알림 허용
iOS: 설정 > T월드 > 알림 > 알림 허용
```

### 본인인증 실패
1. PASS 앱 설치 확인
2. PASS 앱에서 인증서 유효 확인
3. 명의자 본인 휴대폰으로 인증

## 자주 묻는 질문 (FAQ)

**Q: T월드 앱에서 해지할 수 있나요?**
A: 해지는 앱에서 불가능합니다. 114 전화 또는 매장 방문이 필요합니다.

**Q: 가족 회선도 조회할 수 있나요?**
A: 대표 명의자는 가족결합 회선의 요금 조회가 가능합니다. 별도 설정이 필요할 수 있습니다.

**Q: 데이터 사용량이 실제와 다릅니다.**
A: 데이터 사용량은 실시간 반영에 약간의 지연이 있을 수 있습니다. 최대 2시간 내 정확한 수치로 업데이트됩니다.

**Q: 앱에서 요금제 변경하면 바로 적용되나요?**
A: 즉시 적용 또는 다음 요금 청구일 적용을 선택할 수 있습니다.

**Q: 위젯이 업데이트되지 않습니다.**
A: 위젯을 길게 눌러 삭제 후 다시 추가하거나, 앱 강제 종료 후 재실행해 보세요.
