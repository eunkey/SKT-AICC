# 요금 통지서 안내

> 최종 업데이트: 2024년 1월

## 요금 통지서란?

매월 이용 요금을 고객에게 안내하는 문서입니다. 우편, 이메일, 앱 등 다양한 방식으로 받아보실 수 있습니다.

---

## 통지서 수령 방법

### 수령 방법 종류
| 방법 | 설명 | 비용 |
|------|------|------|
| 종이 통지서 | 등록된 주소로 우편 발송 | 무료 |
| 이메일 통지서 | 등록된 이메일로 발송 | 무료 |
| T월드 앱 | 앱에서 실시간 확인 | 무료 |
| 문자(SMS) 알림 | 요금 알림 문자 발송 | 무료 |

### 기본 설정
- 신규 가입 시 **종이 통지서**가 기본 설정
- T월드 앱 가입 시 앱에서도 자동 확인 가능
- 여러 방법 중복 선택 가능

---

## 수령 방법 변경

### T월드 앱에서 변경
1. T월드 앱 로그인
2. 하단 메뉴 > **요금**
3. **청구서 설정** 선택
4. 원하는 수령 방법 선택
5. **저장** 터치

### 고객센터 전화 변경
1. 114 > 1번 (요금 관련)
2. 상담원 연결
3. 본인 확인 후 변경 요청

### 매장 방문 변경
- 신분증 지참
- 본인 또는 법정대리인 방문

---

## 통지서 주소 변경

### 주소 변경이 필요한 경우
- 이사를 한 경우
- 요금 통지서가 수령되지 않는 경우
- 거주지와 통지서 수령 주소가 다른 경우

### T월드 앱에서 변경
1. T월드 앱 로그인
2. **전체메뉴** > **내 정보 관리**
3. **주소 변경** 선택
4. 새 주소 입력 (도로명/지번 주소 모두 가능)
5. **변경** 터치
6. 변경 완료 SMS 수신

### 114 고객센터에서 변경
1. 114 전화
2. 1번 (요금 관련) > 상담원 연결
3. 본인 확인 (생년월일, 가입자명 등)
4. 새 주소 안내
5. 변경 완료

### 필요 정보
| 정보 | 설명 |
|------|------|
| 새 주소 | 도로명 또는 지번 주소 |
| 상세주소 | 동/호수, 건물명 등 |
| 우편번호 | 5자리 신우편번호 |

## 통지서 재발급

### 재발급 방법
| 방법 | 소요 시간 |
|------|----------|
| T월드 앱 다운로드 | 즉시 |
| 이메일 재발송 요청 | 즉시 |
| 종이 통지서 재발급 | 3~5일 |

### T월드 앱에서 확인
1. T월드 앱 > 요금
2. **청구서 조회** 선택
3. 원하는 월 선택
4. PDF 다운로드 또는 출력

### 청구서 출력
- T월드 앱에서 PDF 저장 후 출력
- 가입 증명서 용도로 사용 가능
- 최근 12개월 청구서 조회 가능

## 통지서 미수령 시

### 확인 사항
1. **등록된 주소 확인**
   - T월드 앱 > 내 정보 관리 > 주소 확인
2. **우편함 확인**
   - 공동주택의 경우 우편함 번호 확인
3. **이사 여부 확인**
   - 우체국 전입신고 후에도 별도 변경 필요

### 해결 방법
- 주소가 잘못된 경우 → 주소 변경
- 수령 방법 변경 원할 시 → 이메일/앱으로 전환
- 분실 시 → T월드 앱에서 확인

## 자주 묻는 질문

### Q: 종이 통지서를 안 받으려면 어떻게 하나요?
A: T월드 앱 > 청구서 설정에서 '종이 통지서 수령 안함'을 선택하시면 됩니다. 대신 이메일이나 앱으로 받으실 수 있습니다.

### Q: 이사했는데 통지서가 계속 예전 주소로 가요
A: 우체국 전입신고와 SK텔레콤 주소 변경은 별도입니다. T월드 앱이나 114 고객센터에서 주소를 변경해 주세요.

### Q: 다른 사람 주소로 통지서를 받을 수 있나요?
A: 네, 가능합니다. 본인 확인 후 원하시는 주소로 변경하실 수 있습니다. 단, 통지서 수령인이 아닌 가입자 본인에게 요금 납부 의무가 있습니다.

### Q: 통지서에 적힌 금액과 실제 출금 금액이 달라요
A: 자동이체 날짜와 청구서 발행일 사이에 추가 이용분이 있거나, 전월 미납분이 포함되었을 수 있습니다. 상세 내역은 T월드 앱에서 확인해 주세요.

### Q: 세금계산서가 필요해요
A: 사업자인 경우 세금계산서 발행이 가능합니다. T월드 앱 > 청구서 > 세금계산서 발행을 선택하시거나, 고객센터로 문의해 주세요.
