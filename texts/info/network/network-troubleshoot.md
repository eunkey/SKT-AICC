# 통화품질 문제 해결 가이드

> 최종 업데이트: 2025년 2월

## 일반적인 네트워크 문제

### 증상별 진단

| 증상 | 가능한 원인 | 우선 확인 사항 |
|------|-------------|----------------|
| 통화 끊김 | 신호 약함, 핸드오버 실패 | 신호 세기 확인 |
| 상대방 목소리 안 들림 | 음량 설정, 단말 문제 | 볼륨 설정 확인 |
| 내 목소리 전달 안 됨 | 마이크 문제, 앱 권한 | 마이크 점검 |
| 데이터 느림 | 네트워크 혼잡, 요금제 제한 | 데이터 사용량 확인 |
| 인터넷 안 됨 | APN 설정, 데이터 차단 | 데이터 스위치 확인 |
| 전화 수신 안 됨 | 착신전환, 수신차단 설정 | 설정 확인 |

---

## 기본 점검 절차

### 1단계: 단말기 기본 점검

#### 재부팅
가장 먼저 시도할 방법입니다.
```
Android: 전원 버튼 길게 > 다시 시작
iPhone: 볼륨업 > 볼륨다운 > 측면 버튼 길게
```

#### 비행기 모드 토글
```
1. 비행기 모드 켜기
2. 10초 대기
3. 비행기 모드 끄기
```

#### USIM 재장착
```
1. 전원 끄기
2. USIM 트레이 열기
3. USIM 카드 빼서 먼지 제거
4. 다시 삽입 후 전원 켜기
```

### 2단계: 네트워크 설정 확인

#### 네트워크 모드 확인 (Android)
```
설정 > 연결 > 모바일 네트워크 > 네트워크 모드
권장: "5G/LTE/3G/2G 자동 연결"
```

#### 네트워크 모드 확인 (iPhone)
```
설정 > 셀룰러 > 셀룰러 데이터 옵션 > 음성 및 데이터
권장: "5G 자동" 또는 "LTE"
```

#### APN 설정 확인
```
SK텔레콤 APN: lte.sktelecom.com
(대부분 자동 설정됨)
```

### 3단계: 위치 기반 점검

#### 실내에서 문제 발생 시
- 창가로 이동하여 테스트
- Wi-Fi 통화(VoWiFi) 설정 확인
- 중계기 설치 필요 여부 검토

#### 특정 장소에서만 문제 시
- 해당 위치 주소 기록
- 114로 통화품질 신고

## 문제별 상세 해결

### 통화 중 끊김

**원인 분석**:
1. 신호 약한 지역
2. 기지국 간 핸드오버 실패
3. 단말기 문제

**해결 방법**:
1. 신호 상태 확인 (안테나 표시)
2. 다른 장소에서 테스트
3. 통화 중 이동 자제
4. 지속 시 114 품질 신고

### 상대방 목소리가 안 들림

**점검 순서**:
```
1. 볼륨 버튼으로 통화 음량 확인
2. 블루투스 연결 해제 후 재시도
3. 스피커 출력 테스트 (음악 재생)
4. 다른 번호로 통화 테스트
5. 지속 시 서비스센터 점검
```

### 내 목소리가 안 들린다고 함

**점검 순서**:
```
1. 케이스가 마이크 막는지 확인
2. 마이크 구멍 청소
3. 녹음 앱으로 마이크 테스트
4. 다른 통화 앱에서 테스트
5. 지속 시 서비스센터 점검
```

### 데이터 속도 느림

**점검 순서**:
```
1. 데이터 사용량 확인 (요금제 한도 초과?)
2. 속도 측정 앱으로 실측
3. 백그라운드 앱 정리
4. 네트워크 설정 초기화
5. 다른 장소에서 재측정
```

**속도 측정 앱 추천**:
- Speedtest by Ookla
- FAST.com
- SK텔레콤 T service

### 인터넷 연결 안 됨

**단계별 점검**:
```
1. 모바일 데이터 스위치 확인 (켜짐?)
2. 데이터 사용 한도 설정 확인
3. 요금 미납 여부 확인
4. 비행기 모드 토글
5. 네트워크 설정 초기화
```

#### 네트워크 설정 초기화 (Android)
```
설정 > 일반 > 초기화 > 네트워크 설정 초기화
```

#### 네트워크 설정 초기화 (iPhone)
```
설정 > 일반 > 전송 또는 iPhone 재설정 > 재설정 > 네트워크 설정 재설정
```

### 전화가 안 옴 (수신 불가)

**확인 사항**:
| 항목 | 확인 방법 |
|------|-----------|
| 착신전환 | 설정 > 전화 > 착신전환 |
| 수신차단 | 설정 > 전화 > 차단된 연락처 |
| 방해금지 | 설정 > 집중 모드 |
| 요금 미납 | T월드 앱 > 요금 확인 |
| 분실 정지 | 114 확인 |

## 특수 상황 해결

### 지하철에서 끊김

**원인**: 역간 이동 시 핸드오버, 터널 구간 음영
**해결**:
- VoLTE 활성화 확인
- Wi-Fi 통화(VoWiFi) 활성화
- 중요 통화는 지상에서

### 건물 내부 음영

**원인**: 콘크리트/철근에 의한 전파 차단
**해결**:
- 창가 또는 외부에서 통화
- Wi-Fi 통화 활성화
- 가정용 중계기 신청 (무료)

### 해외 로밍 시 문제

**확인 사항**:
```
1. 로밍 서비스 활성화 여부
2. 데이터 로밍 설정 켜짐
3. 현지 네트워크 수동 선택 시도
4. 로밍 상품 가입 확인
```

## VoLTE / VoWiFi 설정

### VoLTE (HD Voice)
고품질 음성통화 기능

**활성화 방법 (Android)**:
```
설정 > 연결 > 모바일 네트워크 > VoLTE 통화 > 켜기
```

**활성화 방법 (iPhone)**:
```
설정 > 셀룰러 > 셀룰러 데이터 옵션 > LTE 활성화 > 음성 및 데이터
```

### VoWiFi (Wi-Fi 통화)
Wi-Fi로 통화하는 기능 (음영지역 해결)

**활성화 방법**:
```
설정 > 연결 > Wi-Fi 통화 > 켜기
```

**지원 단말**:
- Galaxy S20 이후
- iPhone 6s 이후
- 일부 중저가 단말

## 통화품질 신고

### 신고 방법
| 채널 | 방법 |
|------|------|
| 114 | "통화품질 불량" 선택 |
| T월드 앱 | 고객센터 > 품질 신고 |
| 웹사이트 | tworld.co.kr > 고객센터 |

### 신고 시 필요 정보
- 불편 발생 위치 (상세 주소)
- 불편 유형 (통화 끊김/음질 불량/데이터 등)
- 발생 빈도 (항상/가끔/특정 시간)
- 사용 단말기 모델명

### 처리 절차
```
1. 신고 접수
      ↓
2. 현장 측정 (필요 시)
      ↓
3. 기지국 점검/최적화
      ↓
4. 중계기 설치 검토
      ↓
5. 처리 결과 안내 (SMS)
```

## 자주 묻는 질문 (FAQ)

**Q: 집에서만 통화가 안 되는데 해결 방법이 있나요?**
A: 가정용 중계기 설치를 신청하시면 무료로 설치해 드립니다. 114로 신청해 주세요.

**Q: 특정 번호로만 통화가 안 됩니다.**
A: 수신차단 설정, 착신전환 설정을 확인하고, 상대방에게도 확인 요청해 주세요.

**Q: 통화 중 "삐" 소리가 납니다.**
A: 통화 녹음 알림음일 수 있습니다. 법적 의무 사항으로 끌 수 없습니다.

**Q: USIM 불량인지 어떻게 알 수 있나요?**
A: 다른 단말기에 USIM을 넣어 테스트하거나, T월드 매장에서 무료 점검 가능합니다.

**Q: 네트워크 설정 초기화하면 데이터가 지워지나요?**
A: 네트워크 설정만 초기화되며, 사진/연락처 등 데이터는 유지됩니다. Wi-Fi 비밀번호는 재입력해야 합니다.
