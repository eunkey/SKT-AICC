# 통화품질 문제 해결 가이드

> 최종 업데이트: 2024년 1월

## 증상별 해결 방법

### 1. 통화가 끊김 / 상대방 목소리가 안 들림

#### 즉시 시도할 조치
1. **비행기 모드 ON/OFF**
   - 설정 > 비행기 모드 켜기 > 10초 대기 > 끄기

2. **네트워크 재검색**
   - iPhone: 설정 > 셀룰러 > 네트워크 선택 > 자동
   - Galaxy: 설정 > 연결 > 모바일 네트워크 > 네트워크 운영자 > 자동 선택

3. **USIM 분리 후 재장착**
   - 전원 끄기 > USIM 트레이 분리 > 10초 대기 > 재장착 > 전원 켜기

4. **단말기 재부팅**
   - 전원 완전히 끄고 30초 후 재시작

#### 위 방법으로 해결되지 않으면
- 다른 장소에서 테스트 (장소 문제 확인)
- 다른 USIM으로 테스트 (USIM 문제 확인)
- 114 고객센터 네트워크 상담 요청

---

### 2. 통화 시 잡음 / 울림 현상

#### 원인별 해결방법

**단말기 문제 의심**
- 스피커/마이크 부분 이물질 제거
- 케이스 제거 후 테스트
- 블루투스 연결 해제 후 테스트
- 삼성: 설정 > 소리 > 음질 및 효과 > 통화 음질 조정

**네트워크 문제 의심**
- Wi-Fi 통화 활성화 상태면 일시 해제
- VoLTE 설정 확인 및 재설정

---

### 3. 데이터가 느리거나 안 됨

#### 단계별 확인사항

**1단계: 기본 확인**
- [ ] 모바일 데이터 켜져 있는지 확인
- [ ] 비행기 모드 꺼져 있는지 확인
- [ ] 데이터 잔여량 확인 (T월드 앱)

**2단계: 네트워크 설정**
- [ ] APN 설정 초기화
  - iPhone: 설정 > 셀룰러 > 셀룰러 데이터 네트워크 > 리셋
  - Galaxy: 설정 > 연결 > 모바일 네트워크 > APN > 메뉴 > 기본값으로 리셋

**3단계: USIM/단말기**
- [ ] USIM 분리 후 재장착
- [ ] 단말기 재부팅
- [ ] 네트워크 설정 초기화

---

### 4. 전화가 안 걸리거나 안 받아짐

#### 확인사항
| 상황 | 확인 내용 |
|------|----------|
| 발신 불가 | 요금 연체 확인 (T월드 > 요금) |
| 수신 불가 | 착신전환 설정 확인, 차단 목록 확인 |
| 특정 번호만 | 상대방 번호 차단 여부 확인 |
| 모든 통화 | 이용정지 상태 확인 |

#### 해결 절차
1. T월드 앱에서 이용정지 여부 확인
2. 미납 요금 있으면 납부
3. 착신전환 해제: ##002# 발신
4. 문제 지속 시 114 상담

---

### 5. VoLTE 문제

#### VoLTE란?
- LTE 망을 통한 고품질 음성통화 서비스
- 일반 통화보다 빠른 연결, 선명한 음질

#### VoLTE 활성화 방법
**iPhone**
1. 설정 > 셀룰러 > 셀룰러 데이터 옵션
2. 음성 및 데이터 > LTE, VoLTE 켜기

**Galaxy**
1. 설정 > 연결 > 모바일 네트워크
2. VoLTE 통화 > 켜기

#### VoLTE 문제 해결
1. VoLTE 끄기 > 재부팅 > 다시 켜기
2. 네트워크 모드 LTE/3G 자동으로 변경
3. 문제 지속 시 114 상담

## 네트워크 설정 초기화

### iPhone
1. 설정 > 일반 > 전송 또는 iPhone 재설정
2. 재설정 > 네트워크 설정 재설정
3. 암호 입력 후 확인
4. 재부팅 후 Wi-Fi 재연결 필요

### Galaxy
1. 설정 > 일반 > 초기화
2. 네트워크 설정 초기화
3. 설정 초기화 > 확인
4. Wi-Fi, 블루투스, 모바일 네트워크 재설정 필요

**주의**: 저장된 Wi-Fi 비밀번호, 블루투스 페어링 정보가 삭제됩니다.

## 통화품질 측정 방법

### Ookla Speedtest
1. App Store/Play Store에서 'Speedtest' 검색
2. 앱 실행 후 'GO' 버튼
3. 다운로드/업로드 속도 및 핑 확인

### 정상 속도 기준
| 네트워크 | 다운로드 | 업로드 | 핑 |
|----------|----------|--------|-----|
| 5G | 300~800Mbps | 40~80Mbps | 10~30ms |
| LTE | 50~150Mbps | 20~40Mbps | 20~50ms |
| 3G | 1~10Mbps | 0.5~2Mbps | 50~100ms |

## 음영지역 개선 신청

### 중계기 설치 신청
특정 장소(자택, 사무실 등)에서 지속적으로 통화품질이 불량한 경우:

1. **114 고객센터 신청**
   - 114 > 상담원 연결 > "네트워크 품질 개선 신청"

2. **T월드 앱 신청**
   - 고객지원 > 네트워크 품질 > 품질개선 신청

3. **처리 절차**
   - 신청 접수 → 현장 조사 (1~2주) → 개선 방안 안내
   - 중계기 설치 가능 지역은 무료 설치

## 자주 묻는 질문

### Q: 특정 장소에서만 통화가 안 됩니다
A: 해당 장소가 음영지역일 수 있습니다. 중계기 설치 신청을 해주시면 현장 조사 후 개선 방안을 안내드립니다.

### Q: 통화 중 자꾸 상대방이 제 목소리가 안 들린다고 합니다
A: 마이크 문제일 가능성이 높습니다. 녹음 앱으로 본인 목소리 녹음 테스트 후, 문제 있으면 서비스센터 점검을 권장합니다.

### Q: 갑자기 통화품질이 나빠졌습니다
A: 최근 단말기 업데이트나 설정 변경이 있었는지 확인해 주세요. 없었다면 해당 지역 기지국 점검 중일 수 있으니 114로 문의해 주세요.

### Q: 해외에서 통화품질이 안 좋습니다
A: 해외 로밍은 현지 통신사 네트워크를 이용하므로 품질이 달라질 수 있습니다. 여러 통신사 중 수동으로 다른 통신사 선택을 시도해 보세요.
