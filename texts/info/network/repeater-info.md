# 가정용 중계기 신청 및 설치 안내

> 최종 업데이트: 2025년 2월

## 가정용 중계기란?

집이나 사무실 등 실내에서 통화품질이 좋지 않을 때, SK텔레콤에서 무료로 제공하는 신호 증폭 장치입니다.

### 중계기 효과
| 개선 항목 | 설명 |
|-----------|------|
| 통화 품질 | 끊김 없는 선명한 통화 |
| 데이터 속도 | 안정적인 데이터 연결 |
| 신호 세기 | 실내 전 영역 커버리지 |

---

## 중계기 종류

### 소출력 중계기 (가정용)
| 항목 | 사양 |
|------|------|
| 커버리지 | 약 20~30평 |
| 설치 방식 | 창가 설치 (간단) |
| 전원 | 콘센트 연결 |
| 비용 | **무료** |
| 대상 | 일반 가정, 소규모 사무실 |

### 중출력 중계기
| 항목 | 사양 |
|------|------|
| 커버리지 | 약 50~100평 |
| 설치 방식 | 전문 기사 설치 |
| 비용 | **무료** |
| 대상 | 중형 사무실, 상가 |

### 광중계기 (대형)
| 항목 | 사양 |
|------|------|
| 커버리지 | 100평 이상 |
| 설치 방식 | 현장 실사 후 설치 |
| 비용 | **무료** (특수 환경 별도 협의) |
| 대상 | 대형 건물, 지하 공간 |

---

## 신청 자격

### 신청 가능 대상
- SK텔레콤 가입 고객
- 실내 통화품질 불량 지역 거주/근무
- 정상 요금 납부 고객

### 설치 가능 조건
| 조건 | 설명 |
|------|------|
| 창문 신호 | 창가에서 최소 신호 수신 필요 |
| 전원 | 220V 콘센트 필요 |
| 설치 공간 | 창가 근처 설치 공간 확보 |

### 설치 불가 조건
- 창가에서도 신호가 전혀 없는 경우
- 기지국 설치가 필요한 완전 음영지역
- 이동/임시 거주지

---

## 신청 방법

### 1. 전화 신청 (114)
```
114 전화 > 통화품질 선택 > 중계기 설치 요청
```

### 2. T월드 앱 신청
```
T월드 앱 > 고객센터 > 통화품질 신고 > 중계기 설치 신청
```

### 3. 온라인 신청
```
tworld.co.kr > 고객센터 > 서비스 신청 > 중계기 설치
```

### 4. 매장 방문 신청
```
T월드 직영점/대리점 방문 > 중계기 설치 신청
```

| 항목 | 내용 |
|------|------|
| 설치 주소 | 상세 주소 (건물명, 층수, 호수) |

| 불편 내용 | 통화 끊김, 데이터 불량 등 |

| 설치 희망일 | 방문 가능 날짜/시간 |

## 설치 절차

### 처리 단계
```
1. 신청 접수
      ↓
2. 현장 실사 (필요 시)
   - 신호 측정
   - 설치 위치 확인
      ↓
3. 설치 일정 조율
   - 고객 일정 확인
   - 기사 방문 일시 안내
      ↓
4. 중계기 설치
   - 설치 소요 시간: 30분~1시간
   - 신호 테스트 진행
      ↓
5. 사용 안내
   - 작동 확인
   - 관리 방법 안내
```

### 소요 기간
| 단계 | 소요 기간 |
|------|-----------|
| 접수~실사 | 1~3영업일 |
| 실사~설치 | 3~7영업일 |
| **전체** | **약 1~2주** |

※ 지역/수요에 따라 변동될 수 있습니다.

| 주변 | 통풍 잘 되는 곳 | 전자레인지, 라우터 바로 옆 |

## 중계기 관리

### 일상 관리
| 항목 | 방법 |
|------|------|
| 전원 | 24시간 켜두기 (대기전력 미미) |
| 청소 | 먼지 제거 (마른 천 사용) |
| 이동 | 설치 위치 임의 변경 금지 |

### 문제 발생 시
| 증상 | 조치 |
|------|------|
| LED 꺼짐 | 전원 연결 확인 |
| LED 깜빡임 | 재부팅 (전원 끄고 30초 후 재연결) |
| 신호 불량 지속 | 114 A/S 신청 |

### 고장/교체
- **A/S 신청**: 114 또는 T월드 앱
- **비용**: 무료 (고의 파손 제외)
- **교체 소요**: 약 3~5영업일

---

## 이사/반납

### 이사 시
1. **114 연락**: 이사 2주 전 신청
2. **반납 또는 이전 설치** 선택
3. 이전 설치 시 새 주소 신호 확인 필요

### 반납 방법
| 방법 | 설명 |
|------|------|
| 기사 방문 수거 | 일정 조율 후 방문 |
| 택배 반납 | 착불 택배 발송 (주소 안내) |
| 매장 반납 | T월드 직영점 방문 |

### 해지 시
- 중계기 반납 필요
- 미반납 시 분실 비용 청구 가능

## 자주 묻는 질문 (FAQ)

**Q: 중계기 설치 비용이 드나요?**
A: SK텔레콤 고객에게 무료로 제공됩니다. 설치비, 장비비 모두 무료입니다.

**Q: 중계기로 5G도 사용 가능한가요?**
A: 현재 대부분의 가정용 중계기는 LTE를 지원합니다. 5G 중계기는 별도 문의가 필요합니다.

**Q: 이웃집도 사용할 수 있나요?**
A: 신호가 닿는 범위 내에서는 SK텔레콤 고객이면 누구나 혜택을 받을 수 있습니다.

**Q: 전기료가 많이 나오나요?**
A: 소출력 중계기 기준 월 1,000원 미만의 전기료가 발생합니다.

**Q: 설치 후에도 통화가 안 됩니다.**
A: 114로 연락하시면 현장 점검 후 위치 조정 또는 추가 장비 설치를 검토합니다.

**Q: 지하 주차장에도 설치 가능한가요?**
A: 공용 공간은 건물 관리사무소 동의가 필요하며, 별도 현장 실사 후 설치 여부가 결정됩니다.

**Q: 중계기 소음이 있나요?**
A: 중계기는 무소음 장치입니다. 팬 소리나 기타 소음이 없습니다.
