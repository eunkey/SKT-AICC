# 가정용 중계기 설치 안내

> 최종 업데이트: 2024년 1월

## 중계기란?

가정이나 사무실 등 통화품질이 좋지 않은 실내 공간에 설치하여 이동통신 신호를 증폭해주는 장치입니다. SK텔레콤 고객에게 무료로 제공됩니다.

---

## 중계기 유형

### 1. 소형 중계기 (펨토셀)
| 항목 | 내용 |
|------|------|
| 커버리지 | 약 30~50평 (100㎡ 내외) |
| 설치 조건 | 유선 인터넷(공유기) 연결 필수 |
| 설치 방식 | 셀프 설치 (택배 수령) |
| 전원 | 220V 콘센트 |
| 장점 | 빠른 설치, 즉시 사용 가능 |
| 단점 | 인터넷 연결 필요, 소규모 공간만 |

### 2. 일반 중계기 (옥내용)
| 항목 | 내용 |
|------|------|
| 커버리지 | 약 100~200평 (300~600㎡) |
| 설치 조건 | 외부 안테나 설치 가능 장소 |
| 설치 방식 | 전문가 방문 설치 |
| 전원 | 220V 콘센트 |
| 장점 | 넓은 커버리지, 안정적 품질 |
| 단점 | 설치 일정 조율 필요 |

### 3. 스마트빔 중계기
| 항목 | 내용 |
|------|------|
| 커버리지 | 약 50~100평 |
| 설치 조건 | 창문 부착 가능 장소 |
| 설치 방식 | 셀프 설치 또는 기사 설치 |
| 전원 | 220V 콘센트 |
| 특징 | 창문에 부착하여 외부 신호 수신 |

---

## 설치 신청 절차

### 1단계: 신청
**114 고객센터**
1. 114 전화 > 상담원 연결
2. "중계기 설치 신청" 요청
3. 주소 및 연락처 확인
4. 현장 조사 일정 예약

**T월드 앱**
1. T월드 앱 실행
2. 고객지원 > 네트워크 품질
3. 품질개선 신청 > 중계기 설치
4. 주소 및 상세 정보 입력

### 2단계: 현장 조사
- 신청 후 3~7일 이내 기사 방문
- 현장 전파 환경 측정
- 적합한 중계기 유형 결정
- 설치 가능 여부 및 위치 확인

### 3단계: 설치
**펨토셀 (셀프 설치)**
- 택배로 기기 수령 (2~3일)
- 동봉된 설명서 따라 설치
- 공유기와 랜선 연결
- 전원 연결 후 자동 개통 (약 30분)

**일반 중계기 (기사 설치)**
- 설치 일정 협의
- 기사 방문 설치 (1~2시간)
- 설치 완료 후 품질 테스트

## 비용 안내

| 항목 | 비용 |
|------|------|
| 중계기 기기 | 무료 (임대) |
| 설치비 | 무료 |
| 월 이용료 | 무료 |
| 전기료 | 고객 부담 (월 약 1,000~3,000원) |
| 해지 시 | 기기 반납 필요 |

**주의사항**
- 중계기는 임대 방식으로 SK텔레콤 소유입니다
- 타 통신사 전환 또는 해지 시 반납 필요
- 분실/파손 시 변상금 청구될 수 있음

## 설치 불가 케이스

### 기술적 제한
| 상황 | 사유 |
|------|------|
| 외부 신호가 전혀 없는 지역 | 증폭할 기본 신호가 없음 |
| 기존 중계기 커버리지 내 | 간섭 발생 우려 |
| 군사시설/공항 인근 | 전파법 규제 |

### 환경적 제한
| 상황 | 사유 |
|------|------|
| 외부 안테나 설치 불가 | 건물주 동의 없음, 설치 공간 없음 |
| 인터넷 미설치 (펨토셀) | 펨토셀은 인터넷 연결 필수 |
| 잦은 이사 예정 | 장기 사용 불가 시 |

---

## 펨토셀 셀프 설치 가이드

### 설치 방법
1. **박스 개봉**
   - 펨토셀 본체, 전원 어댑터, 랜선 확인

2. **위치 선정**
   - 창가 또는 통화를 주로 하는 공간
   - 공유기 근처 (랜선 연결 거리)
   - 통풍이 잘 되는 곳 (발열 고려)

3. **연결**
   - 랜선으로 공유기와 펨토셀 연결
   - 전원 어댑터 연결

4. **개통 대기**
   - 전원 LED 점등 확인
   - 약 30분 후 자동 개통
   - 개통 완료 시 SMS 수신

### 설치 후 확인
- 통화 테스트 실시
- 인터넷 속도 저하 없는지 확인
- 기기 발열 상태 확인

## 고장/문제 시 대처

### 증상별 조치
| 증상 | 조치 |
|------|------|
| 전원 LED 안 들어옴 | 전원 어댑터 재연결, 다른 콘센트 사용 |
| 인터넷 LED 깜빡임 | 랜선 재연결, 공유기 재부팅 |
| 통화품질 개선 안 됨 | 위치 변경 (창가로 이동) |
| 기기 과열 | 통풍 확인, 직사광선 피하기 |

### A/S 신청
- 114 고객센터 > "중계기 AS 신청"
- 기기 교체 또는 수리 진행
- 무상 A/S (고객 과실 제외)

## 자주 묻는 질문

### Q: 펨토셀 설치하면 인터넷 속도가 느려지나요?
A: 펨토셀이 사용하는 대역폭은 매우 적어(약 1~2Mbps) 일반적인 인터넷 사용에 영향이 거의 없습니다.

### Q: 펨토셀은 집 전체에 효과가 있나요?
A: 약 30~50평 기준으로 설계되었습니다. 대형 주택의 경우 추가 중계기가 필요할 수 있습니다.

### Q: 중계기 설치 후에도 품질이 안 좋으면요?
A: 114로 재문의하시면 추가 현장 조사 및 장비 조정을 진행합니다.

### Q: 이사 가면 어떻게 하나요?
A: 114로 이전 설치 신청하시면 됩니다. 기사 방문 설치 중계기는 재설치가 필요하고, 펨토셀은 직접 이동 가능합니다.

### Q: 반납 시 택배로 보내면 되나요?
A: 네, 114로 반납 신청하시면 반납용 박스와 송장을 보내드립니다. 착불 택배로 발송해 주세요.
