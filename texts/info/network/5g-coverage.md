## 5G 사용 시 주의사항

### 5G가 잘 되지 않는 환경
1. **지하 주차장**: 기지국 설치가 어려운 폐쇄 공간
2. **건물 깊은 실내**: 고층 건물 내부, 지하 상가
3. **산간/해안 지역**: 기지국 밀도가 낮은 지역
4. **고속 이동 중**: KTX 터널 구간 등

### 5G ↔ LTE 전환
- 5G 신호가 약해지면 자동으로 LTE로 전환됩니다
- 설정에서 "5G 자동" 또는 "LTE 우선" 선택 가능
- 배터리 절약을 원하면 LTE 우선 모드 권장

## 5G 관련 설정

### iPhone 5G 설정
1. 설정 > 셀룰러 > 셀룰러 데이터 옵션
2. 음성 및 데이터
   - **5G 자동**: 배터리와 속도 균형
   - **5G 켜기**: 항상 5G 사용 (배터리 소모 증가)
   - **LTE**: 5G 사용 안 함

### 삼성 Galaxy 5G 설정
1. 설정 > 연결 > 모바일 네트워크 > 네트워크 모드
2. 선택 옵션:
   - **5G/LTE/3G/2G 자동 연결**: 권장
   - **LTE/3G/2G 자동 연결**: 5G 비활성화

## 5G 커버리지 확인 방법

### T월드 앱
1. T월드 앱 실행
2. 메뉴 > 고객지원 > 네트워크 품질
3. 현재 위치 또는 특정 주소 검색
4. 5G/LTE 커버리지 확인

- 특정 지역 커버리지 확인 가능

## 자주 묻는 질문

### Q: 5G폰인데 LTE만 잡혀요
A: 다음 사항을 확인해 주세요:
1. 5G 요금제 가입 여부 확인 (LTE 요금제는 5G 불가)
2. 휴대폰 설정에서 5G 활성화 여부 확인
3. 해당 지역 5G 커버리지 확인
4. USIM 5G 지원 여부 확인 (구형 USIM은 교체 필요)

### Q: 5G 요금제인데 속도가 느려요
A: 다음 상황일 수 있습니다:
- 혼잡 지역/시간대 (점심, 퇴근 시간)
- 5G 음영지역 (LTE로 자동 전환된 상태)
- 데이터 사용량 소진 후 속도 제한

### Q: 5G가 안 되는 곳에서 통화는 가능한가요?
A: 네, 통화는 LTE 또는 3G 망으로 자동 전환되어 가능합니다. 5G는 데이터 통신에 최적화되어 있습니다.

### Q: 지하철에서 5G 사용 가능한가요?
A: 서울 지하철 전 노선(1~9호선, 신분당선)에서 5G 사용 가능합니다. 일부 역간 터널 구간에서 순간적으로 LTE 전환될 수 있습니다.
