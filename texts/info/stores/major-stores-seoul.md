## 지역별 찾아가기

### 강남/서초
```
강남역 → T월드 강남직영점 (도보 2분)
         T월드 강남역점 (도보 3분)
교대역 → T월드 서초점 (도보 5분)
```

### 종로/중구
```
광화문역 → T월드 광화문점 (도보 3분)
명동역 → T월드 명동점 (도보 3분)
종로3가역 → T월드 종로점 (도보 2분)
```

```
여의도역 → T월드 여의도IFC점 (IFC몰 직결)
```

## 자주 묻는 질문 (FAQ)

**Q: 가장 큰 매장은 어디인가요?**
A: T월드 강남직영점이 플래그십 스토어로 가장 크고, 전 기종 체험 및 프리미엄 라운지가 있습니다.

**Q: 법인 계약은 어느 매장에서 가능한가요?**
A: 모든 직영점에서 가능하며, 대규모 법인은 법인영업센터를 권장합니다.

**Q: 외국어 상담이 가능한 매장이 있나요?**
A: T월드 명동점에서 영어/중국어 상담이 가능합니다.

**Q: 프리미엄 라운지는 누가 이용할 수 있나요?**
A: T멤버십 VIP/플래티넘 등급 고객이 이용 가능합니다. 강남직영점, 잠실롯데월드타워점에서 운영합니다.
