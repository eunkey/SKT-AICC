# SK텔레콤 매장 유형 안내

> 최종 업데이트: 2025년 2월

## 매장 유형 구분

### 1. T월드 직영점
SK텔레콤이 직접 운영하는 공식 매장

| 특징 | 설명 |
|------|------|
| 운영 주체 | SK텔레콤 직접 운영 |
| 서비스 범위 | 모든 서비스 가능 (가입, 변경, 해지, A/S 접수 등) |
| 매장 외관 | T월드 로고, 통일된 인테리어 |
| 직원 | SK텔레콤 정규직/계약직 |
| 영업시간 | 평일 10:00~20:00 / 주말 10:00~19:00 (매장별 상이) |

**직영점 주요 서비스**:
- 신규 가입/번호이동/기기변경
- 요금제 변경/부가서비스 신청
- 명의변경/분실 신고
- 단말기 체험존 운영
- 프리미엄 상담 (VIP 전용 공간)

### 2. 공식 대리점
SK텔레콤과 계약한 법인이 운영하는 매장

| 특징 | 설명 |
|------|------|
| 운영 주체 | 대리점 사업자 |
| 서비스 범위 | 대부분의 서비스 가능 |
| 매장 외관 | 'SK텔레콤 공식대리점' 표기 |
| 특징 | 자체 프로모션 가능 |

**대리점 가능 서비스**:
- 신규 가입/번호이동/기기변경
- 요금제 변경/부가서비스 신청
- 명의변경 (일부 제한)

**대리점 제한 서비스**:
- 일부 복잡한 업무 처리 불가
- 법인 계약 일부 제한

### 3. 판매점
소규모 개인 사업자 운영 매장

| 특징 | 설명 |
|------|------|
| 운영 주체 | 개인 사업자 |
| 서비스 범위 | 판매 위주 (가입, 번호이동) |
| 매장 외관 | 다양함 (복합 통신매장 등) |
| 특징 | 경쟁사 상품 함께 취급 가능 |

**판매점 가능 서비스**:
- 신규 가입/번호이동/기기변경
- 단순 요금제 변경

**판매점 제한 서비스**:
- 명의변경 불가
- 복잡한 요금 상담 어려움
- 해지 처리 불가

## 자주 묻는 질문 (FAQ)

**Q: 직영점과 대리점, 어디로 가는 게 좋나요?**
A: 일반적인 업무는 어디서나 가능합니다. 복잡한 업무(명의변경, 법인계약, 해지 등)는 직영점을 권장합니다.

**Q: 대리점에서 받은 사은품에 문제가 있어요.**
A: 사은품 관련 문의는 해당 대리점으로 연락해 주세요. 이동통신 서비스와는 별개입니다.

**Q: 주말에도 매장 이용 가능한가요?**
A: 대부분의 직영점/대리점은 주말에도 운영합니다. 다만 영업시간이 평일보다 짧을 수 있으니 방문 전 확인해 주세요.

**Q: 공식대리점인지 어떻게 확인하나요?**
A: T월드 앱/웹사이트 '매장 찾기'에 등록된 매장은 모두 공식 매장입니다. 간판에 'SK텔레콤 공식대리점' 표기가 있습니다.

**Q: 온라인으로 가입하면 매장보다 혜택이 다른가요?**
A: 온라인 전용 프로모션이 있을 수 있습니다. 매장도 자체 프로모션을 제공하므로 비교 후 선택해 주세요.
