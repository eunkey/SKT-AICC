> 최종 업데이트: 2025년 2월

## 고객센터 채널 안내

### 전화 고객센터 (114)
| 항목 | 정보 |
|------|------|
| 전화번호 | 114 (SK텔레콤 휴대폰) / 1599-0011 (유선/타사) |
| 운영시간 | 상담사: 평일 09:00~18:00 / ARS: 24시간 |
| 요금 | 무료 (114) |

**ARS 메뉴 안내**:
```
1. 요금/납부 관련
2. 요금제/부가서비스
3. 분실/정지
4. 해지/번호이동
5. 기타 문의
0. 상담사 연결
```

### T월드 앱 고객센터
- **위치**: T월드 앱 > 더보기 > 고객센터
- **채팅 상담**: 24시간 AI 상담 / 평일 09:00~18:00 상담사
- **이메일 문의**: 답변 1~2영업일 소요

### 온라인 고객센터
- **URL**: [tworld.co.kr](https://www.tworld.co.kr) > 고객센터
- **FAQ**: 자주 묻는 질문 검색
- **1:1 문의**: 로그인 후 이용

## 주요 민원 처리 절차

### 일반 민원 처리 흐름
```
1. 민원 접수 (114/앱/온라인)
      ↓
2. 담당자 배정 (1영업일 이내)
      ↓
3. 사실 확인 및 검토 (3~5영업일)
      ↓
4. 처리 결과 안내 (SMS/전화)
      ↓
5. 이의 시 재심 요청 가능
```

### 분쟁 조정
SK텔레콤 내부 처리로 해결되지 않는 경우:
1. **통신분쟁조정위원회**: 방송통신위원회 산하
2. **한국소비자원**: 소비자 분쟁 조정
3. **지역 소비자보호원**: 지역별 조정

## 자주 묻는 질문 (FAQ)

**Q: 114가 안 되는데 어떻게 하나요?**
A: 유선전화나 타사 휴대폰에서 1599-0011로 연락해 주세요.

**Q: 상담사 연결 대기가 너무 길어요.**
A: T월드 앱 채팅 상담을 이용하시면 대기 없이 상담 가능합니다.

**Q: 주말에도 상담 가능한가요?**
A: 상담사 연결은 평일만 가능하며, 주말은 ARS 자동 응답 또는 앱 이용을 권장합니다.

**Q: 외국어 상담이 가능한가요?**
A: 영어 상담은 114 연결 후 영어 서비스 요청하시면 됩니다. T월드 명동점 방문도 가능합니다.

**Q: 고객센터 통화 녹음은 어떻게 요청하나요?**
A: 분쟁 발생 시 통화 녹음 확인을 요청할 수 있습니다. 114로 연락하여 녹음 확인 신청해 주세요.
