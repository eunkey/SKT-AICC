## 자가 점검 가이드

### 기본 점검 항목

#### 전원/충전 문제
1. 다른 충전기/케이블 사용해 보기
2. 단자 먼지/이물질 제거
3. 강제 재부팅 시도
   - 삼성: 전원 + 볼륨 하 7초 이상
   - 아이폰: 볼륨 업 → 볼륨 다운 → 측면 버튼 길게

#### 화면 터치 불량
1. 화면 보호필름 제거 후 테스트
2. 안전 모드 부팅 (타사 앱 문제 확인)
3. 화면 민감도 설정 조정

#### 배터리 소모 빠름
1. 배터리 사용량 확인 (설정 > 배터리)
2. 백그라운드 앱 정리
3. 절전 모드 활성화
4. 화면 밝기/AOD 설정 조정

#### 발열 문제
1. 케이스 제거
2. 사용하지 않는 앱 종료
3. 고사양 앱/게임 사용 제한
4. 직사광선/고온 환경 피하기

## 자주 묻는 질문 (FAQ)

**Q: 화면이 깨졌는데 수리하는 게 나을까요, 새로 사는 게 나을까요?**
A: 단말기 사용 기간과 수리비를 고려해 주세요. 보통 2년 이상 사용한 기기는 신규 구매를 권장드립니다.

**Q: 비공식 수리점에서 수리해도 되나요?**
A: 비공식 수리 시 제조사 보증이 무효화되고, 품질을 보장받기 어렵습니다. 공식 서비스센터 이용을 권장합니다.

**Q: 물에 빠진 폰은 어떻게 해야 하나요?**
A: 1) 즉시 전원 끄기, 2) 물기 털어내기, 3) 충전/전원 버튼 누르지 않기, 4) 최대한 빨리 서비스센터 방문

**Q: 보증기간 내인데 수리비를 내라고 해요.**
A: 물/충격에 의한 파손, 비공식 수리 이력, 사용자 과실의 경우 유상 수리입니다. 제조 결함인 경우 무상 수리됩니다.

**Q: 데이터 백업 후 수리 맡겨야 하나요?**
A: 네, 수리 과정에서 데이터가 삭제될 수 있습니다. 삼성 클라우드/iCloud/Google 백업을 권장합니다.
