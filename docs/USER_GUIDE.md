# SKT AICC 사용법 가이드

SKT AICC 웹 애플리케이션 사용 설명서입니다.

---

## 1. 시작하기

### 브라우저 요구사항
- **Chrome 브라우저 필수** (다른 브라우저는 지원되지 않습니다)

### 네트워크 설정
- **사내망 사용 불가**: 보안 정책으로 인해 사내망에서는 접속이 제한됩니다
- **외부망 필요**: 모바일 핫스팟 또는 외부 네트워크를 사용해주세요

### 로그인
- 아무 이메일/비밀번호로 접근 가능합니다
- 별도의 계정 생성 절차가 필요 없습니다

---

## 2. 상담 데모 모드

미리 준비된 시나리오로 AI 상담 데모를 체험할 수 있습니다.

### 사용 방법
1. **데모 탭** 선택
2. **시나리오 선택**:
   - 요금제 변경
   - 프리미엄 해지
   - OTT 정리
   - 가족 결합
3. **"상담 시작"** 버튼 클릭
4. 자동으로 대화가 재생됩니다

---

## 3. AI 음성 상담 (직접 사용)

실제 음성으로 AI 상담사와 대화할 수 있습니다.

### 사용 방법
1. **"AI 대화"** 탭 선택
2. **"상담 시작"** 버튼 클릭
3. **"대화 시작"** 버튼 클릭
4. **"음성 듣는 중"** 표시가 나타나면 말하기 시작
5. 자동으로 자막이 생성됩니다
6. AI 상담사가 자동으로 응답합니다

---

## 4. AI 인텔리전스 기능

상담 중 AI 지원 기능을 활용할 수 있습니다.

### 접근 방법
- 우측 하단의 **보라색 버튼** 클릭
- 또는 단축키 `Ctrl+Shift+A` 사용

### 주요 기능
- **요약**: 현재까지의 상담 내용 요약
- **검색**: 관련 정보 검색

---

## 5. 상담 종료

### 종료 방법
1. **"상담 종료"** 버튼 클릭
2. 자동으로 상담 **요약이 생성**됩니다
3. 고객에게 보낼 **SMS 문자 초안**이 자동 생성됩니다

---

## 6. 고객 관리 탭

고급 고객 관리 기능은 `/advanced` 경로에서 이용할 수 있습니다.

### 프로필
- 고객별 프로필 정보 확인
- 상세 고객 정보 조회

### 요금제
- 요금제 선택 및 비교
- 해지 분석 기능

### 부가서비스
- 부가서비스 선택 및 관리
- 서비스별 해지 분석

### 할인
- 할인 항목 확인
- 해지 분석 제공:
  - 단기 분석
  - 중기 분석
  - 장기 분석

### 요금 시뮬레이터
- 체크박스로 항목 선택
- 선택한 항목에 따라 요금 자동 계산
